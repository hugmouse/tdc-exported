
# మున్సిపల్ ఎన్నికల కోసమే పార్టీల డ్రామా : గంగుల కమలాకర్

Ph: 88865 47666ప్రవేశం ఉచితం ADVT కరీంనగర్ : ఆర్టీసీ కార్మికులు యూనియన్ నేతల మాయలో పడొద్దని మంత్రి గంగుల కమలాకర్ హితవు పలికారు. యూనియన్ నాయకులు ఎప్పుడూ విధుల్లో లేరని ఆయన మండిపడ్డారు. కార్మికుల సంక్షేమం కోసమే ప్రభుత్వం ప్రయత్నిస్తోందని, సీఎం చెప్పిన మేరకు కార్మికులు విధుల్లో చేరాలని ఆయన పిలుపునిచ్చారు. సమ్మెలో రాజకీయ నేతలు చేరి, కార్మికులను రోడ్డుపైకి తెచ్చారని మండిపడ్డారు.  బాబు అనే కార్మికుడు చనిపోవడం చాలా బాధ కలిగించిందని, అంత్యక్రియలకు వెళ్లిన ఏ పెద్ద నాయకుడు వారి కుటుంబానికి సాయం చేయలేదని విమర్శించారు. డ్రైవర్ బాబు కుటుంబాన్ని పరామర్శించి, వారికి ఆర్థిక సాయం అందిస్తానని ఆయన ప్రకటించారు. మున్సిపల్ ఎన్నికల్లో లబ్ధి పొందడం కోసమే పార్టీలు డ్రామాలాడుతున్నాయని గంగుల ధ్వజమెత్తారు.
