
# 復興庁、設置１０年延長へ基本方針…避難者の福島帰還に重点

Published at: **2019-11-07T04:13:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191107-OYT1T50154/)

　復興庁は７日、東日本大震災からの復興方針を議論する復興推進委員会を開き、２０２１年３月末までとなっている同庁の設置期限を１０年間延長し、３１年３月末までとする基本方針の骨子案を示した。復興を着実に進めるため、引き続き首相直轄の組織として運営する。被災３県の知事も加わる委員会での意見を踏まえて修正し、年内に閣議決定したうえで、来年の通常国会に関連法案を提出する。
　骨子案では、延長期間は東京電力福島第一原発事故の影響で復興が長期化する「原子力災害被災地域」の福島県は３１年３月末まで、「地震・津波被災地域」の宮城、岩手両県は２６年３月末までとする。
　福島県での事業では避難した被災者の帰還を促すための環境整備に重点的に取り組むほか、風評被害の払拭（ふっしょく）に向け情報発信を強化する。また、沿岸部の浜通りに新たな産業基盤を作る構想を推進させる。現地組織の福島復興局（福島市）と富岡、浪江両町の支所は維持する。
　福島県では大熊、双葉両町などが、避難指示が続く帰還困難区域となっている。県内外への避難者は、ピーク時の１２年５月の１６万４８６５人から約７５％減ったが、今年７月時点でも４万２７０５人いる。
　一方、宮城、岩手両県では被災者の心身のケアや水産加工業をはじめとする地場産業の販路回復などに力を入れる。仙台市と盛岡市にある復興局はいずれも支所に変更し、両県の沿岸部に復興局を設ける。
　必要な財源については、一般会計から切り離して運用している東日本大震災復興特別会計と、被災自治体を支援する震災復興特別交付税を継続して確保する。
　復興庁の存続を巡っては今年３月、政府が２１年４月以降の新たな復興基本方針を閣議決定した際、後継組織を置くと明記。これに基づき、政府・与党は具体的な組織のあり方の検討を進めてきた。
　与党内では、自然災害全般への対応も受け持つ「防災・復興庁」案なども出ていたが、政府は復興庁が他省庁より高い位置づけとなっている現体制の方が復興を推進できると判断し、同庁存続の方針を決めていた。
