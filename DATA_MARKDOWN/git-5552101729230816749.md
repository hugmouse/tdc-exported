
# 【獨家】男人婆變小女人 　AMOi AMOi少琦陷熱戀

Published at: **2019-11-02T01:58:00+00:00**

Author: **陳于嬙**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191101ent024/)

大馬鬼才黃明志推出的百萬女團AMOi AMOi，陸續傳出新戀情，繼May吳翠媚之後，曾長鬍子的男人婆少琦也脫單，在IG上狂曬與新男友池煒城(Joshua)出遊照，她大方說：「這是第一次帶男朋友跟家人一起去旅遊，當然要以結婚為前提才敢這麼做，不然交往又不結婚，好像很浪費時間。」
黃明志一手打造的網路女神，近期紛紛傳出新戀情，本刊曾拍到四葉草朱主愛與男團「SpeXial」林子閎相戀，兩人隨後也大方認愛；改回本名「何戀慈」的何念兹，也被拍到和《大學生了沒》班底陳希騰(史丹)一同上健身房，同款情侶鞋洩了兩人情愫；而AMOi AMOi的May，也認愛台灣大肌肉猛男，女神們紛紛死會。
如今，AMOi AMOi女團的少琦也傳戀曲，與擔任電影製作兼Youtuber的池煒城(Joshua)相戀，少琦表示，公司跟老闆黃明志沒有「禁愛令」，加上粉絲們都很開放，都希望看到偶像也有幸福的時刻，因此，才決定公開戀情，她表示，兩人交往將近一年，往後餘生都是他了。
有了愛情的滋潤，向來是男人婆的少琦轉變成愛撒嬌的小女人，她甜滋滋說：「我也不知道原因，但可能是愛情的滋潤吧。」
