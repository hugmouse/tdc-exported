
# 両陛下、大嘗祭リハーサル「習礼」に臨まれる

Published at: **2019-11-06T11:47:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/kaigen/news/20191106-OYT1T50301/)

　天皇、皇后両陛下は６日、皇居・東御苑で、皇位継承に伴う伝統儀式「大嘗祭（だいじょうさい）」のリハーサル「習礼（しゅらい）」に臨まれた。本番は１４～１５日に行われる。
　両陛下は午後１時４５分頃、車で皇居に入られた。習礼は非公開で、東御苑に建設された大嘗宮（だいじょうきゅう）で行われた。宮内庁によると、両陛下は本番と同じ装束を着用し、所作などを確認されたという。
