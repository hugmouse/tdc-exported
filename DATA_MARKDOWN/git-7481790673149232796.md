
# ＮＨＫ同時配信「節度を」…民放連会長、改めて注文

Published at: **2019-11-06T12:57:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/culture/20191106-OYT1T50318/)

　日本民間放送連盟（民放連）の大久保好男会長（日本テレビ放送網会長）は６日、ＮＨＫがテレビ番組を放送と同時にインターネットに流す「常時同時配信」について、「真に必要な業務かどうかの精査」を求め、「民間事業と競合しないよう、過度の肥大化が進まないよう、節度をもって抑制的に運営してほしい」と改めて注文した。東京都内で開かれた第６７回民間放送全国大会のあいさつで言及した。
　来賓の上田良一ＮＨＫ会長は「民放との二元体制を維持、発展させていくことがきわめて重要。常時同時配信を実施する中で得られる様々な知見についても極力、民放と情報共有をさせてほしい」と述べた。
　常時同時配信を可能にする改正放送法が５月に成立したことを受け、ＮＨＫは今年度中の実施を目指し、１０月に実施基準案を総務相に認可申請している。
　　　　　◇　
　同大会で、今年の民放連賞のグランプリなどが発表された。テレビのグランプリは、北海道テレビ放送のドラマ「チャンネルはそのまま！」、ラジオのグランプリは、南海放送の報道番組「緊急放流＝逃げろ！～誰が命を奪ったのか～」に決まった。
