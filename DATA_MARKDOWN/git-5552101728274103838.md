
# 【雙11開戰】3C通路也參戰　燦坤送上億優惠、全國電網購上線

Published at: **2019-11-05T09:35:00+00:00**

Author: **曹以斌, 吳貞慧, 林育緯**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105fin003/)

雙11已成人人必搶商機，不只電商業者，實體通路業者也紛紛搶進，包括燦坤與全國電子兩大3C通路也宣告參戰，前一天才透露轉型之路的燦坤，5日宣布結合燦坤全民購物節，打造「雙11全民購物節」，人氣商品低於5折起，總計降價總額超過億元，還獨家首賣三大科技品牌新品。以「破盤價」促銷著稱的全國電子，也將在11月7日，正式推出新電商平台上線，不讓燦坤專美於前。
「燦坤雙11全民購物節與百大品牌合作，將推出千項商品與破億優惠，獨家引進3大科技品牌商品，包含智慧清掃類睿米Roidmi、智慧穿戴類華米Amazfit、智慧照明類易來Yeelight等，要讓消費者可體驗人氣新品，同時與台灣大車隊獨家合作，享有燦坤會員搭車金優惠，全力朝360度的生活體驗型電子產品通路發展。」燦坤總經理李佳峰說。
燦坤宣佈今年的「雙11全民購物節」活動將從11月8日起跑，活動期間至11月12日將推出五大驚喜，包含多款人氣熱銷商品低於五折起；獨家首賣三大科技品牌新品；買百萬滿額禮最狂贈市價99,999元75吋4K電視；每日限量秒殺排隊商品最低34折起；購物小確幸四連發，獨享台灣大車隊55688搭車金、銀行回饋最高4,500元、天降百萬紅包雨、精選百大強品搶先預購等優惠活動。「電商及門市同步打出優惠，會員和非會員亦同享折扣。」李佳峰強調。
至於另一家3C通路大廠全國電子，因應雙11到來，全新打造的電商平台也將於今年的雙11前亮相，全國電子透露，屆時將分享破盤1111一系列線上線下優惠方案。
