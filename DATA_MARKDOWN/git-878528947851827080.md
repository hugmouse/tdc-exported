
# 侍の不安一掃０封リレー　今永５連続Ｋ！貫禄３回０封「必ずいい形で日本に」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/02/0012841529.shtml)

　「侍ジャパン強化試合、日本３－０カナダ」（１日、沖縄セルラースタジアム那覇）
　野球日本代表「侍ジャパン」はカナダとの強化試合第２戦に快勝し、１勝１敗で大会前の実戦を終えた。先発の今永昇太投手（２６）＝ＤｅＮＡ＝が３回を１安打無失点、５者連続を含む６奪三振と圧巻の投球。打線は四回に内野ゴロの間に１点先制し、五回、八回に加点した。東京五輪予選を兼ねた国際大会「プレミア１２」は、メキシコで３日（現地時間２日）に行われる１次ラウンドＡ組で開幕する。Ｂ組の日本は２日に台湾に入り、５日から１次ラウンドを戦う。
　先発の今永が圧巻の奪三振ショーで本大会前最後の実戦登板を終えた。最速１５０キロの直球にカットボールを効果的に投じ、カナダ打線を翻弄（ほんろう）。３回を１安打無失点、無四球６奪三振の内容に「やりたいことは全てできました」と納得の表情を浮かべた。
　初回２死二塁のピンチを切り抜けるとアクセル全開。二回先頭のルブランから５者連続三振を奪い、マウンドに仁王立ちした。試合前のブルペンでは「やばいかな」と大乱調だったが、会沢のリードに救われたという。
　要所でカーブを投じたことで「腕が遠回りせずに体の近くで振れた」と修正に成功。右打者の懐にカットボールを投げ込み、徐々に本来の自分を取り戻していった。会沢には試合の中で全球種を投じるのではなく、終盤に向けて“隠し球”をつくる重要性も説かれている。女房役の存在が左腕を光らせた。
　これで代表戦は４試合に登板し、「自分自身のバロメーター」と話す奪三振数は計１３イニングで２６と圧倒的な数字を誇る。１次ラウンド３戦目の７日・台湾戦の先発が有力な左腕は表情を引き締め、世界一を力強く誓った。
　「世界一になって秋山さんにいい報告ができるように、シャンパンファイトに参加していただけるように」
　前夜のカナダ戦で死球を受けた秋山が無念の負傷離脱。チームにとって大きな痛手だが、結束力は深まった。「必ずいい形で日本に帰ってきたい」。後ろは振り向かず、前だけを見て突き進む。
