
# യുഎപിഎ തന്നിഷ്ടപ്രകാരം; സര്‍ക്കാർ തിരുത്തും: പൊലീസിനെതിരെ പി.ജയരാജന്‍

Published at: **2019-11-03T15:33:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/03/p-jayarajan-slams-kerala-police-on-uapa-case.html)

കോഴിക്കോട്∙ എല്ലാവര്‍ക്കുമെതിരെ ചുമത്താവുന്നതല്ല യുഎപിഎ എന്ന് സിപിഎം സംസ്ഥാനസമിതി അംഗം പി.ജയരാജന്‍. പൊലീസിന്റെ നടപടി തന്നിഷ്ടപ്രകാരമുള്ളതാണ്. സര്‍ക്കാര്‍ തിരുത്തും. യുഎപിഎ കരിനിയമം തന്നെയാണ്. അട്ടപ്പാടിയിലേതു വ്യാജഏറ്റുമുട്ടല്‍ ആണെന്നു വിശ്വസിക്കുന്നില്ല. എന്നാല്‍ സിപിഐ ആവശ്യപ്പെട്ട മജിസ്റ്റീരിയല്‍ അന്വേഷണം നടക്കണമെന്നും ജയരാജന്‍ കോഴിക്കോട് പറഞ്ഞു. യുഎപിഎ കരിനിയമമെന്ന് നിയമമന്ത്രി എ.കെ.ബാലനും ആവർത്തിച്ചു‍. അറസ്റ്റിലായ സിപിഎം പ്രവര്‍ത്തകര്‍ക്ക് എതിരെയുള്ള എഫ്ഐആര്‍ നിയമ വ്യവസ്ഥ പാലിച്ചാണോയെന്നു പരിശോധിക്കും. ഇരുതലമൂര്‍ച്ചയുള്ള വാളാണ് യുഎപിഎ. നിയമം ദുരുപയോഗിക്കാന്‍ പാടില്ല. എഫ്െഎആർ ഇടുമ്പോള്‍ ശ്രദ്ധ വേണമെന്നും മന്ത്രി തിരുവനന്തപുരത്ത് പറഞ്ഞു.
രാഷ്ട്രീയമായ നിർദേശങ്ങൾക്കു വിധേയമായല്ല പൊലീസ് പ്രവർത്തിക്കുന്നതെന്നു വ്യക്തമാക്കുന്ന സംഭവങ്ങളാണു സംസ്ഥാനത്തു നിരന്തരമായി നടക്കുന്നതെന്ന് മുസ്‍ലിം ലീഗ് ദേശീയ ജനറൽ സെക്രട്ടറി പി.കെ.കുഞ്ഞാലിക്കുട്ടി വ്യക്തമാക്കി. യുഎപിഎയെ തുടക്കം മുതലേ മുസ്‌ലിം ലീഗ് എതിർത്തിട്ടുണ്ട്. പാർലമെന്റിൽ പ്രതിപക്ഷ കക്ഷികൾക്കൊപ്പം യുഎപിഎയെ എതിർത്തവരാണ് ഇടതുപക്ഷം. ഇപ്പോൾ അവരുടെ സർക്കാരിനു കീഴിൽ യുഎപിഎ ചുമത്തി അറസ്റ്റ് നടക്കുകയാണ്. യുഎപിഎയെ അനുകൂലിക്കുന്നവർ പോലും കോഴിക്കോട്ട് യുവാക്കളെ അറസ്റ്റ് ചെയ്തതിനെ അനുകൂലിക്കുമെന്നു തോന്നുന്നില്ല. മാവോയിസ്റ്റുകളെ കൈകാര്യം ചെയ്ത രീതിയിലും പൊലീസ് വീഴ്ച വരുത്തി.
വാളയാറിൽ നീചമായ കാര്യത്തിനു വലിയ സഹായമാണ് പൊലീസ് ചെയ്തതെന്നും കുഞ്ഞാലിക്കുട്ടി വ്യക്തമാക്കി. മാവോയിസ്റ്റ് ലഘുലേഖ സൂക്ഷിച്ച സംഭവത്തിലാണ് സിപിഎം പ്രവർത്തകരായ യുവാക്കൾക്കെതിരെ പൊലീസ് യുഎപിഎ ചുമത്തി കേസെടുത്തത്. പാർട്ടി പ്രവർത്തകർ അറസ്റ്റിലായത് സിപിഎം കോഴിക്കോട് ജില്ലാ നേതൃത്വത്തെയും പ്രതിരോധത്തിലാക്കി. ജില്ലാ നേതൃത്വത്തിന്റെ പ്രതിഷേധംകൂടി കണക്കിലെടുത്താണു മുഖ്യമന്ത്രി ഡിജിപിയോട് അടിയന്തര വിശദീകരണം തേടിയത്.
