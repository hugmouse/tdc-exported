
# 即位パレード、奉祝行進曲「令和」演奏…最大計２万６０００人で警備

Published at: **2019-11-07T10:17:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50220/)

　政府の皇位継承式典事務局は７日、１０日に行われる天皇陛下の即位に伴うパレード「祝賀御列（おんれつ）の儀」で、新たに作曲された奉祝行進曲「令和」を披露すると発表した。
　行進曲「令和」は、宮内庁楽部指揮者の北原幸男さんが作曲したもので、パレード出発地の皇居・宮殿と到着地の赤坂御所で演奏される。平成のパレードの時は、当時の楽部指揮者が作曲した行進曲「平成」が披露された。
　パレードは１０日午後３時に始まり、天皇、皇后両陛下を乗せたオープンカーが約３０分かけて皇居から赤坂御所まで約４・６キロを走行する。荒天が見込まれる場合は中止となるが、小雨程度であれば実施する。前日の９日午後６時半に実施の可否を発表する。中止の場合、予備日は設けない。
◇
　パレードの警備について、警察庁は７日、警視庁を中心に最大計約２万６０００人態勢であたると発表した。危険物の投げ込みを防ぐため、沿道で観客の手荷物を検査するほか、コース周辺で大規模な交通規制を行う。
