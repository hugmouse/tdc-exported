
# 老朋友話題葷素不忌　費玉清怕不能給江蕙幸福

Published at: **2019-11-07T13:41:00+00:00**

Author: **陳于嬙, 李鍾泉**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent042/)

費玉清與江蕙擁多年交情，兩人情誼堅貞，聊天話題葷素不拘，如今，江蕙送上熱氣球造型的「小歌號」，一句「別丟包我」，讓費玉清忍不住在台上開玩笑：「讓她（江蕙）破費了，講出這話耐人尋味啊，不好意思告訴大家，最近健康檢查，前列腺有點阻塞，不能給她性福啊！」短短幾句話，逗樂歌迷。
一唱完〈感情放一邊〉，費玉清侃侃而談與江蕙的交情，指兩人一見面、話題一開就沒完沒了，有時，江蕙開車經過費玉清家樓下，打電話劈頭就問：「你在家嗎？我去你家聊天。」就算沒有天天聯絡，但都把對方放在很重要的位置。
費玉清指，演藝圈總愛配對，江蕙的地位與歌唱實力，偶讓人感覺壓力大，但江蕙實則感性，一見面說天道地，如今，他與江蕙一樣選擇「封麥」，告別演藝圈，費玉清幽默說：「退休後，二姐也別丟包我啊！」兩人隔空對談，依舊有哏，對「江費配」，費玉清雖自嘲：「心有餘而力不足。」實則博君一笑，與江蕙有一定的相知。
