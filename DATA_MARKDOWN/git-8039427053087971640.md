
# కోహ్లీ, అక్షయ్, సానియాలను నామినేట్ చేసిన సింధు

హైదరాబాద్: తెలుగునాట టీఆర్ఎస్ ఎంపీ సంతోష్ రావు చేపట్టిన ‘గ్రీన్ ఇండియా ఛాలెంజ్‌’ ప్రచారం జోరుగా జరుగుతోంది. ఇప్పటికే పలువురు సినీ, రాజకీయ ప్రముఖులు ఈ ఛాలెంజ్‌ను స్వీకరించి.. మరింత మందిని భాగస్వాములను చేస్తున్నారు. తాజాగా బ్యాడ్మింటన్ స్టార్ పీవీ సింధు ఈ ఛాలెంజ్‌ను స్వీకరించారు. పుల్లెల గోపిచంద్ నేషనల్ అకాడమీలో మూడు మొక్కలు నాటిన ఆమె.. ఈ కార్యక్రమానికి మరో ముగ్గురిని నామినేట్ చేశారు. టీమిండియా కెప్టెన్ విరాట్ కోహ్లి, బాలీవుడ్ నటుడు అక్షయ్‌కుమార్‌, టెన్నీస్ స్టార్ సానియా మీర్జాలను నామినేట్ చేసినట్లు ట్విట్టర్ వేదికగా తెలిపారు. ఈ సందర్భంగా ఎంపీ సంతోష్‌కుమార్‌కు అభినందనలు తెలిపారు.
