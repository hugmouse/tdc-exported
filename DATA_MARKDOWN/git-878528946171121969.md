
# 嵐・相葉　ＦＮＳ歌謡祭の司会に…「重大発表」の予告はこれだった

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012858148.shtml)

　嵐の相葉雅紀（３６）が、年末恒例のフジテレビ系「ＦＮＳ歌謡祭」の司会を初めて務めることが７日、明らかになった。同日放送された同局系「ＶＳ嵐」（木曜、後７・００）で発表された。相葉が音楽番組の司会を単独で務めるのは、２０１６年のＮＨＫ「紅白歌合戦」以来３年ぶり２度目。永島優美アナウンサー（２７）がパートナーを務める。
　嵐のメンバーが勢揃いしたエンディングで、相葉は「今回から僕と永島さんの２人で司会を務めます。ＦＮＳ歌謡祭、４５周年という節目の年なんで、ここでフレッシュな力を！僕らが」と緊張ぎみに意気込みを語った。
　この日の新聞のテレビ欄でも「嵐から重大発表」と告知され、「ＶＳ嵐」の公式ツイッターでも、番組放送前から「嵐から重大発表も　お楽しみに」と予告ツイート。番組が始まると「相葉から重大」と“見出し”風に画面右上に予告が表示され、ナレーションでも「重大発表」とあおる形に。ネット上は「何？！」「重大発表気になる」「嵐の重大発表見るためだけにテレビつけた」などとザワつき、中には「相葉ちゃんの結婚だと予想」の声も上がっていた。
