
# ​សមត្ថកិច្ច​កំពុង​ធ្វើ​កោសល្យ​វិច្ច័យ​លើ​សាក​សព ស្ត្រី​ជន​ជាតិ​អង់​គ្លេស ដើម្បី​រក​មូល​ហេតុ​ពិត​ប្រាកដ​

Published at: **2019-11-01T07:45:27+00:00**

Author: **ញឹម សុផល**

Original: [RFI](http://km.rfi.fr/cambodia/UK-young-girl-case-in-SHV-01-11-2019)

លោក​ឧត្តម​សេនីយ​ទោ ជួន ណារិន ស្នង​ការ​ប៉ូលីស​ខេត្ត​ព្រះ​សីហនុ បាន​ប្រាប់​វិទ្យុ​បារាំង​អន្តរ​ជាតិ RFI នៅ​ថ្ងៃ​សុក្រ​នេះ​ថា សមត្ថកិច្ច​នៅ​មិន​ទាន់​កំណត់​ពី​មូល​ហេតុ​ច្បាស់​លាស់​យ៉ាង​ណា​នៅ​ឡើយ​ទេ​ជុំវិញ​កា​ស្លាប់​របស់​ស្ត្រី​វ័យ​ក្មេង​ជន​ជាតិ​អង់​គ្លេស ឈ្មោះ Amelia Bambridge ខណៈ​ពេល​នេះ ភាគី​ពាក់​ព័ន្ធ កំពង់​ធ្វើ​កោសល្យ​វិច្ច័យ ដោយ​មាន​ការ​ចូល​រួម​ពី​ក្រុម​គ្រួសារ​ឬ​ស្ថាន​ទូត​អង់​គ្លេស​ប្រចាំ​កម្ពុជា​។ តាម​លោក ជួន ណារិន សមត្ថ​កិច្ច​របស់​លោក បាន​ធ្វើ​ការ​ពង្រាយ​កម្លាំង​សន្តិ​សុខ​ជា​យូរ​មក​ហើយ ដល់​កន្លែង​ដែល​មាន​ប្រជា​ពលរដ្ឋ​និង​ភ្ញៀវ​ទេសចរ ទៅ​ដល់ ដោយ​មាន​ការ​ផ្សព្វ​ផ្សាយ ខិត​ប័ណ្ឌ​ចល័ត នៅ​តាម​តំបន់​សប្បាយ​ភាព​នា​នា​។ សូមស្តាប់ ប្រសាសន៍ លោកឧត្តមសេនីយ៍ ជួន ណារិន៖ 01/11/2019
សូមបញ្ជាក់ថា នារី​វ័យក្មេង​ជនជាតិ​អង់គ្លេសឈ្មោះ Amelia Bambridge អាយុ២២ឆ្នាំ ដែល​បាត់​ខ្លួន​នៅ​កោះរ៉ុង​ ខេត្ត​ព្រះសីហនុ ​ជាង​មួយ​សប្តាហ៍​កន្លងទៅ ត្រូវ​បានរកឃើញថា​បានស្លាប់ ដោយ ​សាកសព​ស្ត្រី​រងគ្រោះ​ បានអណ្តែត ​នៅលើ​លម្ហ​សមុទ្រ​ក្នុងភូមិ​សាស្ត្រ​ស្រុក​កោះ​ស្តេច​ខេត្តកោះកុង​ ចម្ងាយ​ពី​កោះរ៉ុង​ប្រមាណ​ជាង​៦០​គីឡូម៉ែត្រ។​ ​កង​ទ​ព័​ជើង​ទឹកកម្ពុជា​ បាន​ឲ្យ​ដឹងថា​ សាកសព​ស្ត្រី រងគ្រោះ ​ត្រូវបាន​កម្លាំង​ទីបញ្ជាការ​ជួរមុខ​អគ្គលេខាធិការដ្ឋាន​នៃ​គណៈកម្មាធិការ​ជាតិ​សន្តិសុខ​លម្ហ​សមុទ្រ ​ក្រោម​កិច្ចសហការ​ជាមួយ ក្រុមហ៊ុន ជី ធី​វី​ស៊ី​ស្ពី​ត​ប៊ូ​ត​កម្ពុជា​របស់លោក​ឧក​ញ៉ា​ទៀ ​វិចិត្រ​ បាន​ប្រទះឃើញ​សាកសព​ស្ដ្រី​ជនជាតិ​អង់គ្លេស​ឈ្មោះ Amelia Bambridge រូបនេះ អណ្ដែត​នៅលើ​ផ្ទៃសមុទ្រ​នៅ​ចំណុច​( ខន ដរ ) ក្បែរ​ព្រំដែន​សមុទ្រ​កម្ពុជា ថៃ៕
