
# ผลสอบ'3เทศกิจ'คลองเตย ประพฤติชั่วร้ายแรงสั่งให้ออก

Published at: **2019-11-06T03:55:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/bangkok/740211)


        "รองจั้ม" เผยผลสอบ "3 เทศกิจคลองเตย" พบเข้าข่ายประพฤติชั่วอย่างร้ายแรงจริง ทำให้เสื่อมเสียต่อภาพพจน์ของกรุงเทพมหานคร จึงสั่งให้ออกจากราชการเมื่อวันที่ 17 ต.ค.ที่ผ่านมา
      
คืบหน้าการสอบสวนวินัยร้ายแรงพนักงานเทศกิจเขตคลองเตย 3 นาย ที่ปรากฎในคลิปที่แชร์ในสื่อสังคมออนไลน์เมื่อวันที่ 11 ต.ค. ซึ่งมีลักษณะเหมือนเรียกรับเงินพ่อค้าขายผลไม้รถเข็นบริเวณใต้สถานี BTS อโศก เบื้องต้นเมื่อวันที่ 17 ต.ค. นายสุชัย อมรดารารัตน์ ผู้อำนวยการเขตคลองเตย ได้มีคำสั่งให้เจ้าหน้าที่ทั้ง 3 นายออกจากราชการไว้ก่อน พร้อมตั้งคณะกรรมการสอบสวนวินัยร้ายแรงล่าสุดเมื่อวันที่ 6 พ.ย. นายสกลธี ภัททิยกุล รองผู้ว่าราชการกรุงเทพมหานคร กล่าวถึงกรณีนี้ว่า ขณะนี้การสอบสวนวินัยร้ายแรงได้เสร็จสิ้นแล้วและในกระบวนการสอบสวนได้ให้โอกาสผู้ถูกกล่าวหาได้ชี้แจงและนำพยานหลักฐานที่เป็นประโยชน์กับฝ่ายเจ้าหน้าที่ทั้ง 3 นายมาชี้แจงภายในเวลาที่พอสมควร แต่เนื่องจากภาพที่ปรากฎออกมาชัดเจนและรายละเอียดข้อหักล้างของเจ้าหน้าที่ทั้ง 3 นายฟังไม่ขึ้น การกระทำของเจ้าหน้าที่ทั้ง 3 ราย เข้าข่ายประพฤติชั่วอย่างร้ายแรงจริง ทำให้เสื่อมเสียต่อตำแหน่งและเกียรติศักดิ์ของเจ้าหน้าที่เทศกิจและเสียหายต่อภาพพจน์ของกรุงเทพมหานคร เจ้าหน้าที่ทั้ง 3 นายจึงมีความผิดจริงและผู้อำนวยการเขตคลองเตยในฐานะที่มีอำนาจบรรจุแต่งตั้งจึงลงโทษให้ออกจากราชการทั้ง 3 นาย มีผลตั้งแต่วันที่ 17 ต.ค.ที่ผ่านมา ที่ได้ให้ออกจากราชการไว้ก่อน โดยได้เปิดโอกาสให้ทั้ง 3 นายอุทธรณ์ต่อกรุงเทพมหานครได้ภายใน 1 เดือน.
