
# マラソン札幌開催、都が容認検討

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/general/2019/11/01/0012837384.shtml)

　２０２０年東京五輪のマラソン・競歩について、東京都が国際オリンピック委員会（ＩＯＣ）の提案した札幌移転案の容認を検討していることが３１日、都関係者への取材で分かった。反対姿勢の継続による大会運営への支障を考慮。ＩＯＣのコーツ調整委員長、大会組織委員会の森喜朗会長、小池百合子都知事、橋本聖子五輪相が出席する１日の４者協議までに最終判断する見通し。コーツ氏は結論に至るかどうか問われ「間違いない」と述べた。パラリンピックのマラソンは東京で開催の方針。
　都は小池知事や主要幹部が３１日深夜まで都庁内で対応を協議。最終的な結論は１日午前に改めて検討することになった。
