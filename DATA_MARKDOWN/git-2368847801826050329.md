
# 規模6.0地震襲智利 聖地亞哥建物晃動

Published at: **2019-11-05T02:16:56+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/05/a102700273.html)

【新唐人北京時間2019年11月05日訊】美國地質調查所（USGS）4日表示，智利中部沿海城市科津波（Coquimbo）附近發生規模6.0地震，震央在港都瓦巴萊索（Valparaiso）東北方143公里，震源深度49公里。
地震造成首都聖地亞哥（Santiago）建築物晃動。不過，民眾因政府打算調漲地鐵費用仍持續抗議行動。該行動自上月（10月）中旬爆發以來，迅速蔓延至全國，智利總統皮涅拉（Sebastian Pinera）雖然取消漲價，也解散內閣，仍無法滅火，連亞太經濟合作會議（APEC）與Cop 25氣候變遷大會，都被迫取消。
智利國家緊急辦公室表示，尚未收到人員傷亡或建築、基礎建設受損的通報。
