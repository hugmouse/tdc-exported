
# ​មេ​បាតុ​ករ​ហុង​កុងគាំទ្រ​មាតុភូមិនិវត្តន៍​របស់​ លោកសម រង្ស៊ី

Published at: **2019-11-01T07:41:17+00:00**

Author: **វណ្ណ សុគន្ធា**

Original: [RFI](http://km.rfi.fr/cambodia/hk-activist-appeals-for-international-observation-toward-rainsy-return-01-11-2019)


        នៅ​តែ​៨ថ្ងៃ​ទៀត​ប៉ុណ្ណោះ គឺ​ដល់​ថ្ងៃ​កំណត់​ដែល​លោក​សម រង្ស៊ី និង​សហ​ការី​វិល​ត្រឡប់​ចូល​ប្រទេស​កម្ពុជា​វិញ​តាម​ការ​សន្យា​។ មុន​ដល់​ថ្ងៃ​កំណត់​មក​វិញ លោក សម រង្ស៊ី បាន​ចេញ​នូវ​វីដេអូ​ឃ្លីប​ជា​បន្ត​បន្ទាប់​ទាំង​ជា​ភាសា​ខ្មែរ និង​អង់​គ្លេស ដោយ​បាន​ប្រកាស​ដាក់​អាយុ​ជីវិត និង សេរីភាព​ផ្ទាល់​ខ្លួន ដើម្បី​ចូល​មក​កម្ពុជា​ឱ្យ​បាន​។ ចំណែក មេ​បាតុករ​វ័យ​ក្មេង​នៅ​ហុងកុង​វិញ ក៏​បាន​គាំទ្រ​នូវ​ដំណើរ​មាតុ​ភូមិ​និវត្តន៍​នេះ​ផង​ដែរ ហើយ​ថែម​ទាំង​បាន​ស្នើ​ឱ្យ​សហគមន៍​អន្តរជាតិ​ជួយ​ឃ្លាំ​មើល​ព្រឹត្តិការណ៍​នេះ​ថែម​ទៀត​ផង​។ 
      
តាម​រយៈ​បណ្តាញ​សង្គម​ធ្វីត​ធ័រ (Twitter) កាល​ពី​ថ្ងៃទី៣១ ខែតុលា លោក Joshua Wong មេ​ដឹក​នាំ​ក្រុម​បាតុ​ករ​នៅ​ទីក្រុង​ហុង​កុង បាន​សរសេរ​ដូច្នេះ​ថា៖«លោកហៃ វណ្ណា នឹង​ត្រឡប់​ទៅ​កាន់​ប្រទេស​កម្ពុជា​ជាមួយ​មេដឹក​នាំ​របស់​គាត់ គឺ​លោក សម រង្ស៊ី ដែល​ជា​ប្រធាន​ស្តីទី​គណ​បក្ស​សង្រ្គោះ​ជាតិ​នៅ​ថ្ងៃទី០៩ ខែវិច្ឆិកា ដើម្បី​ស្តារ​លទ្ធិ​ប្រជា​ធិប​តេយ្យ និង​សិទ្ធិ​មនុស្ស​ឡើង​វិញ​នៅ​កម្ពុជា​។ ដូច្នេះ​ហើយ ខ្ញុំ​សូម​អំពាវ​នាវ​ឱ្យ​សហ​គមន៍​អន្តរ​ជាតិ​ធ្វើ​ការ​ឃ្លាំ​មើល​ពី​ព្រឹត្តិ​ការណ៍​នេះ​ផង»។ ជាមួយ​នឹង​ការ​អំពាវ​នាវ​យ៉ាង​នេះ លោក​មេ​បាតុករ​វ័យ​ក្មេង​រូប​នេះ ក៏​បាន​បង្ហាញ​ការ​គាំទ្រ​ដល់​ដំណើរ​មាតុ​ភូមិ​និវត្តន៍​របស់​លោក សម រង្ស៊ី ផង​ដែរ ដោយ​បញ្ជាក់​ពី​ជំហរ​ឈរ​ខាង​លោក សម រង្ស៊ី ព្រោះ​លោក​ចាត់​ទុក​ថា ដំណើរ​វិល​ត្រឡប់​របស់​លោក សម រង្ស៊ី​នេះ គឺ​ដើម្បី​លទ្ធិ​ប្រជា​ធិប​តេយ្យ​។
ឆ្លើយ​តប​នឹង​ការ​អំពាវ​នាវ​នេះ លោក សុខ ឥសាន អ្នក​នាំ​ពាក្យ​គណ​បក្ស​ប្រជាជន​កម្ពុជា បាន​ប្រតិកម្ម​យ៉ាង​ខ្លី​ថា រដ្ឋា​ភិបាល និង អាជ្ញា​ធរ​កម្ពុជា​មិន​ចាប់​អារម្មណ៍​បន្តិច​សោះ​ឡើយ​ចំពោះ​ការ​លើក​ឡើង​របស់​មេបាតុករ​ហុង​កុង​រូប​នេះ ហើយ​ថា ស្ថាន​ភាព​នៅ​កម្ពុជា​មិន​ដូច​នៅ​ហុង​កុង​នោះ​ទេ​។ ចំពោះ​ផែន​ការ​វិល​ត្រឡប់​របស់​លោក សម រង្ស៊ី​វិញ លោក​សុខ ឥសាន បាន​គូស​បញ្ជាក់​ជា​ថ្មី​ថា អាជ្ញាធរ​កម្ពុជា​នឹង​ចាត់​វិធាន​ការ​ទៅ​តាម​ផ្លូវ​ច្បាប់ និង បាន​អះអាង​ថា វាសនា​របស់​លោក សម រង្ស៊ី នឹង​ត្រូវ​ស្ថិត​នៅ​ក្នុង​ពន្ធនាគារ បើ​ហ៊ាន​ចូល​មក​កម្ពុជា​។
ក៏​ប៉ុន្តែ លោក សម រង្ស៊ី បាន​ប្រកាស​នៅ​ក្នុង​ឃ្លីប​វីដេអូ​ដែល​បាន​ចេញ​ផ្សាយ​ជា​បន្ត​បន្ទាប់​ទាំង​ជា​ភាសា​ខ្មែរ និង​អង់​គ្លេស​ថា លោក​សុខ​ចិត្ត​បូជា​ជីវិត និង សេរី​ភាព​ផ្ទាល់​ខ្លួន​របស់​លោក ដើម្បី​ប្តូរ​យក​លទ្ធិ​ប្រជា​ធិប​តេយ្យ​ពិត​ប្រាកដ និង សិទ្ធិ សេរី​ភាព​ជូន​ដល់​ប្រជា​ពលរដ្ឋ​ខ្មែរ​តាម​រយៈ​ការ​វិល​ចូល​មក​កម្ពុជា​វិញ​នៅ​ថ្ងៃទី០៩ ខែវិច្ឆិកា ខាង​មុខ​នេះ​។
ដំណើរ​មាតុ​ភូមិន៍​នេះ លោក សម រង្ស៊ី បាន​គូស​បញ្ជាក់​ថា នឹង​បង្ហាញ​ឱ្យ​ដឹង​លម្អិត​នៅ​ថ្ងៃទី០៦ ខែវិច្ឆិកា​។ ប៉ុន្តែ​លោកស្រី មួរ សុខហួរ ដែល​ជា​អនុ​ប្រធាន​អតីត​គណ​បក្ស​សង្រ្គោះ​ជាតិ បាន​អះអាង​ឱ្យ​ដឹង​ថា លោក​សម រង្ស៊ី នឹង​ចូល​មក​តាម​រយៈ​ប្រទេស​ថៃ ហើយ​អះអាង​ទៀត​ថា ពលករ​ខ្មែរ​នៅ​ប្រទេស​ថៃ​នឹង​រៀប​ចំ​ខ្លួន​ជា​ក្រុមៗ ដើម្បី​អម​ដំណើរ​លោក សម រង្ស៊ី ចូល​មក​ប្រទេស​កម្ពុជា​៕
