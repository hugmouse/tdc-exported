
# ラストアイドル、Xマスにデビュー2周年コンサート “ラスアイのスゴ席”も用意

Published at: **2019-11-07T10:00:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148197/full/)

　アイドルグループ・ラストアイドルが、12月25日に神奈川・カルッツかわさきで『ラストアイドル2周年記念クリスマスコンサート』を開催することが決定した。2017年12月20日にシングル「バンドワゴン」でデビューしてから2周年を祝した公演となる。 　クリスマスに行われるコンサートには、1期生、2期生、2期生アンダー（※長期休養中の本城珠莉亜は不参加）が出演。昨年同様、入場時にはメンバー直筆の2周年ありがとうコメント入りのスペシャルフォトカード（クリスマスバージョン全45種／ランダム）1枚がプレゼントされ、終演後はメンバーがロビーでお見送り（中学生メンバーは除く）。クリスマスプレゼント企画も予定されている。 　チケットは「ラストアイドル ファミリー Mobile」会員先行で申し込みスタート。モバイル先行申込者の中から抽選で、客席中央ブロックの1列目から3列目の58席限定“スゴイ近いから、スゴイ見やすくて、スゴイレスがもらえる席”「ラスアイのスゴ席」が当たる。当選者はチケット発券のタイミングで確認できる。
