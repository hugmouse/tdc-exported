
# វៀតណាម ថ្កោលទោសការជួញដូរមនុស្ស និងអំពាវវនាវឱ្យប្រទេសក្នុងតំបន់ជួយនាំគ្នាទប់ស្កាត់

Published at: **2019-11-02T12:49:12+00:00**

Author: **ហេង ឧត្តម**

Original: [RFI](http://km.rfi.fr/vietnam-uk-rfi-oudom-02-11-2019)

          ថ្ងៃ​សៅរ៍​ ទី​២វិច្ឆិកា ក្រសួង​ការ​បរទេស​វៀតណាម​ ចេញ​មុខ​ថ្កោល​ទោស​អំពើ​ជួញ​ដូរ​មនុស្ស​ ហើយ​បាន​អំពាវវនាវឱ្យ​ប្រទេស​ក្នុង​តំបន់​អាស៊ាន​និងអន្តរជាតិ ចូលរួម​នាំគ្នា​ជួយ​ទប់​ស្កាត់​ការ​ជួញដូរមនុស្ស។ ប្រតិកម្ម​របស់​ក្រសួង​ការបរទេស​វៀតណាម​យ៉ាង​
          ដូច្នេះ បន្ទាប់ពីប៉ូលីសស៊ើប​អង្គេតរបស់​អង់គ្លេស​ បញ្ជាក់ពីថ្ងៃសុក្រទី១វិច្ឆិកា ថាសាកសព​មនុស្សទាំង៣៩នាក់​ដែលស្លាប់ក្នុងរថយន្តកាំមីញ៉ូង នៅក្បែរទីក្រុងឡុងនោះ គឺភាគច្រើនជាជនជាតិវៀតណាម។
        
      
ការ​ស៊ើប​អង្កេតលម្អិត​​ រក​អត្តសញ្ញាណ​សាកសព កំពុងតែ​បន្ត​នៅទៀតឡើយ តែលទ្ធផល​បឋម​តាមរយៈ​ការ​ផ្គូផ្គង់​អាដេអ៊ីន ​ប៉ូលីសផ្នែក​​វិទ្យាសាស្ត្រ​អង់គ្លេស​ ​បង្ហាញ​ថា អ្នក​ស្លាប់ភាគច្រើន​ក្នុង​ចំណោម​ជនភៀសខ្លួនទាំង​៣៩នាក់នោះ ជាជនជាតិ​វៀតណាម។ ​នគរបាល​អង់គ្លេស ​​កំពុងតែ​ទាក់ទង​ក្រុម​គ្រួសារ​សព​ និង​អាជ្ញាធរទូត​​វៀតណាម​នៅ​ក្រុង​ឡុង​ដ៍ និង​ក្រុង​ហាណូយ។ ប៉ូលីសអង់គ្លេស ក៏​ប្រកាស​ឱ្យ​អ្នក​ដែល​សង្ស័យ​បាត់​សមាជិកគ្រួសារ ឬ​ក៏​អាច​មាន​តម្រុយ​ឈាន​ទៅ​កំណត់​អត្តសញ្ញាណ​របស់​ជន​រងគ្រោះ​បាន សូម​ឱ្យ​ទាក់ទង​ផ្តល់​ព័ត៌មានបន្ថែមទៀត​ ទៅឱ្យ​​ខាង​ប៉ូលីស​ស៊ើប​អង្កេត​។​
សាក​សព​មនុស្ស​៣៩នាក់ ប្រុស​៣១ ស្រី​៨នាក់ ត្រូវ​គេ​រកឃើញ​ស្លាប់​ កាលពី​ថ្ងៃទី​២៣តុលា ក្នុង​ឡាន​កាំមីញ៉ុង​ ក្បែរ​ទីក្រុង​ឡុង ប្រទេស​អង់គ្លេស។ ឡាន​កុងតឺន័រ ដឹង​​មនុស្ស​ភៀសខ្លួន​នោះ ចេញពី​ប្រទេស​បែលស៊ិក​។ ពេលនេះ​ ប៉ូលីសអង់គ្លេស បាន​ស្រាវជ្រាវ​​និង​ឃាត់​ខ្លួន​​បុរស​ជន​​ជាប់​សង្ស័យ​ពីរនាក់ ម្នាក់​វ័យ​៤០ឆ្នាំ ឈ្មោះ Ronan Hughes និង​ប្អូន​ប្រុស​ Christopher អាយុ​៣៤ឆ្នាំ ដើមកំណើត​មក​ពី​អៀកឡង់​ខាងជើង។  
ក្នុង​សំណុំ​រឿង​ដដែល​​នេះ ​តុលាការ​អៀកឡង់ កាលពី​ថ្ងៃ​សុក្រ​ ទី១វិច្ឆិកា ក៏​បាន​ចោទប្រកាន់​បុរស​ម្នាក់ វ័យ​២៣ឆ្នាំ ពីបទល្មើស​ជួ​ញ​ដូរ​មនុស្ស និង​បទ​ឧក្រិដ្ឋឃាតកម្មអចេតនា ​ទៅលើ​ជនភៀសខ្លួន​ទាំង​៣៩នាក់។ អយ្យការ​អៀកឡង់ បញ្ជាក់​ប្រាប់​កាសែត​បារាំង​AFP ថា យុវជន​នេះ ត្រូវឃុំខ្លួន​បណ្តោះអាសន្ន​ រហូត​ដល់ថ្ងៃទី​១១វិច្ឆិកា។​ ប៉ុន្តែ ប៉ូលីស​អង់គ្លេស កំពុង​ស្នើ​សុំ​ធ្វើប័ត្យាប័ន​ជន​ត្រូវ​​ចោទ​រូបនេះ ទៅប្រឈម​សំណាញ់​ច្បាប់​អង់គ្លេសតែម្តង​​។​ ចំណែក​នគរបាល​​វៀតណាម​វិញ  ក៏​បានប្រកាស​ពីការចាប់ឃាត់​ខ្លួន​មនុស្សប្រុស​​ពីរនាក់ ផង​ដែរ​។​
នៅក្នុង​សន្នីសីទកាសែត​ថ្ងៃ​សៅរ៍​ ទី​២វិច្ឆិកាអ្នក​នាំពាក្យ​ក្រសួង​ការបរទេស​វៀតណាម​ លោកស្រី ឡេ ធី ធូ ហាំង (Lê Thị Thu Hằng) ​បាន​អំពាវវនាវឱ្យ​ប្រទេស​គ្រប់​ជ្រុងជ្រោយ​នៃពិភពលោក​ ចូលរួម​នាំគ្នា​ជួយ​ទប់​ស្កាត់​អំពើ​​ជួញដូរមនុស្ស ដើម្បី​ជៀសវាង​សោកនាដកម្ម​សម្លាប់​មនុស្ស​ ដូច​ករណីមួយ​​នេះទៀត៕​
