
# 泰南2檢查站遇襲15死4傷 傷亡近年最慘重

Published at: **2019-11-06T12:58:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911060107.aspx)

（中央社曼谷6日綜合外電報導）泰國軍方發言人今天表示，暴力頻傳的南部地區有兩處檢查站遇襲，疑為穆斯林叛亂分子犯案，造成至少15人死亡，另有4人受傷，死亡人數為近年來最多。
過去15年來，人口以馬來穆斯林為主的泰南地區頻傳血腥叛亂，已經造成7000多人喪生。
泰國南部軍方發言人普拉莫（Pramote Prom-in）告訴法新社，好戰分子昨晚在也拉省（Yala）攻擊兩處配置民防志願軍的檢查站，他們在一群村民停下來聊天時，朝志願軍開槍。
普拉莫說：「12人死於攻擊現場，2人送醫後不治，另有1人今晨死亡。」他又說，叛亂分子拿走檢查站的M-16步槍與霰彈槍。他說，檢查站周圍地區已經封鎖，目前正在進行鑑識調查。
叛亂分子尋求讓與馬來西亞接壤且具獨特文化的地區自治，一直在對抗以佛教徒為主的泰國。
泰國深南地區專家董恩（Don Pathan）說，昨天的攻擊是「很長一段時間以來」規模最大的協同襲擊。這些事件在最近幾年變得比較少見，但攻擊力道變得「更加密集」。董恩說：「這提醒我們他們還在那裡。」（譯者：張曉雯/核稿：徐睿承）1081106
