
# ద్రావిడ్‌కు మరోసారి నోటీసులు

Published at: **2019-11-01T03:59:06+00:00**

Author: ****

Original: [Andhrajyothy](https://www.andhrajyothy.com/artical?SID=943116)

Ph: 88865 47666ప్రవేశం ఉచితం ADVT
న్యూఢిల్లీ: పరస్పర విరుద్ధ ప్రయోజనాల అంశంలో టీమిండియా మాజీ క్రికెటర్‌ రాహుల్‌ ద్రావిడ్‌కు మరోసారి నోటీసులు అందాయి. ఈనెల 12న వ్యక్తిగతంగా హాజరై వివరణ ఇవ్వాలని బీసీసీఐ ఎథిక్స్‌ ఆఫీసర్‌ డీకే జైన్‌ నోటీసులు జారీ చేశారు. సెప్టెంబరు 26న ద్రావిడ్‌ వ్యక్తిగతంగా విచారణకు హాజరయ్యాడు. జాతీయ క్రికెట్‌ అకాడమీ (ఎన్‌సీఏ) డైరెక్టర్‌గా, ఇండియా సిమెంట్స్‌ ఉపాధ్యక్షుడిగా రాహుల్‌ రెండు రకాలైన పాత్రలు పోషిస్తున్నాడని బోర్డు అంబుడ్స్‌మన్‌కు ఫిర్యాదు అందింది.
