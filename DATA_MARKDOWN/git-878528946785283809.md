
# 三原じゅん子氏　「政権を握っているのは総理大臣だけ」投稿が反響

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012849037.shtml)

　参議院議員の三原じゅん子氏が４日、ツイッターで「政権を握っているのは総理大臣だけ」と投稿。１５００件を超えるコメント、１８００を超えてリツイートされるなど反響を呼んでいる（４日、午後９時半現在）。
　発端は三原氏が１日に同党女性局の組織図をツイッター上で公開したことから始まったとみられる。三原氏の投稿にさまざまな返信があり、三原氏が答えるなか、「皆様、コメントが凄いことになってますね～（笑）では、正確に申し上げましょう。政権を握っているのは総理大臣だけですよ」とツイートした。
　続くツイートで三原氏は「皆様、大元のツイートから離れていってる～（笑）私が政権を握ってるのですか～！？もしそうなら凄いな」と投稿。三原氏は続くツイートで「こんなに沢山のコメントありがとうございます」などと応じていた。
