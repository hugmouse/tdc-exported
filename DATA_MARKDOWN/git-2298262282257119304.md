
# Как показать / проявить смысл данных

Визуализация данных – одна из важнейших составляющих управления бизнесом. Алексей Колоколов, эксперт и бизнес-тренер, написал для менеджеров пошаговое руководство для создания информационных отчетов. Книга называется «Дашборд для директора», и это первое издание по визуализации данных в бизнесе на русском языке. Вся информация, которая есть в публикации, является практическим опытом Алексея Колоколова в торговых компаниях, банках и на производствах в течение десяти лет. Он работал с продуктами от Microsoft, Qlik, Tableau, IBM, настроил сотни управленческих отчетов и, аккумулировав свои знания, описал универсальные правила визуализации и работы с требованиями бизнес-заказчиков. 
В книге даны схемы, как из большого объема данных сделать простой и понятный план для принятия решений. Автор разбирает реальные примеры аналитических отчетов, обучая отличать основные показатели для бизнеса (KPI) от других статистических данных компании и оформлять их в доступные логичные схемы. Воспользовавшись советами бизнес-тренера, специалист научится адаптировать аналитические отчеты под целевую аудиторию, создавая документ необходимого формата для оперативного отчета, клиента или руководства компании. В книге также рассматриваются популярные на рынке BI-программы, объясняются их отличия и преимущества, параметры выбора под конкретный бизнес-проект. 
Книга «Дашборд для директора» – подробная инструкция по созданию профессиональных информационных панелей отчетности (дашбордов) на разных уровнях управления и принятия решений. Она поможет менеджерам освоить правила составления оперативных, стратегических и аналитических отчетов на всех этапах создания, а также их структурирования.
