
# イラン核開発「第４段階」発動へ…米制裁に対抗

Published at: **2019-11-04T11:24:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191104-OYT1T50126/)

　【テヘラン＝水野翔太】イラン核合意を離脱して制裁を再開したトランプ米政権への対抗策として、イランは近く、核開発を拡大させる「第４段階」の措置を発動する見通しだ。高濃縮ウランの生産など大幅拡大に踏み切れば、米欧との対立が先鋭化する恐れがある。
　イラン政府は９月上旬、欧州の経済支援がなければ、６０日後に「第４段階」を発動すると宣言した。イラン政府関係者によると、５日前後が期限となる。最高指導者ハメネイ師は３日の演説で制裁を続ける米国を批判しており、何らかの拡大に踏み切るのは確実とみられている。
　ただ、イランは５月以降、ウラン濃縮の制限を２度にわたって撤廃したが、９月上旬発動の「第３段階」では、濃縮に使う遠心分離器などの研究開発を加速するとの内容にとどめ、米欧への刺激を避けた。
　これに関連し、イラン原子力庁のサレヒ長官は４日、中部ナタンツの核施設で記者会見し、新型の遠心分離器を開発したと明らかにした。ＡＰ通信によると、核合意で使用が認められている旧型の５０倍の濃縮能力だという。「第３段階」の研究加速の結果だとみられる。
　サレヒ氏は、濃縮ウラン保有量が５００キロ・グラムに達し、１日５キロ・グラムずつ増えていることも明らかにした。核合意により、イランの保有量は３００キロ・グラム（六フッ化ウラン換算）に制限されていた。１日の生産量は、第３段階を発動する前の１０倍以上に達しているといい、遠心分離器の改良が影響しているとみられる。
