
# 联赛杯-利物浦补时进球5-5 点球战5-4淘汰阿森纳

Published at: **2019-11-01T02:51:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/10/31/sports-1572533383)

【新浪体育31日讯】(英国当地时间30日19:30)，2019/20赛季英格兰联赛杯第4轮一场焦点战在安菲尔德球场展开争夺，利物浦主场总分10比9淘汰阿森纳，18岁巴西新援马蒂内利梅开二度，红军两度扳平，奥里吉第94分钟救主，双方5-5战平。点球大战，利物浦5-4险胜阿森纳晋级。利物浦近5个赛季9战阿森纳取得5胜4平保持不败，近7个主场对阵阿森纳取得5胜2平。这是双方联赛杯第15次交手，此前分别取得5胜4平5负。双方历史交锋227场，利物浦88胜61平78负占据上风。两队全部轮换11人，17岁荷兰新援范登博格首次代表利物浦出场。厄齐尔本赛季第3次代表阿森纳出战。利物浦(4-3-3)：62-凯莱赫；76-威廉姆斯，12-戈麦斯，72-范登博格，7-米尔纳；15-张伯伦(81’,68-奇里韦拉)，20-拉拉纳，8-凯塔(55’,48-琼斯)；67-埃利奥特，27-奥里吉，24-布鲁斯特。阿森纳(4-2-3-1)：26-马丁内斯；2-贝莱林，20-穆斯塔菲，16-霍尔丁，31-科拉希纳茨(83’,3-蒂尔尼)；11-托雷拉(72’,8-塞瓦略斯)，28-维尔洛克；15-奈尔斯，10-厄齐尔(65’,29-贡多齐)，77-萨卡；35-马蒂内利。
