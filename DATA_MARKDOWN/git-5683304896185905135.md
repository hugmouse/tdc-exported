
# പ്രവാസികൾക്ക് യാത്രാക്കൂട്ടായി വീണ്ടും കാർഡ് ബോർഡ് പെട്ടി

Published at: **2019-11-04T13:58:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/global-malayali/gulf/2019/11/04/cardboard-boxes-sale-in-kuwait.html)

കുവൈത്ത് സിറ്റി ∙ ഗൾഫുകാരുടെ യാത്രാകൂട്ടായി കാർഡ് ബോർഡ് പെട്ടി തിരിച്ചുവരുന്നു. ഏതെങ്കിലും കമ്പനി അവരുടെ ഉത്പന്നങ്ങൾ എത്തിക്കാനായി ഉപയോഗിച്ച് മുക്കും മൂലയും പൊളിഞ്ഞവയല്ല. പുതുപുത്തൻ കാർഡ് ബോർഡ് പെട്ടികൾ. ബഖാലകളിൽ ഇവ വാങ്ങാൻ കിട്ടും. ഗൾഫ് യാത്രികർ സാധനങ്ങൾ നിറച്ചുവരുന്ന പെട്ടികൾക്ക് ഇക്കാലങ്ങൾക്കിടയിലുണ്ടായ പരിണാമങ്ങൾ ഏറെയാണ്. ഗൾഫ് ബൂം തുടങ്ങിയ 1970കളിലെ പെട്ടിയല്ല ഗൾഫുകാരൻ ഇപ്പോൾ കൊണ്ടുവരുന്നത്.
ചുവപ്പും കറുപ്പും നിറമുള്ള പെട്ടിയായിരുന്നു അക്കാലത്ത് പ്രചാരത്തിൽ. ചിലർക്ക് ഇഷ്ടം തകരപ്പെട്ടികളോടായിരുന്നു. കൈവശം സ്യൂട്ട്കെയ്സും പാസ്പോർട്ട് സൂക്ഷിക്കുന്നതിനുള്ള കൊച്ചുബാഗുമുണ്ടാകും. യാത്രതുടങ്ങി വീട്ടിലെത്തും വരെ പാസ്പോർട്ട് ബാഗ് നിലത്ത്‌വക്കില്ലായിരുന്നു അവർ. ക്രമേണ പെട്ടിക്കുവേണ്ടി പണം ചെലവഴിക്കുന്നത് ഗൾഫുകാർ ഒഴിവാക്കി. കാർഡ്ബോർഡ് പെട്ടികളായി ആശ്രയം. വയറ് തുറിക്കുംവിധം സാധനങ്ങൾ നിറച്ച് പ്ലാസ്റ്റിക് കയറുകൊണ്ട് കെട്ടിവരിഞ്ഞ് ഒന്നോരണ്ടോ പെട്ടികളുമായായി യാത്ര. അപ്പോഴും സ്യൂട്ട്കെയ്സിനെ കൈവിട്ടിരുന്നില്ല. ന്യൂജെൻ ട്രൻഡ് തുടങ്ങിയതോടെ കാർഡ്ബോർഡ് പെട്ടികളോട് വിരക്തിയായി. ഒറ്റനിറമുള്ള പെട്ടികൾ ഗൾഫുകാരനൊപ്പം ചേർന്നു.
പിന്നെപ്പിന്നെ അത് ചക്രങ്ങളോട് കൂടിയ ബാഗുകളായി. കൈയിലുള്ള സ്യൂട്ട്കെയ്സ് ചക്രമുള്ള ട്രോളിബാഗിലേക്കും മാറി. ഹാൻഡ് ബഗേജ് ഷോൾഡർ ബാഗിലേക്ക് മാറിയതും വളരെപ്പെട്ടെന്ന്. പാസ്പോർട്ട് ബാഗാണെങ്കിൽ അതിനും മുൻപെ സ്ഥലം കാലിയാക്കിയിരുന്നു. പകരം പാസ്പോർട്ട് കീശയിൽ വെക്കുന്ന രീതിയായി. ഇപ്പോൾ ഇതാ വീണ്ടും കാർഡ് ബോർഡ് പെട്ടി തിരിച്ചുവന്നിരിക്കുന്നു. ഇവയിൽ കമ്പനികളുടെ പേരും വിവരവുനൊന്നുമില്ല. കാണാനും മൊഞ്ചുള്ളവ.
