
# ​លោក សម រង្ស៊ី ​ប្រកាស​សន្យា​ដំឡើង​ស័ក្តិ និង​ប្រាក់​ខែ​ ដល់​កង​ទ័ព​ដែល​ឈរ​ខាង​លោក​ តែ​ក្រសួង​ការពារ​ជាតិ​ថា ជា​រឿង​បោក​ប្រាស់​ដ៏​អាក្រក់​បំផុត​

Published at: **2019-11-04T07:22:14+00:00**

Author: **វណ្ណ សុគន្ធា**

Original: [RFI](http://km.rfi.fr/cambodia/sam-rainsy-willing-to-grand-military-if-they-support-but-is-it-true-04-11-2019)

លោក សម រង្ស៊ី តាម​រយៈ​វីដេអូ​ឃ្លីប​មួយ​នៅ​ព្រឹក​ថ្ងៃ​ចន្ទ​នេះ បាន​ចេញ​មក​បញ្ជាក់​ជា​ថ្មី​ថា ផែន​ការ​មាតុ​ភូមិ​និវត្តន៍​របស់​លោក នៅ​ថ្ងៃទី០៩ វិច្ឆិកា​ខាង​មុខ​នេះ កំពុង​ប្រព្រឹត្ត​ទៅ​ដោយ​រលូន និង គ្មាន​ការ​ប្រែ​ប្រួល​ឡើយ បើ​ទោះ​បី​អាជ្ញាធរ​កម្ពុជា​បាន​ប្រុង​ប្រៀប​លក្ខណៈ​សព្វ​បែប​យ៉ាង ដើម្បី​ត្រៀម​ចាប់​ខ្លួន​រូប​លោក​ក៏​ដោយ​។ ចំណែក ឆ្លើយ​តប​ទៅ​នឹង​ការ​ហាម​ប្រាម​មិន​ឲ្យ​ក្រុម​ហ៊ុន​ហោះ​ហើរ​ណា​ដឹក​រូប​លោក​មក​កម្ពុជា​វិញ​នោះ លោក សម រង្ស៊ី បាន​គូស​បញ្ជាក់​ថា លោក​នឹង​ស្វែង​រក​មធ្យោ​បាយ​ចូល​មក​កម្ពុជា​ឱ្យ​ទាល់​តែ​បាន តាម​ផែន​ការ​គ្រោង​ទុក​។ បន្ថែម​លើ​នេះ និង ជា​ថ្មី​ម្តង​ទៀត លោក​ប្រធាន​ស្តីទី​អតីត​គណបក្ស​សង្រ្គោះ​ជាតិរូប​នេះ បាន​អំពាវ​នាវ​ដល់​កង​ទ័ព​កម្ពុជា​កុំ​ឱ្យ​ស្ដាប់​តាម​បញ្ជា​របស់​លោក​នាយក​រដ្ឋ​មន្រ្តី ហ៊ុន សែន និង ងាក​មក​ចូល​រួម​នឹង​ដំណើរ​មាតុ​ភូមិ​និវត្តន៍​របស់​លោក​វិញ​។ មិន​តែ​ប៉ុណ្ណោះ លោក សម រង្ស៊ី ថែម​ទាំង​សន្យា​ដំឡើង​ឋានន្តរ​ស័ក្តិ និង​ប្រាក់​ខែ ក៏​ដូច​ជា​ផ្តល់​រង្វាន់​ដល់​កង​ទ័ព​ណា​ដែល​ចូល​រួម​ជាមួយ​ក្រុម​របស់​លោក​នា​ពេល​ខាង​មុខ​ផង​ដែរ​។ សូមស្ដាប់ប្រសាសន៍របស់ លោក សម រង្ស៊ី៖ 04/11/2019
ឆ្លើយ​តប​នឹង​ការ​លើក​ឡើង​របស់​លោក សម រង្ស៊ី​នេះ លោក ឈុំ សុជាត អ្នក​នាំ​ពាក្យ​ក្រសួង​ការពារ​ជាតិ​កម្ពុជាបានលើកឡើងយ៉ាងដូច្នេះថា៖
