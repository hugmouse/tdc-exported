
# 女性を困惑させ、強引に修理「瓦が持ち上がった」…台風被害住宅

Published at: **2019-11-01T02:57:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191101-OYT1T50128/)

　台風１５号の強風で被害を受けた高齢女性宅の屋根を強引に修理したとして、千葉県警は３１日、東京都渋谷区の家屋診断・修理会社の社長（２６）（東京都江戸川区）を特定商取引法違反（威迫困惑）の疑いで逮捕した。
　発表によると、社長は仲間と共謀し、台風１５号が上陸した翌日の９月１０日、千葉市の７０歳代女性の自宅を訪れ、「瓦が持ち上がって直さなきゃいけない」などと言って女性を困惑させ、屋根を修理した疑い。女性は２５万円を請求され、その場で支払ったという。
　県警は９月、同社従業員の男（２７）を同容疑で逮捕。社長の関与が浮上し、指名手配して行方を追い、静岡県内の宿泊施設で見つけた。
　県警には２９日までに、家屋修理などを巡る被害相談が約１６０件寄せられた。県警は、同社が関わった事案もあるとみて調べている。
