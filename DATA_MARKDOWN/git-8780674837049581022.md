
# മാവോയിസ്റ്റുകളുടെ പോസ്റ്റ്‌മോർട്ടത്തിനിടെ അപ്രതീക്ഷിത സംഭവങ്ങൾ,അന്വേഷണം തുടങ്ങി...

Published at: **2019-11-04T05:21:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://localnews.manoramaonline.com/thrissur/local-news/2019/11/04/thrissur-attapady-mavoist-death-postmortem-controversy.html)

തൃശൂർ ∙ അട്ടപ്പാടിയിൽ കൊല്ലപ്പെട്ട മാവോയിസ്റ്റുകളുടെ പോസ്റ്റ്‌മോർട്ടത്തിനിടെ മെഡിക്കൽ കോളജ് മോർച്ചറിയിൽ നടന്ന അപ്രതീക്ഷിത സംഭവങ്ങളെക്കുറിച്ച് ആരോഗ്യ വകുപ്പ് അന്വേഷണം തുടങ്ങി.പോസ്റ്റ്‌മോർട്ടം നടന്ന ഹാളിലേക്കു പ്രവേശിക്കാൻ അനുവദിക്കാതെ പുറത്താക്കിയെന്ന് ചൂണ്ടിക്കാട്ടി 3 ഡിവൈഎസ്പിമാർ നൽകിയ പരാതിയുടെ അടിസ്ഥാനത്തിൽ മന്ത്രിയുടെ നിർദേശപ്രകാരം ആരോഗ്യ വകുപ്പ് സെക്രട്ടറിയാണ് അന്വേഷണത്തിന് ഉത്തരവിട്ടത്. പോസ്റ്റ്‌മോർട്ടത്തിൽ പങ്കെടുത്ത എല്ലാ ഡോക്ടർമാരിൽ നിന്നും വിശദ‍ീകരണം തേടിയശേഷം റിപ്പോർട്ട് സമർപ്പിക്കാൻ പ്രിൻസിപ്പലിനു നിർദേശം നൽകി. 
കൊല്ലപ്പെട്ട മാവോയിസ്റ്റുകളായ മണിവാസകം, കാർത്തി, രമ, അരവിന്ദ് എന്നിവരുടെ മൃതദേഹങ്ങൾ പോസ്റ്റ്‌മോർട്ടത്തിനായി എത്തിച്ചതു മുതൽ പൊലീസും ഫൊറൻസിക് സംഘവും തമ്മിൽ അഭിപ്രായ ഭിന്നതകൾ ഉടലെടുത്തിരുന്നതായി വിവരമുണ്ട്. പോസ്റ്റ്‌മോർട്ടം ഹാളിൽ പ്രവേശിക്കാൻ ഡിവൈഎസ്പിമാർ ശ്രമിച്ചെന്നും ഫൊറൻസിക് സംഘം അനുവദിച്ചില്ലെന്നുമായിരുന്നു പുറത്തറിഞ്ഞ വിവരം. പോസ്റ്റ്‌മോർട്ടത്തിനു ശേഷം മാധ്യമങ്ങൾക്കു വിവരങ്ങൾ ചോർന്നുകിട്ടിയതിൽ ഉന്നത ഭരണനേതൃത്വം അമർഷം പ്രകടിപ്പിക്കുകയും ചെയ്തു. ഇതിനുത്തരവാദി ആരെന്നതു സംബന്ധിച്ചും പൊലീസും ഫൊറൻസിക് സംഘവും തമ്മിൽ അസ്വാരസ്യങ്ങളുണ്ടായതായി വിവരമുണ്ട്. 
അതേസമയം, മാവോയിസ്റ്റുകളുടെ പോസ്റ്റ്‌മോർട്ടം റിപ്പോർട്ട് മെഡിക്കൽ കോളജ് ഫൊറൻസിക് വിഭാഗം ഇന്നു പൊലീസിനു കൈമാറിയേക്കും. കോടതിക്കും പൊലീസിനും മരിച്ചയാളുടെ നിയമാനുസൃത അവകാശിക്കുമാണ് റിപ്പോർട്ട് നൽകുക. മണിവാസകത്തിന്റെ മൃതദേഹം മാത്രമാണ് ഇതുവരെ ബന്ധുക്കൾ തിരിച്ചറിഞ്ഞിട്ടുള്ളത്. ഇവരാകട്ടെ റിപ്പോർട്ടിനായി ഫോറൻസിക് വിഭാഗത്തിൽ അപേക്ഷ സമർപ്പിച്ചിട്ടുമില്ല. കോടതിനിർദേശത്തെ തുടർന്ന് നാലു പേരുടെയും മൃതദേഹങ്ങൾ മോർച്ചറിയിൽ സൂക്ഷിച്ചിരിക്കുകയാണ്.
