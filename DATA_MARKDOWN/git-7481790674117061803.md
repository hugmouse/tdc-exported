
# Ｊ２長崎・高田明社長、退任へ…経営危機で１７年に就任

Published at: **2019-11-02T04:22:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/soccer/20191102-OYT1T50175/)

　サッカーＪ２、Ｖ・ファーレン長崎の運営会社の高田明社長（７０）が退任する意向を固めたことが２日、分かった。近く発表する予定。
　運営会社は、前経営体制下にあった２０１７年シーズン開幕前に経営危機が表面化。通販大手「ジャパネットホールディングス」（長崎県佐世保市）が子会社化して立て直しを図ることになり、「ジャパネットたかた」創業者の高田氏が同年４月、社長に就任した。チームは同年のＪ２リーグで昇格を決め、１８年は初めてＪ１リーグで戦った。
