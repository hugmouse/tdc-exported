
# 田代まさし容疑者、裏切り逮捕　薬物では９年ぶり４度目　依存症テーマに講演活動も…

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012855063.shtml)

　元タレントの田代まさし（本名政）容疑者（６３）が６日、都内で、覚せい剤取締法違反の疑いで、宮城県警に逮捕された。県警は認否を明らかにしていない。逮捕容疑は、８月２３日に宮城県塩釜市の宿泊施設で、また１１月６日に杉並区で、それぞれ覚醒剤を所持した疑い。田代容疑者は過去にも薬物で３度（再逮捕は除く）の逮捕歴があり、２０１４年７月に服役を終えて出所していた。最近では薬物依存の恐ろしさを伝える活動を行っていたが、支援者らを裏切る形となってしまった。
　宮城県警塩釜署によると、田代容疑者が利用した塩釜市の宿泊施設から「客が変な物を忘れていった」と８月２４日に通報があり事件が発覚。塩釜市で所持していた案件で１１月６日に、東京・杉並区内で逮捕する際にも、覚醒剤を所持していたため、現行犯逮捕した。
　田代容疑者を乗せたシルバーのワゴン車は午後７時ごろ、塩釜署に到着。後部座席から出てきた田代容疑者は、茶色いジャケットを頭から深くかぶり、警察官に誘導されながら署に入った。
　田代容疑者は０１年、０４年と覚せい剤取締法違反で逮捕され、１０年にはコカインを所持したとして麻薬取締法違反で逮捕され、その後に覚せい剤取締法違反（共同所持）と大麻取締法違反（共同所持）で再逮捕された。薬物では９年ぶり４度目（再逮捕は除く）の逮捕となる。３度有罪となり、計７年服役。１４年７月２日に出所していた。
　現在は薬物更生施設「日本ダルク」に所属しており、自身の経験を踏まえて、薬物依存の実態を伝えるシンポジウムに出席するなどしていた。最近ではＹｏｕＴｕｂｅのチャンネル「ブラックマーシー」で元クールスの佐藤秀光とのトークを公開。逮捕の前日の今月５日には、「刑務所はつらいヨ　よそ見編」と題して、見学者の女性の網タイツに気を取られてよそ見するなどして、「警告」が３回たまり、「懲罰房」に入れられたことなどをぼやいていた。
　また、ツイッターは今月１日が逮捕前最後の更新。女子大生のリンクを貼り、「ミニスカポリス的な衣装でダンスしてるので一見の価値あり！嗚呼！こんな子に取調べられたかったなあ！ｗｗｗ」とコメントしていた。
　田代容疑者は今年７月４、１１日の２回にわたってＮＨＫＥテレのバラエティー「バリバラ」に薬物依存についての講師として出演。収録後の６月末には「依存症から回復している姿を見せていきたい」とコメントを発表していた。
　田代容疑者の逮捕を受けて、ＮＨＫはＥテレ「バリバラ」の公式サイトから今年７月の同容疑者の出演回を削除した。
