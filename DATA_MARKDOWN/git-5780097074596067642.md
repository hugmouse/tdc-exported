
# Brexit៖ រដ្ឋាភិបាលអង់គ្លេស ពន្យារពេលផ្សាយរបាយការណ៍ស្តីពីការជ្រៀតជ្រែករបស់រុស្ស៊ី ទៅលើប្រជាមតិ

Published at: **2019-11-05T12:36:35+00:00**

Author: **ហេង ឧត្តម**

Original: [RFI](http://km.rfi.fr/rapport-brexit-johnson-rfi-oudom-05-11-2019)


        
          រដ្ឋាភិបាល​​អង់គ្លេស​  បាន​ពន្យារពេល​​ការចេញ​​ផ្សាយ​របាយការណ៍​​ស៊ើបអង្កេតស្តីពី​ការ​ជ្រៀតជ្រែក​របស់​រុស្ស៊ី​​ក្នុង​ការ​បោះឆ្នោត​​ប្រជាមតិ​អង់គ្លេសដើរ​ចេញពី​សហភាព​អឺរ៉ុប​ 
          កាលពីឆ្នាំ២០១៦
          ​។ រដ្ឋាភិបាលអង់គ្លេស ទំនង​ជាត្រូវ​លើកពេល​ផ្សាយ​របាយការណ៍​នេះ​ទៅ​ខែ​ធ្នូវិញ នៅពេល​បោះឆ្នោត​សភាចប់​​រួច​រាល់​​​។​  ការ​រាំង​ខ្ទប់និង​ពន្យារ​ពេល​ផ្សាយយ៉ាង​ដូច្នេះ គឺត្រូវ​គេ​រិះគន់​យ៉ាង​ចម្រូង​សម្រាស​ចំពោះ​ការ​សម្រេច​ចិត្ត​ខុស​ប្រក្រតី​​របស់​នាយករដ្ឋមន្ត្រី។ 
        
      
របាយការណ៍ស៊ើបអង្កេតនេះ មាន​កម្រាស​ជាង​៥០ទំព័រ​ ស្រាវជ្រាវ​រៀបចំ​ដោយក្រុម​​អ្នក​ជំនាញនិង​ភ្នាក់​ងារ​ភ្នាក់ស៊ើបការសម្ងាត់អង់គ្លេស​។ ការរបាយការណ៍​​អង្កេត​នេះ ពន្យល់​ជា​សំខាន់​ចំពោះ​ការ​ចោទប្រកាន់​ថា មាន​ការ​ជ្រៀត​ជ្រែង​របស់​រុស្ស៊ី​ក្នុង​ការ​បោះឆ្នោត​ប្រជាមតិ​ Brexit កាលពីឆ្នាំ​២០១៦ ហើយ​ឥទ្ធិពល​នៃការ​ជ្រៀតជ្រែង​នេះ ទៅលើ​លទ្ធផល​នៃការ​បោះឆ្នោត​នោះ។
តាម​ច្បាប់​ទម្លាប់​អង់គ្លេស, របាយការណ៍​ដែល​អាច​ប៉ះពាល់​ដល់​សន្តិជាតិ​យ៉ាង​ដូច្នេះ​ វា​អាចផ្សាយចេញទៅ​បាន គឺទាល់តែទទួលការអនុញ្ញាត ផ្តល់​ភ្លើងខៀវពីប្រមុខរដ្ឋាភិបាល ជា​មុន​សិន​។ តាំងពី​ខែ​​មីនាមកម្ល៉េះ​ ដែល​ក្រុម​​ភ្នាក់ងារ​ស៊ើប​ការ​សម្ងាត់អង់គ្លេស​បានសម្រេច​បញ្ចប់​របាយការណ៍ និងបាន​​​បញ្ចូន​បន្ត​ទៅកាន់​ការិយាល័យ​នាយករដ្ឋមន្ត្រី​រួច​ទៅហើយ តាំងពី​ពាក់​កណ្តាល​​ខែ​តុលា។ ហើយ​បើ​តាមភាព​ត្រឹមត្រូវ នាយករដ្ឋមន្ត្រីអង់គ្លេស​ គួរ​តែត្រូវ​​ផ្តល់​ភ្លើង​ខៀវ​ឱ្យ​ចេញ​ផ្សាយ​តាំងពីមុន​ដាច់​​ខែ​តុលា​មកម្ល៉េះ។ នេះ​បើ​តាម​កាសែត​អង់គ្លេស The Gardian។ ក៏ប៉ុន្តែ នាយករដ្ឋមន្ត្រីអង់គ្លេស លោក បូរីស ចនសុន ទំនង​ជាចង់​​រង់​ចាំបោះឆ្នោតសភាថ្ងៃទី១២ធ្នូ​ចប់​រួចហើយ ទើប​ឱ្យ​ផ្សព្វផ្សាយរបាយការណ៍​​។ ការ​សម្រេច​ដូច្នេះ គឺ​ហាក់​មិន​ទំនង​ ពីព្រោះថា ថ្ងៃអង្គារ៍​នេះ អង្គសភា​អង់គ្លេសនឹង​ត្រូវ​រំលាយ ហើយ​ចាប់​ផ្តើម​រៀបចំ​ការ​បោះឆ្នោត​សភា​ឡើងវិញ ដែល​ចាំបាច់​ត្រូវ​ធ្វើ​របាយការណ៍​បិទ​បញ្ចប់​អាណត្តិ​សភា និង​បូក​សរុប​បង្ហើយ​ការងារ​ទាំងឡាយ មុន​នឹងរៀបចំ​​បោះឆ្នោត​ជាថ្មី។​
ប្រធានគណកម្មាការសភាទទួល​បន្ទុក​កិច្ច​ការ​សម្ងាត់​និង​សន្តិសុខជាតិ លោក Dominic Grieve ​ ដែល​ត្រូវ​ពិនិត្យ​របាយការណ៍នោះ​ បាន​សម្តែង​ក្តីមិន​ពេញចិត្តចំពោះ​ការ​ពន្យារពេល​របស់​លោក ចនសុន​ ហើយរិះគន់ថ្កោលទោស​​ពីការ​សម្រេចចិត្ត​របស់​ប្រមុខ​រដ្ឋាភិបាល​ ដោយ​ចាត់ទុកថា ​មិន​សម​ហេតុ​ផល​​។ ទន្ទឹមគ្នា​នេះដែរ  តំណាង​រាស្ត្រ​ប្រឆាំងជាច្រើន​​ផ្សេងទៀត បាន​ចោទប្រកាន់​លោកនាយករដ្ឋមន្ត្រី បូរីស ចនសុន ថា​ខំប្រឹងរក​គ្រប់​មូលហេតុ​ដើម្បី​បិទខ្ទប់​ការ​ចេញផ្សាយ​របាយការ​ណ៍​អង្កេត​នេះ ពីព្រោះ​ថា បើ​វាចេញផ្សាយ​ទៅ នឹង​ធ្វើ​ឱ្យ​លោក​នាយករដ្ឋមន្ត្រី​ បាក់​មុខ​ បាត់​បង់​ប្រជាប្រិយភាព និង​អាច​នឹង​មាន​សំណាង​តិ​ចតួច​ ដើម្បី​ឈ្នះឆ្នោត​ នៅថ្ងៃទី​១២ធ្នូ ខាងមុខ៕​
