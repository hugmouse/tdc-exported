
# 绿驰通讯与中国移动签MoU 探讨彼此的产品分销

Published at: **2019-11-01T11:57:04+00:00**

Author: **Ahmad Naqib Idris**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E7%BB%BF%E9%A9%B0%E9%80%9A%E8%AE%AF%E4%B8%8E%E4%B8%AD%E5%9B%BD%E7%A7%BB%E5%8A%A8%E7%AD%BEmou-%E6%8E%A2%E8%AE%A8%E5%BD%BC%E6%AD%A4%E7%9A%84%E4%BA%A7%E5%93%81%E5%88%86%E9%94%80)

（吉隆坡1日讯）绿驰通讯（Green Packet Bhd）与中国移动通信集团设备有限公司（China Mobile Group Device Co Ltd）签署谅解备忘录（MoU），以寻求双方推广和分发彼此产品和服务的机会。
这家中国公司是国有的中国移动通信集团有限公司（China Mobile Communications Group Co Ltd）的子公司，其其他子公司包括中国最大的电信公司中国移动有限公司（China Mobile Ltd）。
绿驰通讯表示，谅解备忘录的有效期为1年，但经双方同意可续签。
除了推广和分发产品外，绿驰通讯和中国移动集团设备还将提供销售和服务援助，并提供技术支持和服务。
绿驰通讯预计，一旦双方达成最终协议，将对其业务做出积极贡献。
闭市时，绿驰通讯跌0.5仙或0.68%，至73仙，市值为6亿7931万令吉。
