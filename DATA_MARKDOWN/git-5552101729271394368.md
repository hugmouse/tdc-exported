
# 施壓台肥借錢買豪宅？　韓國瑜：應該要到按摩店紓壓

Published at: **2019-11-07T09:12:00+00:00**

Author: **謝文哲, 楊子磊**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107edi015?utm_source=facebook&utm_medium=mmpage)

全力衝刺2020總統大選的高雄市長韓國瑜，近日被週刊踢爆，在無業期間購入南港高達7,200萬元的豪宅，不只重創長期塑造的「庶民」形象，還有不少網友質疑炒房。民進黨立委林俊憲今（7）日加碼再爆，指韓國瑜曾找人施壓、要求台肥借款2成房價（約1,400萬元）。對此，韓國瑜稍早回應強調，當時已離開立法院十多年，如何向台肥施壓？他更自嘲「那時候的心情，應該要到按摩店紓壓」。
韓國瑜遭爆料曾購買7,200萬豪宅，後續爭議事件越演越烈，綠委林俊憲今（7）日加碼再爆，韓國瑜曾在2014年11月要求台肥借2成房價。台肥隔月立刻發存證信函回覆拒絕借款，沒想到韓國瑜再找政治人物施壓，讓台肥在2015年4月借出1,400多萬元，約為房價2成的款項。
台肥今（7）日特別澄清此事，強調當年因法令更改，豪宅原本能7成貸款降為5成，導致許多買家無法負擔，並轉向要求借款，為此台肥曾找律師研究、也請示過金管會，「對於有需要的住戶由台肥提供借款，這是通案，不只借給韓國瑜！」
稍早，韓國瑜回應買豪宅施壓疑雲，指當時他已離開立法院十幾年，也長期處於無業狀態，如何施壓？韓國瑜隨後更自嘲，「那時候的心情，應該要到按摩店紓壓」。他最後更請大家試想一下，「一個中年人長期沒有工作，離開立法院11、12年，轉頭可以去施壓？欲加之罪，何患無辭。」
