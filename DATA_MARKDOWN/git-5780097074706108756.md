
# ថៃ៖កិច្ចប្រជុំកំពូលអាស៊ានលើកទី៣៥ និងអាស៊ានជាមួយដៃគូនឹងគ្មានលោកដូណាល់ ត្រាំ

        ប្រទេស​ថៃ​ធ្វើ​ជា​ម្ចាស់ផ្ទះ​រៀបចំ​កិច្ចប្រជុំកំពូល​អាស៊ាន និង​អាស៊ាន​ជាមួយ​ប្រទេស​ដៃគូ​លើកទី៣៥ រយៈពេល ៣ថ្ងៃ​ដោយ​ចាប់​ផ្តើម​ពី​ថ្ងៃ​សៅរ៍ ទី២ ដល់​ថ្ងៃ​ចន្ទ ទី៤ វិចិ្ឆកា។ ក្រៅ​ពី​កិច្ចប្រជុំកំពូល​អាស៊ាន និង​អាស៊ាន​ជាមួយ​ដៃគូ ក៏​នៅ​មាន​គម្រោង​ជំនួប​ជា​លក្ខណៈ​ទ្វេភាគី ឬ​ត្រីភាគី​ជា​ច្រើន​ផ្សេង​ទៀត។ ក៏​ដូច​ជា​នៅ​ក្នុង​ជំនួបកំពូល​លើកទី៣៣​កាល​ពី​ឆ្នាំ ២០១៨ ក្នុង​ប្រទេស​សិង្ហបុរី​ដែរ លោក​ប្រធានាធិបតី​អាមេរិក​ដែល​ជា​តួអង្គ​ម្នាក់​ដែល​គ្រប់​គ្នា​បាន​រង់ចាំ​ជាង​គេ​បាន​លុប​ចោល​ការ​ចូលរួម​របស់​លោក​នៅ​នាទី​ចុង​ក្រោយ។ លោក​ត្រាំ​បាន​បញ្ជូន​ទីប្រឹក្សាសន្តិសុខ​ជាតិ​ថ្មី​លោក Robert O’Brien និង​រដ្ឋមន្រ្តី​ក្រសួង​ពាណិជ្ជកម្ម​លោក Wilbur Rossឲ្យ​ចូលរួម។ ភាព​ជា​ដៃគូ​សេដ្ឋកិច្ច​ថ្នាក់តំបន់​គ្រប់ជ្រុងជ្រោយ​ជា​ប្រធានបទ​អាទិភាព​ដែល​អាស៊ាន និង​ក្រុម​ប្រទេស​ដៃគូ​ចង់​សម្រេច​ឲ្យ​បាន​នៅ​ក្នុង​កិច្ចប្រជុំកំពូល​លើកទី៣៥ នេះ។
      
អ្វីៗ​បាន​ត្រៀម​រួច​រាល់​អស់​ហើយ​សម្រាប់​ប្រទេស​ថៃ​ក្នុង​នាម​ជា​ម្ចាស់ផ្ទះ​រៀបចំ​កិច្ចប្រជុំកំពូល​អាស៊ាន​លើកទី៣៥ និង​អាស៊ាន​ជា​មួយ​ប្រទេស​ដៃគូ។ ដើម្បី​ធានា​សន្តិសុខ និង​សុវត្ថិភាព​នៅ​ក្នុង​កិច្ចប្រជុំកំពូល​ដែល​មាន​មនុស្ស​ចូលរួម​ប្រមាណ​ជា ៣ ០០០នាក់ រដ្ឋអំណាច​ថៃ​មិន​ញញើត​នឹង​ប្រើ​មធ្យោបាយ​ចាស់ដៃ​ដោយ​ដាក់​ពង្រាយ​កម្លាំង​ប៉ូលិស​ចំនួន ១ ៥០០នាក់ និង​ភ្នាក់ងារ​សន្តិសុខ​ចំនួន ២ ០០០នាក់។ ការ​ប្រើ​មធ្យោបាយ​ចាស់ដៃ​លើក​នេះ គឺ​ដើម្បី​ចៀសវាង​កុំ​ឲ្យ​កើត​មាន​ព្រឹត្តិការណ៍​បំផ្ទុះ​គ្រាប់បែក​ដូច​នៅ​ក្នុង​កិច្ចប្រជុំ​ថ្នាក់រដ្ឋមន្រ្តី​ការបរទេស​អាស៊ាន​កាល​ពី​ខែ​សីហា​កន្លង​ទៅ។ គួរ​បញ្ជាក់​ថា កិច្ចប្រជុំកំពូល​ពេល​នេះ​ត្រូវ​បាន​រដ្ឋអំណាច​ថៃ​រៀបចំ​នៅ​ជាយ​ក្រុង​បាងកក ពោល​មិនមែន​ចំ​កណ្តាល​ក្រុង​បាងកក​ទេ។
កំណត់សម្គាល់​មួយ​ទៀត រដ្ឋាភិបាល​បាន​អនុញ្ញាត​ឲ្យ​មន្រ្តីរាជការ​ដែល​ធ្វើ​ការ​ក្នុង​ក្រុង​បាងកក​ឈប់សម្រាក​ចំនួន ២ថ្ងៃ គឺ​ថ្ងៃ​ចន្ទ ទី៤ និង​ថ្ងៃ​អង្គារ ទី៥ វិច្ឆិកា។ គោលដៅ គឺ​កាត់​បន្ថយ​ការ​កកស្ទះ​ចរាចរ​ក្នុង​ទីក្រុង​ដើម្បី​ជួយ​សម្រួល​ដល់​ការ​ធ្វើ​ដំណើរ​ចេញ ចូល​របស់​អ្នក​ដែល​មក​ចូលរួម​នៅ​ក្នុង​កិច្ចប្រជុំកំពូល​ពេល​នេះ។ ភាព​ជា​ដៃគូ​សេដ្ឋកិច្ច​ថ្នាក់តំបន់​គ្រប់ជ្រុងជ្រោយ​ត្រូវ​បាន​គេ​មើល​ឃើញ​ថា ជា​ប្រធានបទ​អាទិភាព​ដែល​អាស៊ាន និង​ក្រុម​ប្រទេស​ដៃគូ​ចង់​សម្រេច​ឲ្យ​បាន​លឿន​បំផុត។ មាន​ការ​លើកឡើង​ថា គឺ​បរិបទ​សង្រ្គាមពាណិជ្ជកម្ម​រវាង​អាមេរិក និង​ចិន​ដែល​កំពុង​រុញ​អាស៊ាន និង​ក្រុមប្រទេស​ដៃគូ​ឲ្យ​ពន្លឿន​កិច្ចចរចា​លើ​ភាព​ជា​ដៃ​គូ​សេដ្ឋកិច្ច​តំបន់​គ្រប់​ជ្រុងជ្រោយ​ដែល​មាន​សមាជិក​អាស៊ាន​ទាំង ១០ប្រទេស​បូក​នឹង​ដៃគូ​សន្ទនា ៦ប្រទេស គឺ ចិន ជប៉ុន កូរ៉េខាងត្បូង ឥណ្ឌា អូស្ត្រាលី និង​នូវែលសេឡង់។
ក្រៅ​ពី​ប្រធានរឿង​ស្តីពី​ភាព​ជា​ដៃ​គូ​សេដ្ឋកិច្ច​តំបន់​គ្រប់​ជ្រុងជ្រោយ កិច្ចប្រជុំ​ក៏​គ្រោង​ជជែក​លើ​ប្រធានបទ​តំបន់ និង​អន្តរជាតិ​ជា​ច្រើន​ផ្សេង​ទៀត រួម​មាន​បញ្ហា​នៅ​ឧបទ្វីបកូរ៉េ ជម្លោះ​នៅ​សមុទ្រ​ចិន​ខាងត្បូង បញ្ហា​ជនភៀសខ្លួន​រ៉ូហ៊ីងយ៉ា​នៅ​ភូមា ឬ​មួយ​ការ​បម្រែបម្រួល​អាកាសធាតុ​ជា​ដើម។ ដោយឡែក និង​ក៏​ដូច​ជា​នៅ​ក្នុង​កិច្ចប្រជុំកំពូល​លើកទី៣៣ ក្នុង​ប្រទេស​សិង្ហបុរី​កាល​ពី​ឆ្នាំ ២០១៨ ដែរ ប្រធានាធិបតី​អាមេរិក​លោក​ដូណាល់ ត្រាំ​មិន​បាន​មក​ចូលរួម​ទេ ដោយ​លោក​បាន​បញ្ជូន​ទីប្រឹក្សាសន្តិសុខជាតិ​ថ្មី និង​រដ្ឋមន្រ្តី​ក្រសួង​ពាណិជ្ជកម្ម​ឲ្យ​ចូលរួម។ ទោះ​ជា​យ៉ាង​នេះ​ក្តី អវត្តមាន​លោក​ត្រាំ​មិន​បាន​ធ្វើ​ឲ្យ​មាន​ការ​ផ្លាស់ប្តូរ​កម្មវិធី​នៃ​កិច្ចប្រជុំកំពូល​ពេល​នេះ​ទេ៕
