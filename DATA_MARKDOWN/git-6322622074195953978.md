
# 获2.62亿合约 商业机器扬至两个月高位

Published at: **2019-11-01T04:51:20+00:00**

Author: **Ahmad Naqib Idris**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E8%8E%B7262%E4%BA%BF%E5%90%88%E7%BA%A6-%E5%95%86%E4%B8%9A%E6%9C%BA%E5%99%A8%E6%89%AC%E8%87%B3%E4%B8%A4%E4%B8%AA%E6%9C%88%E9%AB%98%E4%BD%8D)

（吉隆坡1日讯）商业机器（Mesiniaga Bhd）获得一份价值2亿6156万令吉的合约，为隶属财政部的大马会计总处（AGD）兴建新基建设施后，上扬7.38%至两个月高位。
休市时，该股起9仙，挂1.31令吉，市值为7912万令吉。半天交投量有29万9800股，相比200天平均的5万1261.5股。
根据向大马交易所的报备，合约将从今日生效，至2025年4月杪。
根据合约，商业机器将设计、供应、调试和维护基础设施硬件和软件，并为联邦政府综合财务和会计管理系统（iGFMAS）以及AGD新基础设施的支持系统进行安装、迁移测试和调试。
供应和实施期限为10个月，而硬件保修期限为60个月，从2020年5月至2025年4月。
