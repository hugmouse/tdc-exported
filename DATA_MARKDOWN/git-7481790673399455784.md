
# 「いじめ自殺」第三者委の調査結果、高校側が内容不服と１年公表拒否

Published at: **2019-11-07T15:12:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50226/)

　長崎市の私立海星高２年の男子生徒（当時１６歳）が自殺した問題で、学校は７日、「いじめが自殺の要因」とする第三者委員会の調査結果を同校ホームページ（ＨＰ）で公表した。学校側は調査結果を不服として約１年間、公表を拒否していたが、遺族の意向を受けて公表に応じた。同時に、「（報告書の）自死との因果関係に論理的飛躍がある」などとする従来の見解をＨＰで表明した。
　生徒は２０１７年４月、長崎市内の公園で自殺しているのが見つかった。学校が設置した第三者委員会は昨年１１月、空腹時のおなかの音を侮辱するなどの同級生からのいじめが主な要因とする調査結果をまとめた。これに対し、学校は「説得力を欠く」として受け入れを拒否。遺族が７月、公表を申し入れていた。
　学校は７日にＨＰに掲載した文書の中で、「第三者委員会に対し根拠資料の開示や説明を求めたが、拒否され続けてきた」などとして、第三者委のメンバーに対し、近く長崎簡裁に調停を申し立てる意向を示した。
　男子生徒の父親（５２）は「公表が遅すぎる。二度と同じようなことが起きないようにするためにも、報告書の内容を真摯（しんし）に受け止めてほしい」と学校への不信感をあらわにした。
