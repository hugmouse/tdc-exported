
# លោកសមរង្ស៊ី ថា​មាន​គេ​លួច​ដូរ​សំបុត្រ​យន្តហោះ​របស់​លោក

Author: **អៀង សុខម៉ិញ**

​​លោក​សម រង្ស៊ី បាន​អះអាង​ប្រាប់​វិទ្យុ​បារាំង​អន្តរជាតិ​ នៅ​រសៀល​ថ្ងៃ​ព្រហស្បតិ៍​ទី ៧​វិច្ឆិកា ឆ្នាំ​២០១៩ ក្រោយ​ពី​ខក​ខាន​មិន​បាន​ឡើង​យន្តហោះថា  ​ក្រុមហ៊ុនអាកាសចរណ៍​​ថៃ បាន​ទទួល​បញ្ជា​ពី​ថ្នាក់​លើ​ ទើប​មិន​ព្រម​ឱ្យ​លោកចេញ​​ដំណើរ មិនមែន​ដោយ​បញ្ហា​សំបុត្រ​ឡើយ។ ​លោក​បាន​ទិញ​សំបុត្រ​ថ្មី​មួយទៀត ក្រោយ​ពីលោក​បាន​សម្គាល់​​ឃើញ​មាន​ការ​លួច​ប្តូរ​សំបុត្រ​យន្តហោះ​​របស់​លោក​។ ប្រសាសន៍​របស់​​លោក​ សម រង្ស៊ី 07/11/2019
