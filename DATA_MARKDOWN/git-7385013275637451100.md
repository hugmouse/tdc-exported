
# なみいる人気声優を引き離し1位になったのは、なんとあの人！ 波乱含みの「2019秋アニメ女性声優人気投票」結果発表！

Published at: **2019-11-03T15:00:00+00:00**

Author: ****

Original: [アキバ総研](https://akiba-souken.com/article/42215/)

アキバ総研が、2019年10月10日～24日の期間で開催していた公式投票企画「今期のイチオシは？ 2019秋アニメ女性声優人気投票」の投票が締め切られた。話題作の多い2019年秋アニメの中で、ユーザーからの期待値が高い女性声優さんはどなたになるのか？　では、さっそくベスト10を発表しよう。
1位：内田彩　1193票「Z/X Code reunion」リゲル役、「神田川JET GIRLS」翠田いのり役、「本好きの下剋上 司書になるためには手段を選んでいられません」フリーダ役
2位：茅野愛衣　822票「ソードアート・オンライン アリシゼーション War of Underworld」アリス役、「ちはやふる3」大江奏役、「真・中華一番！」メイリィ役
3位：悠木碧　412票「七つの大罪 神々の逆鱗」ディアンヌ役、「ラディアン(第2期)」メリ役、「僕のヒーローアカデミア(第4期)」蛙吹梅雨役
4位：沢城みゆき　363票「PSYCHO-PASS サイコパス３」唐之杜志恩役
5位：豊崎愛生　298票「慎重勇者 ～この勇者が俺TUEEEくせに慎重すぎる～」リスタルテ役
6位：M・A・O　266票「バビロン」瀬黒陽麻役、「七つの大罪 神々の逆鱗」メラスキュラ役
7位：戸松遥　247票「ソードアート・オンライン アリシゼーション War of Underworld」アスナ（結城明日奈）役、「俺を好きなのはお前だけかよ」［パンジー］三色院董子役
8位：花澤香菜　188票「ファンタシースターオンライン2 エピソード・オラクル」メルフォンシーナ役、「食戟のソーマ 神ノ皿」紀ノ国寧々役
9位：早見沙織　181票「魔入りました！入間くん」アザゼル・アメリ役
9位：高橋李依　181票「天華百剣 ～めいじ館へようこそ！～」桑名江役
見事1位の座に輝いたのは、内田彩さんとなった。全体的に票数が伸び悩んだ今回の投票であるが、唯一の1000票超えでの堂々1位となっている。今期は「Z/X Code reunion」にて主人公のリゲル役を務めるほか、「神田川JET GIRLS」「本好きの下剋上 司書になるためには手段を選んでいられません」などの作品にもメインキャストとして出演。アニメ作品ではややご無沙汰感もあったが、この秋一気に活躍の第一線に返り咲いた印象。多くのファンの票を集めての1位となった。
続く2位は、822票で茅野愛衣さん。今期は何と言っても、「ソードアート・オンライン アリシゼーション War of Underworld」の主役であるアリス役を熱演。いつもはどちらかと言えば、やさしくておっとり系のお姉さん役が多い茅野さんだが、本作では勇ましい女騎士・アリスを熱演している。このほかにも、久々の続編として注目度の高い「ちはやふる3」のメインキャラクターのひとりである大江奏役などで出演している。
さらに3位には、412票で悠木碧さんがランクインした。こちらも今期は久々の続編となる「七つの大罪 神々の逆鱗」でディアンヌ役を演じるほか、「ラディアン(第2期)」では主演級のメリ役、「僕のヒーローアカデミア(第4期)」では蛙吹梅雨（あすいつゆ）役など、人気作品には欠かせない存在の悠木さんらしく、さまざまな作品に登場して活躍している。
以下、沢城みゆきさんは、今期の注目作「PSYCHO-PASS サイコパス３」での唐之杜志恩役1本での4位ランクイン。豊崎愛生さんも、今期は「慎重勇者 ～この勇者が俺TUEEEくせに慎重すぎる～」のリスタルテ役1本で5位にランクインするなど、さすがの存在感を示した。6位のM・A・Oさんは、最近めきめきとその頭角を現している印象だが、今期も「バビロン」で瀬黒陽麻役、「七つの大罪 神々の逆鱗」でメラスキュラ役と、大作に相次いで登場。今年ご結婚された戸松遥さんは、鉄板とも言える人気を持つ、「ソードアート・オンライン アリシゼーション War of Underworld」のアスナ役のほか、「俺を好きなのはお前だけかよ」では、毛色が全く異なる［パンジー］三色院董子役を熱演している。
8位の花澤香菜さん、9位の早見沙織さんは、ランク上位常連のベテランだが、今期は主演級の出演作品が少ないことが影響したのか、やや順位を下げた印象。前期は「からかい上手ずの高木さん2」で大人気を博した高橋李依さんも、今期はやや出演が少なく、10位という結果になった。
