
# 欧冠-半场2乌龙+连追3球! 切尔西4-4九人阿贾克斯

Published at: **2019-11-07T03:00:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/06/sports-1573052280)

【网易体育6日讯】欧冠联赛小组赛第4轮H组的一出场较量，切尔西在斯坦福桥主场迎战阿贾克斯。上半场亚布拉罕和凯帕先后乌龙，切尔西半场1-3落后阿贾克斯。下半场切尔西围攻阿贾克斯，阿斯皮利奎塔、若日尼奥和詹姆斯先后破门，阿贾克斯布林德和费尔特曼连续红牌被罚下。最终这场进球大战切尔西和阿贾克斯4-4平。相比上周末的联赛首发，兰帕德只作出一个调整，左后卫位置阿隆索取代埃莫森先发。中前场依然是后腰位置若日尼奥和科瓦契奇的搭档，威廉和普利希奇各在一边，芒特居中，亚布拉罕一人突前。出场阵容：切尔西（4231）：凯帕/阿斯皮利奎塔、祖马、托莫里、阿隆索（46’詹姆斯）/若日尼奥、科瓦契奇（87’巴舒亚伊）/威廉、芒特（60’奥多伊）普利希奇/亚布拉罕。阿贾克斯（4231）：奥纳纳/马兹拉维、费尔特曼、布林德、塔利亚菲科/范德贝克、利桑德罗-马丁内斯/内雷斯（72’舒尔斯）、齐耶赫（72’阿尔瓦雷斯）、普罗梅斯/塔迪奇。
