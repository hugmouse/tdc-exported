
# 看好頂級投資市場 羅芙奧20週年頂級珠寶登場

Published at: **2019-11-05T02:23:00+00:00**

Author: **財經組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191105fin001/)

國內年度秋拍盛事—羅芙奧2019秋季拍賣將於12月初登場，除了一系列華人經典，中國、台灣與日本現當代藝術精品進行拍賣之外，今年適逢羅芙奧藝術集團20週年，羅芙奧也在「秋季手袋及精品鑑賞會」上呈獻時尚珍稀的愛馬仕包，更隆重推出多件頂級稀世珠寶，總金額逾新台幣20億元。
羅芙奧表示，此次精選展品包含了特殊皮革的柏金包、凱莉包，更有拍賣市場屢創佳績的限量版霧面SO BLACK ALLIGATOR短吻鱷魚皮30公分柏金包。當年是時任的愛馬仕創意總監尚-保羅·高緹耶（Jean Paul Gaultier）親自操刀設計，並且僅在2010年推出一季，爾後再也未曾出現過，流通量極少，至今亦成為拍賣場上最亮眼的逸品，是為藏家夢寐以求的罕見包款。
另外，本次秋拍羅芙奧也精選一套海瑞‧溫斯頓（HARRY WINSTON）設計的祖母綠珠寶套裝，由8顆共89.23克拉的祖母綠配鑲鑽石組成，另配以13.22及12.70克拉的祖母綠配鑽石耳環。每顆祖母綠皆呈現出飽滿的色澤且透亮濃綠，散發出深邃迷人的光采，而鑽石的排列則是呈現經典的海瑞‧溫斯頓設計，完美體現珍稀寶石的超凡美感以及獨特魅力。
為了讓更多頂級鑽石收藏家能夠珍藏稀世美鑽，羅芙奧今年也隆重推出一顆逾80克拉的豔彩黃色鑽石。一般而言，一顆切割及打磨後重逾100克拉的鑽石，其原石的重量應為成品的2倍以上，在處理過程中經過最精密的計算以及鬼斧神工的技術，傾全力控制最大克拉數並保留最美色澤，最後成就一顆逾80克拉的豔彩黃色鑽石。
