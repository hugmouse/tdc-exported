
# 直擊！沈玉琳背著老婆　大膽聞解婕翎腳味

Published at: **2019-11-06T10:40:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191106ent029/)

「Footer除臭襪」品牌為衝高買氣，祭出總獎項高達百萬的抽獎活動，首獎為價值30幾萬的勞力士名錶，並請來荒謬大師沈玉琳擔任活動推薦人，昨（5日）該品牌粉絲專頁直播公布第一波獎項得獎名單，大方送出最大獎LV精品包。
活動由沈玉琳擔任主持人，潘若迪和解婕翎擔任嘉賓，2個男人直接在節目中使出撩妹金句，沈玉琳甚至大膽聞解婕翎腳味，並讚「散發果香味」，讓解直呼「太誇張了！」。
有趣的是，潘若迪在現場帶2人跳有氧舞蹈，但沈玉琳招架不住，跳到氣喘吁噓，瞎扯眼前已經看到阿公；隨後解婕翎送上苦茶為沈玉琳消火解熱，潘若迪藉機爆料沈玉琳火氣大、要治療痔瘡，笑翻所有人。
此外，當天陸續送出3台伊萊克斯吸塵器、5台Dyson吹風機、1台Apple Watch、2台Philips氣炸鍋、2台Nintendo Switch，壓軸獎項為價值將近5萬元的 LV精品包，由沈玉琳親自call-out給幸運得主，送出驚喜大獎。
「Footer除臭襪」品牌週活動從1日起至11日止，官網及直營門市同步優惠，合作通路也另有活動，大獎包含勞力士名錶、iphone 11、現金10萬、台北關島雙人來回機票、Tokuyo按摩椅等，業者可以說是卯足全力。Footer行銷部經理徐煒傑表示：「這次為慶祝Footer邁向第7年，2019更跨足越南市場，成為台灣除臭襪品牌中，第一家開發越南市場的公司，意義非凡；所以今年特別加碼單筆消費滿2,000就可參加抽獎活動，獎項全是國際知名品牌，希望能藉此回饋老顧客，同時也吸引新客目光焦距」。
