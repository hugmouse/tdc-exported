
# 공군 '골든 아이'에 임지훈·박인용 선발

Published at: **2019-11-05T18:33:08+00:00**

Author: **조선일보 양승식 기자**

Original: [Chosun](https://news.chosun.com/site/data/html_dir/2019/11/06/2019110600267.html)

입력 2019.11.06 03:33
공군은 올해 최우수 방공무기통제사인 '골든 아이(Golden Eye)'에 임지훈(29·학군 42기·왼쪽) 대위와 박인용(28·공사 62기) 대위를 선발했다고 5일 밝혔다.방공무기통제사는 영공을 침범하거나 한국방공식별구역(KADIZ)을 무단 진입한 타국 항공기에 대응해 아군 항공기를 최적의 전술적 위치로 유도하는 '요격 통제' 임무를 수행한다. 이들은 타국 항공기의 고도·속도·방위, 아군기의 위치와 기상 등을 종합적으로 판단해 아군 항공기를 통제해 '하늘을 지키는 잠들지 않는 눈'으로 불린다. 좋아요 0 Copyright ⓒ 조선일보 & Chosun.com 제휴안내구독신청
