
# ロヒンギャ帰還問題、ミャンマーに対応求める…国連事務総長

Published at: **2019-11-03T11:14:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191103-OYT1T50154/)

　【バンコク＝杉目真吾、大重真弓】国連と東南アジア諸国連合（ＡＳＥＡＮ）は３日、バンコクで首脳会議を開催した。国連のグテレス事務総長は、バングラデシュに逃れて難民となったミャンマーのイスラム系住民ロヒンギャの問題に懸念を表明し、早期帰還のための対応をミャンマー政府に求めた。
　グテレス氏は会談で、ロヒンギャ問題について「多くの難民が依然として困難な状況に置かれていることを深く懸念する」と述べた。その上で、ミャンマー政府に対し、「国際的な規範と基準に従い、安全で自発的なロヒンギャ難民の帰還を促す環境を確保する責任がある」と指摘した。ミャンマーがロヒンギャに国籍を付与せず、移動の自由や教育・医療など社会サービスを受ける権利も制限していることを批判したものだ。
　グテレス氏はまた、帰還実現に向けてＡＳＥＡＮがミャンマー支援を打ち出していることについて、「関与を歓迎する」と語り、国連とＡＳＥＡＮの連携を強化する考えを示した。
　ミャンマーでは２０１６年以降、国軍が武装勢力掃討の名目でロヒンギャへの攻撃を強め、７０万人以上のロヒンギャがバングラに流入し、難民キャンプで暮らしている。
