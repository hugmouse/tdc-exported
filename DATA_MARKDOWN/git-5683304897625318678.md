
# മുഖ്യമന്ത്രിയുടേത് ഭീഷണിയുടെ സ്വരം; അവഗണനയോടെ തള്ളുന്നു: എന്‍എസ്എസ്

Published at: **2019-11-02T11:42:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/02/nss-against-cm-pinarayi-vijayan.html?utm_source=dlvr.it&utm_medium=twitter)

കോട്ടയം ∙ മുഖ്യമന്ത്രി പിണറായി വിജയനെതിരെ കടുത്ത വിമര്‍ശനവുമായി എന്‍എസ്എസ്. കാലത്തിനനുസരിച്ച് സ്വയം നവീകരിക്കാത്തവര്‍ അസാധുവാകുമെന്നുള്ള മുഖ്യമന്ത്രിയുടെ പ്രസ്താവന ദുരുദ്ദേശപരമാണെന്ന് എൻഎസ്എസ് ജനറൽ സെക്രട്ടറി ജി. സുകുമാരൻ നായർ പറഞ്ഞു. എന്‍എസ്എസിനെ ഉദ്ദേശിച്ചാണ് മുഖ്യമന്ത്രി പറഞ്ഞതെങ്കില്‍ തികഞ്ഞ അവഗണനയോടുകൂടി തള്ളിക്കളയുന്നു. 
‘ഞങ്ങള്‍ മാത്രമാണ് നവോത്ഥാനത്തിന്‍റെ പ്രവാചകര്‍, ഞങ്ങള്‍ പറയുന്ന വഴിയേ വന്നോണം, അല്ലാത്തവരെയൊക്കെ അപ്രസക്തമാക്കും’ എന്ന ഭീഷണിയുടെ സ്വരം അതിന്‍റെ പിന്നില്‍ ഇല്ലേ എന്ന് സംശയിക്കുന്നു. ശബരിമല വിഷയത്തോടനുബന്ധിച്ചാണ് സര്‍ക്കാര്‍ നവോത്ഥാനം ഉയര്‍ത്തിപ്പിടിക്കാന്‍ തുടങ്ങിയത്. അതിനെ തുടര്‍ന്നാണ് ഇതുവരെയില്ലാത്ത തരത്തില്‍ സവര്‍ണ്ണ-അവര്‍ണ്ണചേരിതിരിവും മുന്നാക്ക-പിന്നാക്കവിഭാഗീയതയും ജാതിതിരിവും ഉണ്ടായതുമെല്ലാം.
ഇതെല്ലാം രാഷ്ട്രീയനേട്ടങ്ങള്‍ക്കു വേണ്ടിയാണെന്നുള്ളത് ആര്‍ക്കാണ് അറിഞ്ഞുകൂടാത്തത്. നവോത്ഥാനമൂല്യങ്ങളില്‍ അടിയുറച്ചു നിന്നുകൊണ്ട് സമുദായപുരോഗതിയിലൂടെ സമൂഹത്തിന്‍റെയും നാടിന്‍റെയും നന്മയ്ക്കുവേണ്ടി സത്യസന്ധമായി പ്രവര്‍ത്തിച്ചു വരുന്ന സംഘടനകള്‍ക്കെതിരെ ജനങ്ങളുടെ വോട്ടുവാങ്ങി അധികാരത്തിലേറിയ മുഖ്യമന്ത്രി ഉപദേശ രൂപേണ ഇത്ര വില കുറഞ്ഞ രീതിയില്‍ പ്രതികരിച്ചത് അവിവേകമാണെന്നും എൻഎസ്എസ് പത്രക്കുറിപ്പിൽ അറിയിച്ചു.
