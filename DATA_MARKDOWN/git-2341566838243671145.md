
# 臉書公司換新商標 字體色彩象徵家族成員

Published at: **2019-11-05T12:39:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/ait/201911050101.aspx)

（中央社舊金山4日綜合外電報導）臉書公司（Facebook Inc.）今天公布新的公司商標，套用象徵「家族成員」的彩色字體，代表這間設於矽谷的公司，以與旗下核心社群網站有所區別。
臉書經營一系列應用程式（app）和服務，其中包括訊息傳遞、照片分享、虛擬實境，同時也在開發數位貨幣使用的電子錢包，推出新商標是為了凸顯臉書「品牌」。
根據臉書行銷長魯西歐（Antonio Lucio）所言，臉書公司新的商標，基本上是以正楷書寫公司的名字，將印在旗下「家族」服務上。
美國有線電視新聞網（CNN）的報導中提到，臉書公司新商標的顏色象徵其家族成員，包括代表臉書的藍色、象徵WhatsApp的綠色，還有Instagram的代表色紫色、紅色與橘色。
魯西歐說：「今天我們更新了我們公司的商標，好讓人更清楚知道來自臉書的產品。」
「我們向大家介紹新的公司商標，並更近一步將臉書公司與臉書app區別開來，而臉書社群網站將保留原本的名稱。」
魯西歐補充道，新的公司商標「是一個更好的溝通方式，以向使用臉書服務連結社群的個人或企業，更清楚展現我們的所有權結構」。
在此同時，臉書公司維護平台內容的方式，受到全球監管機關的嚴格審查，一些政治人物與運動人士企圖分拆這個矽谷巨擘。
臉書是在15年前以社群網路應用程式起家，之後收購了虛擬實境穿戴裝置公司Oculus，和以分享照片為主的社群網路Instagram，臉書還推出智慧螢幕Portal，和為工作場所量身定做的社群網路Workplace。
臉書的傳訊應用程式WhatsApp和Messenger，各自擁有超過10億用戶。
臉書建立了電子錢包Calibra，以搭配擬議發行的加密貨幣Libra使用，但Libra計畫自宣布以來，不斷遭受多國監管機管和立法者批評。（譯者：李晉緯/核稿：張曉雯）1081105
