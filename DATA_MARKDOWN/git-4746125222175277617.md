
# AKB48・太田奈緒、グループ卒業を発表 今後は「小さい頃からの夢である女優に」

Published at: **2019-11-06T22:57:00+00:00**

Author: ****

Original: [ORICON NEWS](https://www.oricon.co.jp/news/2148195/full/)

　アイドルグループ・AKB48チーム8の太田奈緒（24）が6日、東京・秋葉原のAKB48劇場で行われた公演で卒業を発表した。23日が最後の握手会となり、12月20日に卒業公演を行う。 　公演後に太田は自身のツイッターで卒業を報告し、「今の私はアイドルとしてやり切ったなと心の底から思います」と胸中を告白。「卒業後は、小さい頃からの夢である女優に、少しでも近づけるような活動ができればと思っています」と今後の展望をつづった。 　卒業まで残り1ヶ月となり、「メンバーやファンの方々とたくさん思い出を作って最後まで頑張りたいと思います。よろしくお願いします」と呼びかけている。 　太田は2014年、各都道府県の代表メンバーで結成されたチーム8の京都府代表として加入し活動を開始。選抜総選挙では第9回65位、第10回62位にランクイン。今年は女優として『山犬』『エラリー・クイーン ミステリーオムニバス〜観客への挑戦〜』『フラガール -dance for smile-』などの舞台に出演している。
