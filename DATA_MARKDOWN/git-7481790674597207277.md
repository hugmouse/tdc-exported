
# 「スマホで錬金術」と虚偽広告、消費者庁が企業名公表

Published at: **2019-11-07T09:41:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191107-OYT1T50216/)

　「スマホで錬金術」「年収１０００万円超えをクリアして頂きます」などの虚偽広告で消費者に金を支払わせたとして、消費者庁は７日、消費者安全法に基づいて「ＷＡＶＥ」（東京都新宿区）の企業名を公表し、注意を呼びかけた。
　同庁によると、同社は２０１８年１２月以降、自社ウェブサイトに虚偽広告を掲載し、消費者を勧誘。実際には購入していない商品に利益を上乗せして米国の大手通販サイトに出品し、買い手がついた後で商品を購入して販売する、などとするマニュアルを１万円程度で提供。さらに、販売をサポートするとして８万～１５０万円の有料プランに加入させた。多くの消費者には、全く売り上げがなかった。
　同社は同庁に対し、「８月までに約８２００人から計約７億７０００万円を売り上げた」と説明している。
