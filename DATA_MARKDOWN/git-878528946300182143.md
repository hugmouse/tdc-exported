
# 田代容疑者起用のＮＨＫ大阪「とても残念」　局には厳しい声…「バリバラ」で講師

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/07/0012857406.shtml)

　ＮＨＫ大阪放送局の有吉伸人局長（５６）が７日、大阪市の同局で定例会見を行った。大阪放送局が制作する「バリバラ」（Ｅテレ、木曜、後８・００）の７月４、１１日放送回で、「講師」として起用した元タレントの田代まさし容疑者（６３）が、６日に覚せい剤取締法違反（所持）の疑いで逮捕されたことに対し、制作部部長が「とても残念」とコメント。局には厳しい批判の声が届いていることも明かされた。
　同番組のテーマは「薬物依存症」だった。有吉局長は「出演者は企画に合わせて、その都度、いろんな難しい状況があるが、そのこともしっかり考えながら人選することに尽きる」と話すにとどめたが、番組担当の福島広明制作部部長（５３）は、田代容疑者の周囲などを取材した上で最終的に出演判断した結果の事件について「とても残念」と肩を落とした。
　収録は大阪放送局で６月８日に行われ、田代容疑者の不審な様子は「収録当日もなかったし、ほかの出演者からの指摘もなかった」と振り返った。
　６日時点でホームページなどから当該番組の動画などが削除されたことについて「逮捕を受けて、総合的に判断して取りやめた。そのまま掲載を続けることは適切ではないと判断した」と説明。「（事実関係が）はっきりしたら別の判断もあるかと思う。逮捕は重いこと、一旦、そういう（削除の）判断をしたということ」と話した。
　田代容疑者の逮捕を受けて、同局には「本人の起用についての厳しいご意見が届いている」（広報担当者）という。
　７月の同局定例会見の際、福島部長は田代容疑者の起用について「依存症の当事者で、家族から孤立して再犯を繰り返す。『バリバラ』らしいスタイルで、依存症の人がどう回復しているか、どう社会と向き合うかいい材料になる」と説明していた。
