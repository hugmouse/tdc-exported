
# เลขาฯยูเอ็นขอไทยดันแก้ปัญหาสิ่งแวดล้อม-ลดขยะทะเล 

Published at: **2019-11-02T10:12:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/politics/739678)


        นายกฯยันไทยและอาเซียนร่วมมือยูเอ็นเดินหน้าการพัฒนาที่ยั่งยืน เลขาฯยูเอ็นขอไทยช่วยผลักดันการแก้ปัญหาสิ่งแวดล้อม-ลดขยะในทะเล 
      
เมื่อวันที่ 2 พ.ย. ที่อาคารอิมแพค ฟอรั่ม  พล.อ.อก ประยุทธ์ จันทร์โอชา นายกรัฐมนตรีและรมว.กลาโหม ได้พบหารือทวิภาคีกับนายอันโตนิ​อู กุแตเรช เลขาธิการสหประชาชาติ ในระหว่างการประชุมสุดยอดอาเซี​ยน ครั้งที่ 35 และการประชุมสุดยอดที่เกี่ยวข้​องทั้งนี้  นางนฤมล ภิญโญสินวัฒน์ โฆษกประจำสำนักนายกรัฐมนตรี กล่าวหลังการหารือ ว่า  นายกรัฐมนตรียืนยันว่​าประเทศไทยให้ความสำคัญกั​บการสร้างหุ้นส่วนการพัฒนาที่ยั่​งยืนระหว่างประเทศสมาชิกอาเซี​ยนกับภูมิภาคต่างๆ ขณะที่อาเซียนพร้อมร่วมมือกั​บสหประชาชาติ (ยูเอ็น) เพื่อส่งเสริมการแลกเปลี่​ยนประสบการณ์กับกลุ่มต่างๆ เกี่ยวกับแนวทาง องค์ความรู้ และประสบการณ์ เพื่อที่ประเทศต่างๆจะร่วมกั​นบรรลุเป้าหมายการพัฒนาอย่างยั่​งยืน  นอกจากนี้ไทยยินดีเพิ่มบทบาท และมีส่วนร่วมในกรอบยูเอ็นมากขึ้​นสำหรับการจัดกิจกรรมต่างๆ ในภูมิภาคเอเชียแปซิฟิก  
ด้านเลขาธิการยูเอ็นกล่าวชื่​นชมไทยและนายกรัฐมนตรีที่มี​บทบาทในเวทีระหว่างประเทศอย่​างกระตือรือร้น รวมถึงการแสดงวิสัยทัศน์​ในประเด็นท้าทายต่างๆ อีกทั้งชื่นชมบทบาทของไทยที่​โดดเด่นในการดำเนินการเพื่​อบรรลุการพัฒนาอย่างยั่งยืน และยืนยันว่ายูเอ็นให้ความสำคั​ญกับประเด็นด้านสิ่งแวดล้อม โดยขอให้ไทยช่วยเป็นส่วนสำคั​ญในการผลักดันการแก้ปัญหาเรื่​องสิ่งแวดล้อม การลดการปล่อยก๊าซ เรือนกระจก การใช้พลังงานหมุนเวียน อีกทั้งชื่นชมความริเริ่​มของไทยในการประชุมเกี่ยวกั​บการลดขยะในทะเล                 โฆษกประจำสำนักนายกรัฐมนตรี กล่าวอีกว่า นอกจากนี้ นายกรัฐมนตรีย้ำนโยบายของไทยที่​สอดคล้องกับแนวทางความร่วมมือกั​บยูเอ็น อาทิ การแก้ไขปัญหาการทำประมงที่ผิ​ดกฎหมายและไร้ระเบียบ การต่อต้านการค้ามนุษย์ การป้องกันการแสวงหาประโยชน์​ทางเพศ และสิทธิมนุษยชน  ขณะที่เลขาธิการยูเอ็นอวยพรให้​นายกรัฐมนตรี​และประเทศไทยประสบความสำเร็​จในการจัดการประชุมสุดยอดอาเซี​ยนในครั้งนี้.
