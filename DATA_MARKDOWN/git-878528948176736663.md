
# 桂華紋がＮＨＫ新人落語大賞「上方落語ブームが来ているので流れに乗る」

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012848134.shtml)

　ＮＨＫ新人落語大賞の本選がこのほど、大阪市のＮＨＫ大阪ホールで行われ、桂華紋（かもん、３２）が大賞を受賞した。若手落語家の登竜門を制した華紋は「上方落語ブームが来ているので、この流れに乗って、戦力のひとつとして活躍できるように頑張らないといけないなという気持ちです」と殊勝に語った。
　予選会には東西の落語家１２１人が参加。勝ち抜いた６人が本選に出場し、１１分の演目で競い合った。華紋は２０１０年４月、五代目桂文枝門下の桂文華に入門。本選では「ふぐ鍋」を披露し、会場を笑いの渦に巻き込んだ。
　落語家生活１０年目で、歴史ある賞をつかんだ華紋は「（演目順が）１番だったので、変な緊張感もなく臨めたのでよかったです。いつもどおりに奇をてらわず、１番手なので他の方を意識する必要がなかったので逆に良かったと思いました」と、平常心で臨んだ高座が奏功した。
