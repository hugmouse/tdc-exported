
# 【11月2日のスカーレット】第30話　草間の妻を訪ねるふたり　常治からは緊急連絡が…

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/01/0012840860.shtml)

　ＮＨＫ連続テレビ小説の１０１作目となる「スカーレット」（月曜～土曜８時）。１１月２日は第３０話が放送される。喜美子の後押しで、草間と妻の復縁はかなう？
　再会した草間（佐藤隆太）から、生き別れた妻が別の男と店をやっていると聞いた喜美子（戸田恵梨香）。そのまま草間を連れて店を訪ねる。遠慮する草間の背中を押して入店し、草間はついに妻と再会。しかし草間は何事もなかったかのように普通の客として振る舞う。喜美子も草間に合わせて、声をかけられない。そして草間は厨房にいる妻の新たな男を見つけて近づく。一方、信楽の父、常治（北村一輝）から緊急の連絡が…。
　「スカーレット」は究極の働き女子の物語。焼き物の里として知られる信楽の女性陶芸家が、創造する情熱と喜びを糧に、失敗や挫折にめげず生きていく姿を描く。がむしゃらな強さと明るさを持つヒロインを神戸出身の人気女優、戸田恵梨香が演じる。ドラマのオープニングを飾る主題歌「フレア」は、Ｓｕｐｅｒｆｌｙがこの作品のために書き下ろした入魂の一曲。
