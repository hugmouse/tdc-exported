
# いだてん、４３分の放送時間が４２分に　チュート徳井余波で…「できる限りの配慮」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012842572.shtml)

　ＮＨＫ大河ドラマ「いだてん」制作統括の訓覇圭氏は２日、３日にオンエアされる第４１回の放送時間が、通常の４３分から４２分に短縮されることを明らかにした。女子バレーボール日本代表の大松博文監督を演じるチュートリアル・徳井義実（４４）が、東京国税局から申告漏れを指摘された問題を受けて再編集を施したため。
　訓覇氏は「大河ドラマ『いだてん』第４１回の放送内容・時間について」と題した文章をこの日、報道各社に送付し、番組の公式サイトでも発表。
　それによると、徳井が演じる大松監督について「１９６４年の東京オリンピックで、東洋の魔女と呼ばれた日本女子バレーボールチームを率い、金メダルを獲得しました。その活躍ぶりは『いだてん』で欠かすことのできない物語のひとつでした」と説明。
　「いだてん」は「１０月１日に全編の収録が終了し、撮り直しができない状況にあるため、徳井さんのシーンを中心に、再編集して放送」することが既に発表されており、「再編集にあたっては、視聴者のみなさまに満足していただけるよう、ドラマ全体の流れを損なわないことにもできる限りの配慮をいたしました。その結果、１１月３日（日）放送予定の第４１回は、通常４３分の放送時間が４２分となります」と、短縮せざるを得なかった事情を訴えている。
