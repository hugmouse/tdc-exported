
# 侍・山口　準備ＯＫ　５日ベネズエラ戦先発濃厚

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/04/0012846916.shtml)

　国際大会「プレミア１２」に臨む野球日本代表「侍ジャパン」が３日、台湾・桃園市の桃園国際野球場で３時間の調整を行った。
　５日・ベネズエラ戦に先発濃厚な山口が、ブルペンで最終調整を行った。打席に立った丸にボールの軌道を確認し「いい感覚でした」と納得の表情。前回登板した１０月３１日・カナダ戦は２回６失点の大乱調だったが「ボール（の違い）は関係ない。自分がアジャストするだけなので」と不安はない。
