
# Ｃｈａｒ、石田長生さんの思い出を語る（２）名コンビの理由は「違うから」

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/04/0012847710.shtml)

　関西を代表するギタリストで、熱烈な阪神ファンとしても知られた故石田長生さんが２０１５年７月に６２歳の若さで死去してから、４年が過ぎた。７月２４日に２枚組トリビュートアルバム「ＳＯＮＧＳ　ＯＦ　Ｉｓｈｉｙａｎ」がリリースされ、１１月２０日には東のＣｈａｒ、西の石田と並び称されたギタリスト、Ｃｈａｒ（６４）との名コンビ「ＢＡＨＯ（バホ）」の５枚組ＣＤ－ＢＯＸ「大馬呆展（おおばほてん）」が発売される。石田さんの足跡に改めて光が当てられている今、ギターと阪神で結ばれた盟友Ｃｈａｒが、石田さんの思い出を語った。第２回はＢＡＨＯの内幕を振り返る。
　　◇　　◇
　ＢＡＨＯは９１年、唯一のスタジオアルバム「ＴＲＥＭＥＮＤＯＵＳ」発表を挟み、９０～９３年に精力的なライブ活動を行った。「タモリ倶楽部」や「笑っていいとも！」などのテレビ番組にも出演。抱腹絶倒の掛け合いと変幻自在なプレイでリスナーを魅了し、日本におけるアンプラグドライブの嚆矢（こうし）となった。２１世紀に入っても断続的に活動を続けていたが、石田さんの死をもって終止符が打たれた。
　なぜ育ってきた環境が違う２人が名コンビになったのか。Ｃｈａｒは「違うからじゃないですか」と明快だ。
　「バンドだとかユニット作るっつうのは、趣味の合った者同士が同調して何かを目指すじゃないですか。俺も石やんも既に色んな経験もしてきた上で、求めるもの…影響受けるものはどんどん少なくなっていくわけですよ。そん中で、同じ日本人でこんだけ違うものっていうのは、逆に言うと日本人同士だから面白い、通じるものはあって。ルーツも違う、言葉も違う、お互いにとって、関西人が東京人と、東京人が関西人と長い時間を過ごすってことが、そんなになかったんじゃないかと思うんですよ」
　「だからぶつかれないっていうか、こっちがこういうセンスでギターを弾くと、俺じゃ絶対弾かないフレーズを石やんが持ってきてくれる。その逆、石やんが書いた曲に俺がつける、『ええな、そうやってくれ』。２人ともシンガーソングライター、最小限の歌２本ってところで、可能性がすごく広いっていうか。同じような趣味同士だったら、やっぱりどっかでぶつかる。それはそこだったら俺がそこソロ取るとか、俺がやった後、お前が弾けばいいじゃんとか、猫の喧嘩になるんですよ。それが楽曲として認めていくセンス、知識とか、そういうものを持ってたので、ウマが合ったというか」
　Ｃｈａｒにはない石田さんの持つ経験も、新鮮な驚きだった。
　「石やんはＢＡＨＯやる前に一人旅みたいなのをアコギ一本でやってて、ホントに小さいジャズ喫茶みたいなところで。こういうところで石やんやってるのか。イス、みかん箱やんか。そういうところでいい影響を与えてくれて。２人だけでツアーやろうやということが始まって、ボーヤと３人でハイエースで北海道回ってみたり、九州回ってみたり、最初の１～２年で寝食を共にするわけで」
　密度の濃いツアーで、お互いをよく知ることができた。
　「石やんの方が俺より２つ３つ上で大学も行ってたから、非常にアカデミックなバックボーンも持ってて、かつ俺がけっこう生意気なことを言ってもそれを返す関西人の－俺の生意気さ、それはそれで江戸っ子的なてやんでえかもしれないですけど－そこのキャッチボールがうまくいってたんじゃないか。完璧な江戸っ子と浪速っ子みたいなところで。
　石やんが俺のことをどう思ったか分かんないですけど。ナンボのもんやねんと思ったと思うんですよ。それは新鮮だったし、たぶんこっちの出す音楽的な要素みたいなものも、石やんも非常に影響を受けただろうし、俺もやっぱり石やんの出す音楽的なセンスみたいものを、はあ～、そういくか。『エブリバディ調子はどうだい？』って東京人は言いませんからね。コール＆レスポンス的な、いわゆるエンターテインメントでももっとベタな」
　ＢＡＨＯ以前、Ｃｈａｒのパブリックイメージは「気絶するほど悩ましい」のロックアイドルであり、ジョニー、ルイス＆チャー（後のピンク・クラウド）のギターヒーローだったが、ＢＡＨＯでは石田さんとの抱腹絶倒の掛け合いや、硬軟織り交ぜた演奏で、新たな顔を見せた。
　「もともと持ってるふざけた部分っいうのは、石やんに引き出された部分は多分にあります。石やんも新喜劇とか往年のお笑いで育ってるので、こう言ったらこうやという、いわつるツッコミボケの体質つうのは、教育つうのは完璧になされてて、その辺は音楽とは別のところで、石やんがこう言ったらこう言ってくれとか、分かった分かったと。そこはやっぱりツッコミとボケと。早くからツッコミツッコミとかボケボケも俺は面白いんじゃないか、それはそうやな、こんなことまで研究してて。企画を先に立てて。そこから音楽を考えていく。面白かったですね。どんだけあの人のアイデアを却下したか」（続く）
