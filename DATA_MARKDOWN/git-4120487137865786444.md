
# 意甲-皮亚特克造乌龙 AC米兰遭绝杀1-2不敌蓝鹰

Published at: **2019-11-05T03:43:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/04/sports-1572882086)

【新浪体育4日讯】（意大利当地时间3日20时45分），意甲第11轮开始1场较量，AC米兰主场1比2负于拉齐奥，排名跌回下半区。因莫比莱中楣，随后打进效力蓝鹰正式比赛第100球，皮亚特克制造巴斯托斯乌龙。终场前，科雷亚上演绝杀。AC米兰新添两名伤员，穆萨基奥上轮伤退，替补出场打进制胜任意球的苏索髋部受伤。卡拉布里亚回到首发，杜阿尔特改打中路。克鲁尼奇取代凯西。里卡多-罗德里格斯与卡尔达拉伤愈担任替补。拉齐奥周中4比0大胜都灵，上赛季意大利半决赛曾经客场击败AC米兰，但上次意甲客胜要追溯到1989年。AC米兰（4-3-3）：99-唐纳鲁马/2-卡拉布里亚，43-杜阿尔特，13-罗马尼奥利，19-特奥-埃尔南德斯/33-克鲁尼奇(85’博纳文图拉)，4-本纳赛尔，39-帕克塔(53’莱昂)/7-卡斯蒂列霍(36’雷比奇)，9-皮亚特克，10-恰尔汉奥卢。拉齐奥（3-5-1-1）：1-斯特拉科沙/15-巴斯托斯，33-阿切尔比，26-拉杜/39-拉扎里，21-米林科维奇-萨维奇(60’帕罗洛)，6-卢卡斯，10-阿尔韦托，19-卢利奇/11-科雷亚/17-因莫比莱(60’凯塞多；82’卡塔尔迪)。
