
# អង្គការពាណិជ្ជកម្មពិភពលោក សម្រេចអនុញ្ញាតឱ្យចិនដាក់ទណ្ឌកម្មចំពោះទំនិញអាមេរិក

          អង្គការពាណិជ្ជកម្មពិភពលោក (
          OMC) 
          បាន​ចេញសេចក្តីសម្រេចអនុញ្ញាតឱ្យចិនអាច​ដាក់ទណ្ឌកម្មចំពោះពន្ធនាំចូលទំនិញរបស់អាមេរិក។ អង្គការពាណិជ្ជកម្មពិភពលោក អនុញ្ញាត​ឱ្យ​ចិន​ដាក់​ពិន័យអាមេរិក​ ក្នុងរង្វង់​ទឹកប្រាក់​ជាង​៣,​៥ពាន់​លានដុល្លារ ចំពោះ​ការ​នាំចូល​ទំនិញ​អាមេរិក ក្នុង​មួយឆ្នាំៗ។​ គឺ​ជាសេចក្តីសម្រេចលើកទី១ក្នុងប្រវត្តិសាស្ត្រនៃ​ជម្លោះ​ពាណិជ្ជកម្ម​រវាង​ចិន​និងអាមេរិក​។
        
      
​សំណុំ​រឿង​នេះ គឺ​ចិនបានប្តឹងអាមេរិក តាំងពីឆ្នាំ ២០១៣មកម្ល៉េះ តែអង្គការពាណិជ្ជកម្មពិភពលោក ទើបតែ​សម្រេច នៅថ្ងៃ​សុក្រទី១វិច្ឆិកា​​​។ ចិន​ប្តឹង​អាមេរិក​ពីកាល​នោះ ព្រោះតែ​អាមេរិក​បន្ថោក​តម្លៃ និង​ទម្លាក់​តម្លៃ​ទំនិញ​របស់​ចិន។​ ការ​ប្តឹង​ផ្តល់គ្នា​ស៊ី​ពេល​យូរ ដោយ​​មាន​ការ​តទល់​មុខគ្នា​ច្រើន​ដងផង​ ​ភាគី​តំណាង​របស់​អាមេរិក ក៏​ធ្លាប់​បាន​សុំ​ពន្យារ​ពេលអស់​​ច្រើនលើក​ដែរ​។
នៅទីបំផុត អង្គការពាណិជ្ជកម្មពិភពលោក ចេញ​សេ​ចក្តី​សម្រេចជាចុងក្រោយ ហើយ​​ការ​សម្រេចនេះ គឺ​បិទផ្លូវ​តវ៉ា​ ដែល​ភាគី​អាមេរិកមិន​អាច​ប្តឹង​តវ៉ា​ទៅមុខ​ទៀត​ឡើយ ក៏ប៉ុន្តែ ចំពោះ​​ភាគ​ចិនវិញ ចិន​អាច​មាន​ជម្រើស​ច្រើនតាមឆន្ទៈ​របស់ខ្លួន​។ ពោលគឺ​ ចិន​អាច​នឹង​សិទ្ធិ​សម្រេច​ដាក់ទណ្ឌកម្ម​ ឬមិន​ដាក់​​ទណ្ឌកម្ម​ចំពោះ​ទំនិញ​អាមេរិក​ក៏​បាន​ ក៏ប៉ុន្តែ ប្រសិន​បើ​ដាក់​ពិន័យនោះ ​កម្រិត​ទណ្ឌកម្ម​នេះ គឺ​ក្នុង​រង្វង់​ទឹកប្រាក់ តែ​ជាង​៣,៥ពាន់​លាន​ដុល្លារ តែប៉ុណ្ណោះ ក្នុង​មួយ​ឆ្នាំៗ​។ កន្លងមក​ ​ចិនធ្លាប់​​ចង់​ទាមទារលទ្ធភាព​ដាក់​ពិន័យ​អាមេរិក ​ដល់​ទៅ ៧ពាន់​ដុល្លារ​ឯណោះ ក្នុង​មួយ​ឆ្នាំៗ​។
ក្រោយ​ពេល​ទទួល​បាន​សេចក្តីសម្រេច, ភាគី​តំណាង​អាមេរិក​នៅ​អង្គការពាណិជ្ជកម្មពិភពលោក បាន​សម្តែង​ក្តី​សោកស្តាយ​ជាខ្លាំង ក៏ប៉ុន្តែ ថា​នឹង​យក​រឿង​នេះ ទៅជជែក​ជាមួយ​ថ្នាក់​ដឹកនាំ​ក្រុង​វ៉ាស៊ីនតោន ព្រមទាំង​ភាគី​ចិន​ផងដែរ ដើម្បី​រកច្រក​​ផ្លូវ​ធ្វើ​កិច្ច​បន្ត​។ ទោះ​យ៉ាងណាក្តី ​ភាគី​អ្នកតំណាង​អាមេរិក លើកឡើងថា ការ​សម្រេចខាង​​លើនេះ មិន​ប៉ះពាល់​ដល់​ដំណើរ​ការ​ចរចាក្នុង​ជម្លោះ​ពាណិជ្ជកម្ម​​រវាងក្រុង​វ៉ាស៊ីនតោន​និង​ក្រុង​ប៉េកាំង​ឡើយ។
​ដំណើការ​ចរចាក្នុង​ជម្លោះ​ពាណិជ្ជកម្មរវាង​មហាយក្សទាំងពីរនេះកំពុង​តែ​បន្ត​!​ អាមេរិក​បាន​លើកឡើង​ពី​ការ​ចរចា​គ្នា​ក្នុង​រយៈ​កាល​បីដំណាក់កាល ដើម្បី​ចុះ​កិច្ចព្រមព្រៀង​ថ្មី​។ «ដំណាក់​កាលទី១ កំពុងតែ​ជិត​បាន​សម្រេច​ហើយ និង​ឈរ​លើ​ផ្លូវ​ល្អ​ស្រុះ​ស្រួល​គ្នា ហើយ​និង​មាន​ការ​ជឿនលឿន​ខ្លាំង»។ នេះបើ​តាម​តាម​ការ​បញ្ជាក់​របស់​ទីប្រឹក្សា​សេដ្ឋកិច្ច​របស់​សេតវិមាន​អាមេរិក លោក Larry Kudlow ដែល​ប្រាប់​អ្នក​កាសែត​ ពី​ថ្ងៃ​សុក្រ ទី១វិច្ឆិកា។ ទីប្រឹក្សា​សេដ្ឋកិច្ចអាមេរិក​បញ្ជាក់ទៀត​ថា «ការ​ចរចាលើ​​វិស័យ​កសិកម្ម, សេវាកម្ម​ហិរញ្ញវត្ថុ ជា​ដើម​បាន​ចប់រួចរាល់​​ហើយ។ ចំណែក ការ​ចរចាទៅលើ​សំណុំរឿង​លួច​កម្មសិទ្ធិបញ្ញា គឺកំពុងតែ​បន្ត និង​មាន​ការ​វិវឌ្ឍ​ល្អ»។ ពីភាគ​ចិន​វិញ កាសែត​ផ្លូវការ​ចិន Chine Nouvelle បាន​ទទួល​ស្គាល់​ថា ការ​ចរចា​ចុះកិច្ចព្រមព្រៀង​អាមេរិក​ចិន គឺ​បាន​យល់ព្រម​គ្នាហើយ ទៅលើផ្នែក​គោលការណ៍រួម៕
