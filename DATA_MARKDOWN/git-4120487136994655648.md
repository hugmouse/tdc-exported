
# 女学生网络卖淫案 大部分嫖客都是当地官员和政客

Published at: **2019-11-02T02:18:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/01/society-1572617494)

【打横Tribun新闻网10月31日讯】西爪省打横市女学生网络卖淫案，大部分嫖客据称都是当地官员和政客。警方是从其中一名被拘捕的卖淫女薇（22岁）口中得知此事的。薇是年龄最大的卖淫女，她和另4名女学生艾（17岁）、菲（18岁）、芬（16岁）和莉（17岁）一起从事卖淫工作。周四（10月31日），薇对警察说：“周末最多接客两次，平日只接客一次。客人是打横市官员、政客和商人。”她坦承虽然刚从事卖淫工作两个月，但已接客很多次，几乎每天都接客。3名男子给这些女孩在社交媒体上介绍嫖客。他们都已被警方拘捕。援引《罗盘》新闻网报道，她说：“一次收费50到70万盾，包一天270万盾，这已经包含了酒店房间。”周三（10月30日），打横市警方破获一起少女网络卖淫案。3名老鸨和5名卖淫女在某家酒店房间接客时被警察抓捕。警方接到某家酒店报警，说有几个举止可疑的女子，有几个男子轮流进入她们的房间。
