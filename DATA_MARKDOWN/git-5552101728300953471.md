
# 坐副駕完全不睡　江宏傑開車福原愛忙餵食

Published at: **2019-11-07T10:20:00+00:00**

Author: **陳于嬙**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent026/)

江宏傑、福原愛夫妻檔攜手代言TOYOTA SIENTA新車，兩人一起拍廣告，靈感來自福原愛小時候參加桌球比賽的場景，劇情是小小愛因發球不順而落淚，奔向媽媽尋求安慰，福原愛看到場設，驚呼：「場景、演員跟氛圍都跟當年去比賽時一模一樣，好像穿越到小時候。」
廣告中一幕夫妻倆送女兒上幼稚園的第一天，回想到之後就要面對，江宏傑與福原愛都相當激動，福原愛更說：「我覺得我一定會馬上哭。」雖然代言車子，福原愛還沒考上駕照，她說：「一方面是當媽媽了，也覺得自己開車很酷，但我個性不敢轉彎，怕旁邊有摩托車。」
回想起兩人世界，福原愛透露江宏傑經常載她去台南、墾丁約會，她在副駕駛座忙著遞食物、礦泉水給江宏傑吃，完全不會睡覺，福原愛說：「可能以前我哥開車，會一直叫我拿東西、咖啡養成習慣。坐小傑的車，我就變成DJ，一直選歌播放。」
