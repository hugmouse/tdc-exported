
# 剛力彩芽の『ラヴ・レターズ』に感激　吉田麻梨紗が朗読劇に

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012843144.shtml)

　モデルの吉田麻梨紗が１日に更新したブログで、同じ事務所の剛力彩芽の朗読劇「ラヴ・レターズ」を見たと報告した。恋愛において、文字を書いて伝える手紙こそが大きな役割を果たすとの思いを強くしたとしている。
　吉田は「昨日、彩芽の舞台観劇に行って来たよ」と投稿。「舞台『ラヴ・レターズ』笑いあり、涙ありの２時間弱の朗読劇だったー」と充実した様子で記した。
　「字を書いて想いを伝える手紙は本当に素敵なものですよね♪　恋愛においては、もしかしたら一番感情を素直に表現出来る方法なのかもしれない。そう感じさせられた朗読劇でした」と、つづった。「舞台観劇をすると・・・映像のドラマや映画はもちろんですが、舞台にも挑戦してみたいなと、意欲が湧いてくるぐらい感情を刺激されるんよなぁ」と剛力の舞台を見て大きな刺激を受けた様子。
　「舞台だと自分の演技を直接みせることが出来る、皆さんの反応を直接肌で感じられる、それが素晴らしい魅力的な部分ですよね」と記した。
