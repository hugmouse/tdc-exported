
# 日向坂がドローンレース決勝に登場　潮「妖精みたいに見えます」と大興奮

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012843619.shtml)

　アイドルグループ・日向坂４６が２日、都内で開催された、公式アンバサダーを務める「ＦＡＩドローン東京２０１９レーシング」の決勝大会に登場した。
　４０人が参加した予選を勝ち抜いたファイナリスト１６人による空中スピードバトルを見届けて、潮紗理菜（２０）は「（ドローンが）妖精みたいに見えます。こんなにキレイに光ってるの、現実世界で見たことなかった」と大興奮。新曲「こんなに好きになっちゃっていいの？」などのライブパフォーマンスも繰り出して、ドローン頂上決戦に花を添えた。
