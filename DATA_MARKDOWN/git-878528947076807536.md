
# 新田真剣佑　三浦春馬のエールに感激「あとでこの動画ください！」

Published at: **2019-11-06T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/06/0012853368.shtml)

　俳優・新田真剣佑（２２）が６日、俳優の岸谷五朗（５５）と寺脇康文（５７）が主宰する演劇ユニット・地球ゴージャスの２５周年祝祭公演「星の大地に降る涙」（２０２０年３月１０日～４月１３日、東京・舞浜アンフィシアター）の製作発表に出席した。
　２００９年、舞台初出演だった三浦春馬（２９）主演で行った作品を約１０年ぶりに再演。岸谷と寺脇以外のキャストを一新し、パワーアップした舞台を作り上げる。
　三浦が演じた主人公・シャチ役を受け継ぐ新田。同ユニットの舞台に参加するのは１８年の「ＺＥＲＯＴＯＰＩＡ」に続く２度目で、今回が舞台初主演となる。米国で生まれ育った新田は、「まだお仕事を始めていない頃の自分が見た作品に出られることをうれしく思う」と憧れの作品だったことを明かした。
　新田の背中を押すべく、三浦からビデオメッセージが届いた。三浦は「（自分と）比べることはなくて、今のまっけん（真剣佑）は、本当に優に超えてくる期待感が大きい。楽しみにしている。まっけんの実力で引っ張って行ってほしい」とエール。それを聞いた新田は、「あとでこの動画ください！ファンです！本当にうれしいです。前作を超えるような作品にしなきゃと思いました」と感無量だった。
　会見の最後には１０日後（今月１６日）に２３歳の誕生日を迎える新田にバースデーケーキのサプライズ。初演から１０年という今作にちなんで、「役者として生きていれば良い。説得力のある大人な役者になりたい」と、１０年後の自分へ誓っていた。
