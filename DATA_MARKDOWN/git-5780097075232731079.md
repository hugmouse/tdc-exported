
# ថៃ ផ្ទុះ​ការវាយ​ប្រហារ​ប្រដាប់​អាវុធ​នៅ​ភាគ​ខាងត្បូង​ស្លាប់​មនុស្ស​ជាច្រើន​នាក់

        មាន​ស្លាប់​យ៉ាង​ហោច​ណាស់ ១៥​នាក់ នៅ​ក្នុងការ​វាយប្រហារមួយ នៅភាគ​ខាង​ត្បូង​នៃ​ប្រទេស​ថៃ ពី​សំណាក់​ក្រុម​ប្រដាប់អាវុធ។ នេះ​បើ​តាម​យោធា​ថៃ ដែល​ចោទ​​ភ្លាម​ថា​ជា​ស្នាដៃ​របស់​ពួក​ឧទ្ទាម​ចង់​ផ្តាច់​ទឹកដី​អ៊ីស្លាម។ តំបន់ចុង​ខាង​ត្បូងបំផុត​នៃ​​ប្រទេសថៃ ដែល​មាន​ប្រជាជនភាគ​ច្រើន​ជា​អ្នកកាន់​សាសនា​អ៊ីស្លាម តែង​រងភេរវកម្ម និងការ​​វាយប្រហារ​​ជា​ញឹកញយ ប៉ុន្តែ​ការ​វាយ​ប្រហារ​នៅ ល្ងាច​ម្សិលមិញ មាន​លក្ខណៈ​​ធ្ងន់ធ្ងរ​ខ្លាំង​បំផុត​មួយ។
      
ជនប្រដាប់​អាវុធ ដែល​សំងំលាក់​ខ្លួន​ក្នុង​ព្រៃ​ចំការ​កៅស៊ូ ដ៏​ធំធេង​របស់​ខេត្ត​យ៉ាឡា បានចេញ​មក​សង្គប់​​​ ​លើ​ប៉ុស្តិ៍​ល្បាត​ គ្រប់គ្រង​ដោយ​ជនស៊ីវិល​ និង​បើក​ការ​​វាយប្រហារដោយ​អាវុធ​ និង​បំផ្ទុះ​គ្រាប់​ដែក​គោល លាន់​ទ្រហឹង កាលពី​ព្រលប់​ថ្ងៃ​អង្គារ​ម្សិលមិញ។ មនុស្ស​១២​នាក់ បាន​ស្លាប់​ភ្លាម​នៅ​នឹង​កន្លែង​កើត​ហេតុ បី​នាក់​ទៀតបាន​ស្លាប់​នៅ​ពេល​បញ្ជូន​ទៅ​ដល់​មន្ទីរ​ពេទ្យ ដែល​កំពុង​ទទួល​ព្យាបាល​​អ្នករង​របួស​ជា​ច្រើន​​ផ្សេង​ទៀត។
នេះ​មិនមែន​ជា​លើក​ទី​មួយ​ទេ ដែល​តំបន់​ភាគ​ខាង​ត្បូង​រង​ការ​ប្រហារ​ពី​សំណាក់​ក្រុមឧទ្ទាម​ប្រដាប់​អាវុធ​អ៊ីស្លាមនោះ​។ តែ​ការវាយប្រហារ​នៅ​ល្ងាចមិញ ដែលបាន​​បង្ក​ឱ្យ​មាន​ជន​រងគ្រោះ​​​មរណៈ​ច្រើន​ជាង​បំផុត​មួយ គិត​ចាប់​តាំង​វិបត្តិ​បាន​ផ្ទុះ​ឡើង បាន​​ឆ្លុះ​បញ្ចាំង​ឱ្យ​ឃើញ​កាន់​តែ​ច្បាស់​អំពី​ភាព​ប្រេះ​ឆា​កាន់​តែ​ខ្លាំង​រវាង​ប្រជាពលរដ្ឋ​​ថៃ​​ភាគ​ច្រើន​កាន់​ព្រះពុទ្ធ​សាសនា​​ និង​ប្រជាពលរដ្ឋ​កាន់​សាសនា​អ៊ីស្លាម​នៅភាគខាង​ត្បូង​។ តាម​ធម្មតា ការវាយប្រហារ​តែង​ធ្វើ​ឡើង ដោយ​ដៅ​ទិស​​ប្រឆាំង​នឹង​លើ​យោធា ​និង​ទីតាំង​របស់​ជា​តំណាង​ស្ថាប័ន​រដ្ឋ​។
សកម្មភាព​ឧទ្ទាម​ប្រដាប់​អាវុធ ចង់​ផ្តាច់​ទឹក​ដី​ របស់​ក្រុម​អ្នក​កាន់​សាសនាអ៊ីស្លាម​​ភាគ​តិច​ក្នុង​ប្រទេស បាន​ចាប់​កកើត​ឡើង​តាំង​ពី​ឆ្នាំ​២០០៤​មក នៅ​តំបន់​ភាគ​ខាង​ត្បូង ក្នុង​​ខេត្ត​បី ជា​សំខាន់​ គឺ​យ៉ាឡា ណារ៉ាធីវ៉ាត់ និង​ប៉ាតានី។ បើ​តាម​ក្រុម​សង្កេតការណ៍​ឯករាជ្យ​ Deep South Watch គិត​ចាប់​ពី​ពេល​នោះ​មក​ទល់​បច្ចុប្បន្ន​ វិបត្តិ​នៅភាគ​ខាង​ត្បូង​ប្រទេស​ថៃ បាន​ធ្វើ​ឱ្យ​​មនុស្ស​ជាង​៧ពាន់​បាន​បាត់ប​ង់​ជីវិត។ ប៉ុន្តែ​កងកម្លាំង​ប៉ូលិស​សង្កេត​ឃើញ​ថា​ ក្រៅអំពី​ជម្លោះ​ប្រដាប់​អាវុធ ដើម្បី​​បុព្វហេតុ​នយោបាយ ការ​វាយប្រហារ-សម្លាប់​ដោយ​អាវុធ ច្រើន​ផ្ទុះ​ឡើង​ដោយ​មូលហេតុ​ដ៏​សំខាន់​ខ្លាំង​មួយ​ទៀត គឺ​ការ​ជួញដូរ​គ្រឿង​ញៀន និងជួញដូរ​មនុស្ស នៅ​ក្នុង​តំបន់​ជាប់​ព្រំដែន​ជា​ម៉ាឡេស៊ី៕
