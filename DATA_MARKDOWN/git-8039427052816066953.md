
# నవంబర్‌ 5 అర్ధరాత్రిలోగా కార్మికులు చేరకపోతే...

హైదరాబాద్: నవంబర్‌ 5వ తేదీ అర్ధరాత్రి లోగా ఆర్టీసీ కార్మికులు విధుల్లో చేరకపోతే.. మిగతా 5 వేల రూట్లను కూడా ప్రైవేట్‌కే ఇస్తామని తెలంగాణ ముఖ్యమంత్రి కేసీఆర్ స్పష్టం చేశారు. అందుకు ప్రైవేట్ ఆపరేటర్లు కూడా సిద్ధంగానే ఉన్నారని చెప్పారు. అయితే ఆ ప్రైవేట్ ఆపరేటర్లు కూడా ప్రభుత్వ నియంత్రణలో ఉంటారన్నారు. పర్మిట్లు ఇచ్చే విధానంలో కూడా మార్పుండదని తేల్చిచెప్పారు. ప్రభుత్వం మంచి అవకాశం ఇచ్చిందని.. దీన్ని చేజార్చుకోవద్దని ఆర్టీసీ కార్మికులకు ఈ సందర్భంగా కేసీఆర్ విజ్ఞప్తి చేశారు.
 
రాష్ట్రంలో 90కి పైగా కార్పొరేషన్లు ఉన్నాయని, ప్రభుత్వంలో ఆర్టీసీ విలీనం డిమాండ్ నేపథ్యంలో ఆ కార్పొరేషన్ల ఉద్యోగులు కూడా విలీనం చేయమంటారని కేసీఆర్‌ పేర్కొన్నారు. పాలసీల విషయంలో ప్రభుత్వాన్ని ఎవరూ శాసించలేరని, ఆర్టీసీలో పోటీతత్వం ఉండాలనే ప్రైవేట్ వాళ్లకు ఇస్తున్నామని చెప్పారు. తాను చెబుతున్న విషయాలు మోటార్‌ వెహికల్‌ చట్టంలోనే చెప్పారంటూ ఇది తాము కొత్తగా చేస్తున్నదేమీ కాదన్నారు.
