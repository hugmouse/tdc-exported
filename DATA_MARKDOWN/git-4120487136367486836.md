
# 佐科威总统出席曼谷第35届东盟峰会全体会议 东盟-中国建立战略伙伴关系是区域和平稳定的支柱

Published at: **2019-11-04T03:51:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/03/politics-1572796221)

【印尼CNBC新闻网11月3日讯】东盟峰会及东亚合作领导人系列会议11月2日在泰国曼谷Impact Exhibition and Convention Center国际会议中心举行。我国总统佐科威及东盟国家元首均出席第35届东盟峰会全体会议。陪同佐科威总统出席第35届东盟峰会的有政法安统筹部长马福特(Mahfud MD)、 经济统筹部长艾尔朗卡．哈尔丹托(Airlangga Hartarto)、外交部长蕾特诺马尔苏迪（Retno L P Marsudi）、贸易部长阿古斯.苏巴尔曼多(Agus Suparmanto)和内阁秘书帕拉摩诺阿侬（Pramono Anung）。佐科威总统感谢和赞赏泰国政府2019年担任东盟轮值主席期间对东盟的贡献。总统也感谢东盟国家元首和代表于10月20日出席在雅京国会大厦举行佐科威-马鲁夫阿敏正副总统2019-2024届宣誓就职典礼。雷诺外长周日（11月3日）在峰会会议后向传媒发表讲话说，东盟国家领导人峰会的讨论议题其中是关于区域全面经济伙伴关系协议（RCEP）的谈判进程。东盟国家领导人一致同意并希望加快与RCEP有关的谈判进展能尽快完成，亦推动实现地区共同繁荣和发展。东盟10国与中国、日本、韩国、澳大利亚、新西兰、印度6个对话伙伴国领导人还将举行区域《全面经济伙伴关系协定》领导人会议，力争推动这一多边贸易协议达成。RCEP由东盟10国于2012年发起，邀请中国、日本、韩国、澳洲、新西兰、印度共同参加，旨在通过削减关税及非关税壁垒，建立区域统一市场。雷诺外长接着说，佐科威总统还传达了许多其他问题，例如缅甸若开邦（Rakhine State）的人道危机问题并呼吁东盟国家领导人携手协助解决相关问题。雷诺外长如是阐述。此外，佐科威总统冀望东盟国家加强合作共同应对《东盟关于印度太平洋展望》的议题。出席第35届东盟峰会全体会议国家和政府领导人有泰国总理巴育、新加坡总理李显龙、菲律宾总统杜特尔特、老挝总理通伦、缅甸国务资政昂山素季、柬埔寨总理洪森、马来西亚总理马哈蒂尔、越南政府总理阮春和汶莱苏丹哈吉哈桑纳尔。（v）
