
# പ്രതികൾക്കെതിരെ പ്രഥമദൃഷ്ട്യാ  യു.എ.പി.എ നിലനില്‍ക്കുന്നു; കസ്റ്റഡിയില്‍ ചോദ്യം ചെയ്യണമെന്ന പൊലീസിന്റെ ആവശ്യം അംഗീകരിച്ച് കോടതി

Published at: **2019-11-06T15:20:00+00:00**

Author: **Ruhasina J R**

Original: [Anweshanam | Part of A.N.N News Private Limited](http://anweshanam.com/index.php/kerala/news/court-on-uapa-arrest)

കോഴിക്കോട്: അലന്‍ ശുഹൈബിനും താഹ ഫസലിനുമെതിരെ പ്രഥമദൃഷ്ട്യാ  യു.എ.പി.എ നിലനില്‍ക്കുമെന്ന് കോടതി. ജാമ്യാപേക്ഷ തളളിക്കൊണ്ടുളള ഉത്തരവിലാണ് കോഴിക്കോട് സെഷന്‍സ് കോടതി പരാമര്‍ശം.  കസ്റ്റഡിയില്‍ ചോദ്യം ചെയ്യണമെന്ന പൊലീസിന്റെ ആവശ്യം അംഗീകരിക്കുന്നതായും കോടതി ഉത്തരവില്‍ വ്യക്തമാക്കുന്നു.  അതേസമയം വിദ്യാര്‍ഥികളുടെ മാവോയിസ്റ്റ് ബന്ധത്തിന് കൂടുതല്‍ തെളിവ്. അട്ടപ്പാടിയില്‍ നിന്ന് ലഭിച്ച ലഘുലേഖയും വിദ്യാര്‍ഥികളില്‍ നിന്ന് കണ്ടെടുത്തും ഒന്നുതന്നെയെന്ന് പൊലീസ് പറയുന്നു. അട്ടപ്പടിയില്‍ നിന്ന് കണ്ടെടുത്ത പെന്‍ഡ്രൈവിലാണ് ലഘുലേഖയുടെ പകര്‍പ്പ് ഉണ്ടായിരുന്നത്.
യു.എ.പി.എ ചുമത്തപ്പെട്ട രണ്ട്  സി.പി.എം പ്രവര്‍ത്തകരുടെ  ജാമ്യാപേക്ഷ കോടതി തള്ളിയിരുന്നു. യു.എ.പിഎ നിലനിൽക്കുമെന്ന പ്രോസിക്യൂഷൻ നിലപാട് പരിഗണിച്ചാണ് വിധി. കേസിൽ എഫ്.ഐ.ആർ റദ്ദാക്കണമെന്നാവശ്യപെട്ട് ഹൈക്കോടതിയെ സമീപിക്കുമെന്നു പ്രതിഭാഗം അഭിഭാഷകൻ മനോരമ ന്യൂസിനോട് പറഞ്ഞു. യു.എ.പി.എ നിലനിൽക്കുമെന്ന  വാദവും  പ്രോസിക്യൂഷൻ ഹാജരാക്കിയ തെളിവുകളും പ്രതിഭാഗത്തിന് വിനയായി. നിരോധിത പ്രസ്ഥാനങ്ങളിൽ അംഗമാവുക അവരുടെ ആശയങ്ങൾ പ്രചരിപ്പിക്കുക തുടങ്ങി കുറ്റങ്ങളാണ് ഇവർക്കെതിരെ  ചുമത്തിയിരിക്കുന്നത്. പന്തീരാങ്കാവിൽ ഇവർക്കൊപ്പം നിന്ന മൂന്നാമന് വേണ്ടിയുള്ള തിരച്ചിൽ തുടരുകയാണ്.
