
# 従業員と「合意に基づく関係」…解任の米マクドナルドＣＥＯ、過ち認める

Published at: **2019-11-04T01:16:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191104-OYT1T50085/)

　【ニューヨーク＝小林泰明】世界最大手のハンバーガーチェーン、米マクドナルドは３日、スティーブ・イースターブルック最高経営責任者（ＣＥＯ）を解任したと発表した。同社は声明で、イースターブルック氏が「会社の規定に違反し、従業員と関係をもつという誤った判断をした」と述べた。
　同社は詳細を明らかにしていないが、従業員との「合意に基づく関係」があったとし、恋愛や性的関係にあったことをにおわせた。後任にはマクドナルドＵＳＡ社長のクリス・ケンプチンスキー氏が就任した。
　マクドナルドによると、イースターブルック氏は従業員向けのメールで「過ちだった」と認め、解任を受け入れる姿勢を示した。
　米国では、個人的な人間関係が業務に影響を及ぼす懸念があるとして、上司が部下をデートに誘うことを禁止している会社が珍しくないとされる。昨年６月には、米半導体大手インテルの当時のＣＥＯが社内規定に反する社員との関係があったとして辞任している。
