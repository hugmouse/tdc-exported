
# 高橋克典　早くもラグビーＷ杯ロス「あぁ、明日からはどうすればいいんだ」

Published at: **2019-11-02T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/02/0012844327.shtml)

　俳優の高橋克典（５４）が２日、公式ブログを更新し、この日、開催されたラグビーＷ杯日本大会の決勝戦、南アフリカ－イングランドを「最後まで観られた！！」と喜びつつ、早くもＷ杯ロスに陥っている。
　高橋は中学から高校１年までラグビー部に所属し、２００３年にはテレビ東京でＷ杯のＭＣを担当。昨年はＮＨＫ総合でラグビーがテーマのドラマ「不惑のスクラム」に主演したラガーマンだ。
　この日、４本目の投稿で「今夜は決勝戦　南アフリカｖｓイングランド　楽しみすぎる　けど仕事　いや、ありがたいですよ。け、けど、けど！！よりにもよって…だばだばだば～録画見ます」と、仕事で決勝が見られないことを嘆いていたが、次の投稿では「今夜は『カクホの女』ロケ。でもね、なんとか移動と支度の合間を縫って最後まで観戦できました」と、リアルタイムで見られたことを報告。「先ほど今帰宅。これから録画を観るのです　素晴らしい試合でしたね」と、もう一度見るという。
　高橋は今回のＷ杯について「本当に素晴らしい体験、そして素晴らしい感動と記憶の数々をいただきました　素晴らしい時間だったなぁ　私的にも、本当に辛い時の心を支えられました」と感謝し、「あぁ、明日からはどうすればいいんだ」と、早くもロスになっている様子。
　「しばらく録画を繰り返し観たり、今年は絶対トップリーグや、大学、高校ラグビーもか観ようーーー！！と心に決めたのであります」と、ますますラグビー愛が高まったようで、「関係者の皆様、本当に素晴らしい時間をありがとうございました」と再び感謝し、「願わくば、今こそ去年ＮＨＫでやらせていただいた『不惑のスクラム』も観ていただきたいなぁと思うのであります」と訴えていた。
