
# 宇野８位、ジャンプミス響く…シニアＧＰで初めて表彰台逃す

Published at: **2019-11-02T15:18:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/winter/20191103-OYT1T50000/)

　フィギュアスケートのグランプリ（ＧＰ）シリーズ第３戦、フランス杯は２日、グルノーブルで男子フリーが行われ、ショートプログラム（ＳＰ）４位の宇野昌磨（トヨタ自動車）は、ジャンプのミスが響いて１３６・７９点の９位にとどまり、合計２１５・８４点で８位だった。ファイナルを含め、宇野がシニアのＧＰで表彰台を逃すのは１３戦目で初めて。
　ＳＰ首位で世界選手権２連覇中のネーサン・チェン（米）が１９４・６８点を挙げてフリーも１位となり、合計２９７・１６点で優勝した。第１戦のスケートアメリカに続きＧＰ２連勝で、ファイナル進出を決めた。（時事）
