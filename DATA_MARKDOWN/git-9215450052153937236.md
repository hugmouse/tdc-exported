
# 澳門旅客人數增速下降 料博彩業收入錄負增長｜國泰君安

Published at: **2019-11-02T09:00:30+00:00**

Author: **投資**

Original: [經濟一週](https://www.edigest.hk/article/122101/%e6%8a%95%e8%b3%87/%e6%be%b3%e9%96%80-%e6%97%85%e5%ae%a2%e4%ba%ba%e6%95%b8-%e5%a2%9e%e9%80%9f%e4%b8%8b%e9%99%8d-%e5%8d%9a%e5%bd%a9%e6%a5%ad-%e6%94%b6%e5%85%a5%e9%8c%84-%e8%b2%a0%e5%a2%9e%e9%95%b7-%e5%9c%8b%e6%b3%b0/)

由於香港的社會事件對赴港旅遊產生負面影響，同時赴澳門旅遊人數的增速也在下降，因此將2019年澳門博彩毛收入的增長預測將從3.8%下調至-2.6%，行業評級下調至「中性」。
撰文：國泰君安 ｜ 圖片：iStock
 
今年截至6月份，香港累計接待遊客3,490萬人次，同比增長13.9%，其中2,760萬人次來自內地，同比增長16.4%。在同一時間段內，前往澳門的遊客人數同比增長20.6%至2,020萬人次。
在7月初，隨著反修例運動的升級，7月份赴港旅遊人數同比下降4.8%，8月份赴港旅遊人數更同比大幅下降39.1%，為2003年「沙士」爆發以來最大的單月跌幅。隨著負面影響蔓延到澳門，7月、8月以及9月赴澳門旅遊人數同比增速分別降至16.3%、6.5%和8%。
今年截至8月份，前往澳門的遊客數量同比增長18%至2,740萬人次；相對2018年創新高的3,580萬遊客人次，澳門原有望在2019年輕鬆刷新紀錄。事實上，自2018年10月24日港珠澳大橋開通以來，赴澳門遊客數量快速增長；然而，遊客數量的增長預期將在今年餘下時間顯著放緩。在地緣政治和宏觀經濟前景惡化的情況下，澳門博彩的毛收入（GGR）增長亦不那麼樂觀。過去幾個月以來，中美貿易戰不斷升級，中美貿易關係亦持續惡化，美國甚至將中國列為匯率操縱國。
在2019年前八個月間，持續進行的貿易戰導致美國從中國的進口同比下降12.5%，這拖累了澳門博彩業的收入增長以及估值。全球股市也正在變得越來越不穩定，將帶來負面的財富效應，降低人們對澳門博彩業的興趣，特別是對於貴賓分部而言。2019年前九個月，澳門的博彩毛收入同比下降1.7%，增長率在第一季度和第二季度分別為-0.5%和-0.5%，在第三季度則擴大至-3.5%。在最近的7月、8月及9月，博彩毛收入同比增長率分別為-3.5%、-8.6%和0.6%。儘管9月份錄得了略為正面的增速，但這亦不是一個積極的訊號，主要是由於2018年9月份的比較基數很低。
當日澳門的賭場由於颱風山竹吹襲而關閉超過24小時，導致去年9月赴澳門旅遊人數同比增速僅為3%，遠低於2018年全年的9.8%增速，對去年9月的博彩毛收入產生了負面影響。
2019年第一季度、第二季度和第三季度，貴賓分部業務博彩毛收入分別同比下降13.4%、15.6%和22.5%，分別佔博彩毛收入的48.9%、47.2%和43.9%。與之相反，中場分部業務的博彩毛收入在2019年第一季度、第二季度和第三季度分別增長16.1%、18.6%和17.7%。貴賓分部業務的表現下滑，主要是由於貴賓賭客避險的觀望態度，而中場分部業務的持續強勁增長則是由於去年10月開通的港珠澳大橋帶來的強勁客流增長。但現在大橋的開通運營已滿一年，預計客流的增長將放緩。
