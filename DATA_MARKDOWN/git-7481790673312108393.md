
# 節電要請、今冬は行わず…供給力確保と判断

Published at: **2019-11-06T12:22:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/economy/20191106-OYT1T50304/)

　経済産業省は６日、電気利用者への今冬（１２月～来年３月）の節電要請は行わないことを決めた。昨冬は北海道地震の影響で安定供給が不安視された北海道に節電要請を行ったが、この冬は２年ぶりに全国すべての地域で十分な供給力を確保できると判断した。
　大手１０電力会社の供給力にどれだけ余裕があるかを示す「予備率」が、安定供給に必要とされる３％を上回るメドが立った。暖房需要の高まりで需給が最も逼迫（ひっぱく）する２月でも、全国で平均６・６％（沖縄を除く）の予備率を確保できるとしている。
