
# മൊബൈലില്‍ വിളിക്കരുത്; സിപിഎമ്മുകാരെ നിരീക്ഷിക്കുക: മാവോയിസ്റ്റ് രഹസ്യരേഖ

Published at: **2019-11-05T09:22:00+00:00**

Author: **മനോരമ ലേഖകൻ**

Original: [ManoramaOnline](https://www.manoramaonline.com/news/latest-news/2019/11/05/dont-use-mobile-phones-maoist-secret-documents-out.html?utm_source=dlvr.it&utm_medium=twitter)

കോഴിക്കോട്∙ സംഘാംഗങ്ങളുടെ മൊബൈല്‍ അടക്കം ഫോണ്‍ ഉപയോഗത്തിനു നിര്‍ദേശങ്ങളുള്ള മാവോയിസ്റ്റ് രഹസ്യരേഖ പുറത്ത്. സഖാക്കളുടെ പേരോ ‘ടെക്’ പേരോ ഫോണില്‍ പറയരുത്. സഖാക്കള്‍ പരസ്പരം മൊബൈലില്‍ വിളിക്കരുതെന്നും ടെലിഫോണ്‍ ബൂത്തുകള്‍ ഒഴിവാക്കണമെന്നും രഹസ്യരേഖ നിര്‍ദേശിക്കുന്നു.
യോഗങ്ങള്‍ ചേരുന്നതിനും രഹസ്യരേഖയില്‍ പ്രത്യേക നിര്‍ദേശങ്ങളുണ്ട്.  കോഴിക്കോട്ട് പിടിയിലായ രണ്ട് വിദ്യാര്‍ഥികളുടെ പക്കല്‍ നിന്ന് കണ്ടെടുത്ത രഹസ്യരേഖയിലെ വിവരങ്ങളാണു പുറത്തുവന്നത്. രഹസ്യരേഖയുടെ പകര്‍പ്പ് മനോരമ ന്യൂസിന് ലഭിച്ചു. അതേസമയം അറസ്റ്റിലായ അലന്‍ ഷുഹൈബിന്റെയും താഹ ഫസലിന്റേയും ജാമ്യാപേക്ഷയില്‍ കോഴിക്കോട് സെഷന്‍സ് കോടതി ഇന്നു വിധിപറയും.
യോഗം ചേരുന്നതിനുള്ള മാര്‍ഗരേഖ
∙ പ്രാദേശിക സിപിഎം പ്രവര്‍ത്തകരെ നിരീക്ഷിക്കുക
∙ യോഗത്തിന് വരുമ്പോള്‍ മൊബൈല്‍ പാടില്ല
∙ രണ്ടിലേറെ കൂടിക്കാഴ്ചകള്‍ നിശ്ചയിക്കുക, ആദ്യത്തേതിന് വരിക
∙ അജന്‍ഡയോ തീരുമാനങ്ങളോ എഴുതിവയ്ക്കരുത്
∙ ആധുനിക ഇലക്ട്രോണിക് ഉപകരണങ്ങള്‍ ഉപയോഗിക്കുന്നവര്‍ ജാഗ്രത പാലിക്കുക
ഇതിനിടെ മാവോയിസ്റ്റുകള്‍ക്കെതിരായ പൊലീസ് നടപടിയെ ന്യായീകരിച്ച് ചീഫ് സെക്രട്ടറി ടോം ജോസും രംഗത്തെത്തി. സാധാരണക്കാര്‍ക്കുള്ള മനുഷ്യാവകാശം സായുധ മാവോസിയ്റ്റുകള്‍ക്ക് അവകാശപ്പെടാനാകില്ല. മാവോയിസ്റ്റുകളുടെ മനുഷ്യാവകാശത്തിനു വേണ്ടിയുളള വാദങ്ങള്‍ നിയമം അനുസരിക്കുന്നവരെ അവഹേളിക്കലാണെന്നും ചീഫ് സെക്രട്ടറി ലേഖനത്തില്‍ പറയുന്നു.
സിപിഐ (മാവോയിസ്റ്റ്) അംഗങ്ങൾക്കു കൈമാറാനായി തയാറാക്കിയ ‘ ശത്രുവിന്റെ അടയാളങ്ങളും നമ്മുടെ പ്രത്യാക്രമണ മാർഗങ്ങളും’ എന്ന 7 പേജുകളുള്ള രേഖ വിദ്യാർഥികളില്‍നിന്നു കണ്ടെത്തിയതായി പൊലീസ് അറിയിച്ചിരുന്നു. അല‍ൻ ഷുഹൈബും താഹ ഫസലും മാവോയിസ്റ്റുകളുടെ കോഡ് ഭാഷ ഉപയോഗിച്ചിരുന്നതായും പൊലീസ് വ്യക്തമാക്കി. താഹയുടെ വീട്ടിൽനിന്നു കണ്ടെത്തിയ ബുക്കിലാണ് കോഡ് ഭാഷയുള്ളത്. ഇതു വായിച്ചെടുക്കാൻ വിദഗ്ധരുടെ സഹായം തേടിയിട്ടുണ്ട്.
