
# 堀江貴文氏　面識ない人と写真ＮＧの理由「情報商材屋」に“被害”も

Published at: **2019-11-05T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/05/0012850471.shtml)

　「ホリエモン」こと実業家の堀江貴文氏が５日、ツイッターを更新。新幹線車内で「大阪のおばちゃん」に盗撮されて注意したところ、同行男性からもキレられ「モラルないんかなこの人ら」と激怒した前日のツイートが反響を呼ぶ中、改めて、面識のない人との写真ＮＧの理由を吐露した。
　フォロワーから「堀江さんですよね？よろしかったら一緒にお写真撮らせて戴いても宜しいですか？と聞かれたら、いいですよ。と言って戴けますよね」と質問されたところ、堀江氏は「言わないよ。基本的に知らん人と写真は撮影しない」と自身のスタンスを明白にした。
　堀江氏は「なぜなら、その人の素性が分からないと反社会的勢力かもしれず週刊誌に『黒い交際』とか書かれる事もあるだろうし、情報商材屋で『ホリエモンおすすめ』とか書かれた事もある」とツイート。実際に“被害”をこうむったという「情報商材屋」について、他のフォロワーから「かわいい女の子なら撮らせる？」と突っ込まれると、堀江氏は「随分前に撮影してあげたら情報商材屋の手先の女だった。悪用された」と明かした。
　「だから普通の芸能人よりリスク高いのね」とつづった堀江氏。別のフォロワーから「有名人ならではのことで、ある程度は仕方ないのでは？」と指摘されると、「は？嫌なものは嫌だから注意すればいいだけのこと。有名人が我慢してると図に乗ってくるぞ、この手の輩は。仕方ないとか勝手に決めつけんなボケが」と反論した。
