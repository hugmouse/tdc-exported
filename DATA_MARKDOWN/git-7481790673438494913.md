
# アパート２階から煙、掛け布団１枚焼き女性死亡

Published at: **2019-11-01T00:09:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/national/20191101-OYT1T50130/)

　１日午前０時４５分頃、名古屋市中村区長筬（ながおさ）町の鉄筋２階建てアパート２階、小浜里加さん（５１）さん方で煙が出ていると１１０番があり、敷いてあった掛け布団１枚を焼いて約２０分後に消えた。女性が掛け布団の上にうつぶせで倒れており、病院に搬送されたが、約４５分後に死亡した。愛知県警中村署は女性が小浜さんとみて身元確認を急いでいる。
