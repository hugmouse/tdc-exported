
# 侍・稲葉監督　打線は心配？７四球も６安打で３得点「とにかく１点１点」

Published at: **2019-11-01T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/baseball/2019/11/01/0012840999.shtml)

　「侍ジャパン強化試合、日本３－０カナダ」（１日、沖縄セルラースタジアム那覇）
　侍ジャパンが被安打４の完封リレーでカナダに勝利。投手力の高さを見せつけたが、打線は７四球を選びながら６安打で３得点に抑えられた。
　四回に２四球を絡め、浅村（楽天）の二ゴロの間にノーヒットで先制。五回は山田（ヤクルト）の適時打、八回は近藤（日本ハム）の適時打で小刻みに加点したが、試合全体では外国人投手のパワーに押され、迫力を欠いた。
　稲葉監督は「選手にはとにかく１点１点とっていこう、と。国際大会は簡単に点を取れないのでコツコツいこう、と」と、苦戦も覚悟している様子。５日にベネズエラとの初戦を迎える「プレミア１２」に向けて、「１点を取る野球ができた。また次につながると思います」と前を向いた。
