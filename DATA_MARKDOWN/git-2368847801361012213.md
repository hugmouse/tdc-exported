
# 【精彩推薦】四中全會出人命 /盤點中南海九大淫棍

Published at: **2019-11-05T09:28:38+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/05/a102700432.html)

【新唐人北京時間2019年11月05日訊】新唐人為讀者推薦每日精彩文章和視頻節目：
四中全會鬧出人命 缺席五人中兩人離奇死亡剛結束不久的中共十九屆四中全會公報中，沒有顯示人事異動。但媒體發現，全會期間鬧出了人命，在缺席四中全會的5人中，有兩人離奇死亡，其中包括中共重慶市委副書記任學鋒。
巴格達迪姐姐被活捉 軍方獲ISIS機密恐怖組織伊斯蘭國（ISIS）原頭目巴格達迪在美軍突襲中自殺後，他的姐姐拉斯米亞．阿瓦德日前被土耳其政府活捉。一名土耳其高級官員稱，阿瓦德就像一個「情報寳藏」，將從她身上挖出更多ISIS內部機密情報。
前中共公安局副局長家產上億元 涉迫害人權罪中共河北省張家口公安局前副局長王守成，日前，被以涉嫌巨額財產來源不明等四宗罪受審，案件未當庭宣判。其家產超過1億元人民幣，其中5千多萬來源不明。王守成還涉迫害人權罪。
猝死重慶副書記傳與陳敏爾不睦 學者:疑反習勢力被整肅重慶市委副書記任學鋒在四中全會閉幕日突然死亡。中共官方在其已去世3天後，才在深夜對外發佈消息，稱其近日因病醫治無效死亡，引發外界質疑。有消息指任某系跳樓身亡，而多個信息源指其或因捲入體制內的路線鬥爭而被清除。
胡耀邦去世前 女兒感知到神秘第六感心理學家認為，所謂「第六感」是人體一種瞬間而直接的預知。據統計，在第六感方面，女性的正確率普遍要大於男性。胡耀邦的女兒在回憶錄中提到父親去世前，她感知到神秘的第六感，「它實在太神秘，神秘得連它的存在都變得可疑。但我確確實實地感知到了。」
江澤民令計劃共享情婦？盤點中南海九大淫棍自江澤民腐敗治國開始，中國社會道德急劇下滑，官場色情泛濫。日前，媒體翻出中南海九大淫棍的醜聞，這些大淫棍從副國級到正國級，大多是江澤民直接培植的鐵桿親信，更離譜的是江竟然還與淫棍之一的令計劃共享情婦。
習面授林鄭「最重要任務」 公安部長隨行引猜測中共四中全會後，香港事態並未緩和，相反更加嚴峻，港警創下日捕逾200人新記錄。特首林鄭月娥11月4日赴上海向習近平匯報工作，習對林鄭面授當前「最重要的任務」，二人會見時中共公安部部長趙克志也陪同在一旁，引發外界猜測。
傳任學峰被約談事涉P2P 從京西賓館墜樓亡訃告流出年僅54歲的中共重慶市委副書記任學峰突然離世，自殺傳聞滿天飛。網傳訃告顯示，任學峰是10月31日死於北京。還有消息稱，任學峰進京參加中共四中全會即被約談，四中閉幕當天在京西賓館跳樓。
大陸民眾「戴銬跳舞」避審查 陳秋實：要砸碎鐐銬中共當局政治言論審查的力度日趨嚴重，大陸民間出現了一種「戴着鐐銬跳舞」的應對方式。曾直播香港「反送中」運動的北京律師陳秋實遭到中共官方全網封殺後，日前有粉絲在視頻中，幫陳秋實在臉部貼上表情包，再進行傳播。不過，陳秋實表示能夠理解，但應該考慮怎麼把鐐銬砸碎。
鄭中原：習近平急赴上海 變相宣示接班人 要動江澤民侄？中共十九屆四中全會剛落幕，習近平隔天就跑到上海，還到了楊浦區。除了準備出席下星期在當地舉辦的第二屆國際進口博覽會的開幕儀式，還事涉接班人動向，以及四中全會前一樁未及處理的官場大醜聞，或因觸動層級太高，習近平要親自出馬。
全民監控再升級 中共在新疆測試「情緒識別」技術近日，中共以預防犯罪為由，在新疆的機場等公眾場合利用新研發的情緒辨識技術來分析行人的情緒及心理狀態的做法，在海外引起輿論非議。專家學者指出，情緒識別並非新技術，但中國是第一個公然在社會上廣泛使用這種技術的國家。政府利用電子高科技對民眾實行全面監控，等於給全中國人都戴上電子監控腳環，本質上是把所有人都視為罪犯或嫌疑人。
中共首次宣布「公安姓黨」 「黨衛隊」呼之欲出中共19屆四中全會再次強化「黨管一切」，以維護中共岌岌可危的統治。隨後，公安部部長趙克志即宣布「公安姓黨」，要保衞所謂「社會主義制度安全」。評論人士指，這等於直接宣誓公安是「黨衛隊」。
中美東盟峰會角力 美批北京把海變湖李克強溫和回應本屆東盟峰會上，白宮安全顧問奧布萊恩批評中共封鎖南海，威嚇他國，把國際海域變成內陸湖。中共外交部高官和美方針鋒相對，互相指責。而中共總理李克強會見奧布萊恩時，則避開分歧，強調「互利共贏」的美中關係。
中共恐嚇東盟挨批 華為欲挾東南亞抗美圍堵第35屆東盟峰會及東亞峰會週一在泰國曼谷閉幕。白宮國家安全顧問在會上批評，北京威嚇阻止東盟國家開發南海資源。華為公司則在峰會上稱為在東南亞鋪設5G做好準備，被指意圖對抗美國圍堵。
彭博：習近平對赴美簽協議持開放態度智利宣布取消主辦APEC峰會後，美中兩國首腦將在何處簽署首階段貿易協議頓成懸疑。彭博社日前引述知情人士稱，中國國家主席習近平對赴美持開放態度。週一，中共外交部發言人在記者會上被問及「川習會」可能的地點時卻未給予正面回答，僅稱兩位領導人一直保持聯繫。
「東北3兄弟」襲擊港人被圍毆 微博熱捧成「英雄」中共當局不僅對香港反送中運動進行抹黑式宣傳，更僱傭、煽動陸港黑幫分子襲擊恐嚇香港市民，製造混亂。日前，自稱「東北3兄弟」的3名大陸男子赴港襲擊市民遭到圍毆，3人返回大陸並在微博上傳挺港警視頻，被不明真相的大陸網民捧為「英雄」。
香港5示威者被當庭釋放 百餘港警堵庭外再抓香港反送中運動持續多月以來，數千名示威者被抓捕，其中一些被捕者被撤銷控罪，但一出法院又被警方抓捕。據港媒報導，4日晚，有百餘名防暴警在東區法院外，把已經獲撤銷控罪的示威者帶上警車，再次抓捕。
太古血案內幕曝光 專家揭中共超限戰撕裂香港中共四中全會後，港警暴行進一步升級。11月2日，港警抓捕200多人，並致54人被送醫救治。3日，港警在多區抓人並一度舉槍。當日，在太古城更出現砍人、咬掉議員耳朵的血腥暴行。有專家指出，這是北京精心製造的陰謀，用超限戰製造白色恐怖，撕裂香港社會。
