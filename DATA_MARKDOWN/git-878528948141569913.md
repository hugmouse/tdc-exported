
# 山谷初男さん死去　８５歳、間質性肺炎…映画「八つ墓村」など多数

Published at: **2019-11-03T00:00:00+00:00**

Author: ****

Original: [デイリースポーツ online](https://www.daily.co.jp/gossip/2019/11/03/0012846846.shtml)

　多くの映画やドラマに出演し、素朴な役柄で親しまれた俳優の山谷初男（やまや・はつお、本名山谷八男）さんが１０月３１日に間質性肺炎のため秋田市内の病院で死去したことが３日、分かった。８５歳。秋田県出身。
　山谷さんは１９３３年１２月９日生まれ。１９６３年に俳優座小劇場の第１回研究生となった。６６年、「裏切りの季節」で映画デビュー。日活ロマンポルノにも出演した。他の映画に「八つ墓村」（７７年）、「少年Ｈ」（１３年）などがある。テレビドラマでは「リーガル・ハイ」（１２年）、「天皇の料理番」（１５年）など多数。
