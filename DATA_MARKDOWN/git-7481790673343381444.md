
# 文議長、上皇さまに謝罪の手紙…韓国側「事実ではない」

Published at: **2019-11-07T11:25:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191107-OYT1T50231/)

　日韓議員連盟の幹事長を務める自民党の河村建夫・元官房長官は７日、韓国の文喜相（ムンヒサン）国会議長が慰安婦問題を巡る自らの発言を謝罪する手紙を上皇さまに送ったと明らかにした。河村氏は自身が所属する二階派の会合で「おわびしたいということで、上皇さまにも手紙を出しているという話があった」と語った。
　一方、韓国国会報道官室は７日、文氏が上皇さま宛てに謝罪の手紙を送ったとの日本の一部報道について、事実ではないと発表した。文氏は今年２月に上皇さまに元慰安婦への謝罪を求める発言をしていた。
