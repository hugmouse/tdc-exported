
# SC Estate Builder放眼太阳能新项目

Published at: **2019-11-01T07:59:51+00:00**

Author: **Nazuin Zulaikha Kamarulzaman**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/sc-estate-builder%E6%94%BE%E7%9C%BC%E5%A4%AA%E9%98%B3%E8%83%BD%E6%96%B0%E9%A1%B9%E7%9B%AE)

（吉隆坡破1日讯）SC Estate Builder Bhd与Dekad Aliran私人有限公司（DASB）签署一项谅解备忘录（MoU），拟议在工程、采购、建设与调试（EPCC）结构下开展太阳能业务合作。
SC Estate Builder今日向大马交易所报备，MoU旨在建立和确认公司与DASB之间的相互理解和意图，以便在未来10年内展开合作。
“双方应在MoU签订之日起的24个月内，或在双方一致同意的期限内，就联营合作事宜签订联营协议，进行排他性谈判。”
该公司指出，DASB设计、实施和支援太阳能发电系统，并参与太阳能购电协议和再生能源供应协议。
截至3时35分，该股扬0.5仙或16.67%，报3.5仙，交投量有211万9600股。
