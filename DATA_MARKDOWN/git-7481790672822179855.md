
# 西武・秋山が足の指骨折、巨人・丸を招集…プレミア１２

Published at: **2019-11-01T07:23:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/sports/npb/20191101-OYT1T50228/)

　野球の国際大会「プレミア１２」に出場予定だった日本代表の西武・秋山が、右足薬指骨折のため出場を辞退した。１日、ＮＰＢエンタープライズが発表した。替わって巨人の丸が招集された。
　秋山は１０月３１日、那覇市内で行われたカナダとの強化試合「ＥＮＥＯＳ侍ジャパンシリーズ２０１９」第１戦に先発出場し、死球を受けて途中交代していた。
　今季の秋山は１７９安打を放ち、３年連続４度目の最多安打のタイトルを獲得。日本代表でもリードオフマンとして期待されていた。
　稲葉監督は、秋山の離脱について「ジャパンへの強い気持ちを持った選手がいなくなるのは非常に残念なことだが、チームとしては前に進んでいくしかない」と話した。
　秋山、丸のコメントは以下の通り。
　西武・秋山「このような結果となり残念ですが、試合に出る上では起こりえることなので仕方ないと思っています。世界一を獲るためにこのメンバーに参加できたことは、自分にとって、何より財産です。ケガを恐れて参加しない方が後悔したと思います。今後はケガの治療に全力で取り組んでいきます。プレミア１２の優勝のシャンパンファイトにはゴーグル持参で参加します！」
　巨人・丸「突然のことで大変驚きました。合流するからには日本代表の一員として一生懸命プレーし、チームの勝利に貢献したいと思います」
