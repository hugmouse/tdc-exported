
# 高材生嫖妓變業配網紅　出道拍AV前卻慘染性病

Published at: **2019-11-07T01:40:00+00:00**

Author: **娛樂組**

Original: [鏡週刊 Mirror Media](https://www.mirrormedia.mg/story/20191107ent005/)

日本名校「早稻田」學生篠塚康介之前因為上網分享買春經驗成名，又放話想當「日本第一AV男優」，結果現在是風俗界的爆紅人物，不僅到風俗店嫖免錢，還因此接到許多業配。
只是原本篠塚康介今年夏天準備正式AV出道，結果卻染了性病，讓他星途暫時夢碎，近來他在當地電視節目受訪時也分享，要等到自己痊癒，之後才能順利展開自己的色情片第一砲。
一名AV製作人曾經對篠塚康介分析，日本目前只有80名男優，卻有成千上萬女優，正式他投入成人產業的最佳時機。不過篠塚康介正面臨嚴重的家庭革命，目前已經離家，靠著寫作與朋友幫忙才能維生。
其實篠塚康介去年夏天還是處男，但因在街頭想嫖妓被騙了錢，他憤而到處花錢找女人、甚至「到府服務」一洩怨氣，並將心得po文，結果因此成為網紅。由於他完全沒有掩飾真實姓名以及身分，遭人向學校舉報，結果老師沒有懲處，反而大讚他的文筆很好。
