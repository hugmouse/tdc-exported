
# ล่าไอ้หื่น!อ้างเป็นตำรวจ ลากสาว16บังคับขืนใจในโกดังร้าง

Published at: **2019-11-03T06:00:00+00:00**

Author: ****

Original: [เดลินิวส์](https://www.dailynews.co.th/crime/739781)


        สาว 16 โร่แจ้งความ ถูกคนร้ายอ้างตัวเป็นตำรวจ จับไม่สวมหมวกกันน็อก ขู่พาไปโรงพัก แต่กลับโบ้ยสายรายงานฝ่ายหญิงพัวพันค้ายา ขอค้นร่างกายลวนลามในรถ ก่อนพาไปบังคับขืนใจในโกดังร้าง ด้านตำรวจสภ.สตึก-บุรีรัมย์ เร่งไล่กล้องวงจรปิดล่าตัวภัยสังคมแล้ว!
      
เมื่อวันที่ 3 พ.ย. น.ส.เอ (นามสมมุติ) พร้อมผู้ปกครองได้เข้าแจ้งความกับ ร.ต.อเกรียงศักดิ์ กิจไธสง รองสารวัตร(สอบสวน) สภ.สตึก จ.บุรีรัมย์ ว่าเมื่อวันที่ 2 พ.ย.ที่ผ่านมา ระหว่างที่ น.ส.เอ กำลังขี่จักรยานยนต์(จยย.)ไปร่วมกิจกรรมเดินพาเหรดงานแข่งเรือที่อำเภอสตึกวันแรกแล้วกำลังขี่จยย.กลับบ้าน ปรากฏว่า มีคนร้ายอายุประมาณ 50 ปี ผิวดำแดงตัดผมเกรียน สูงประมาณ170-180 เซนติเมตร ขับรถเก๋ง โตโยต้า สีน้ำเงินเข้มมาปาดหน้า ก่อนจะอ้างตัวเป็นตำรวจขอตรวจดูใบขับขี่จักรยานยนต์ แล้วก็แจ้งข้อหาไม่สวมหมวกกันน็อก แต่กลับไม่ยอมให้ใบสั่ง ทั้งยังบังคับให้ขึ้นรถไปด้วยกัน โดยให้ทิ้งรถจยย.เอาไว้ริมถนน
น.ส.เอ เล่าอีกว่า ด้วยความกลัวจึงยอมขึ้นรถไป แต่ปรากฏว่าชายคนร้ายกลับพูดจาข่มขู่ทำนองว่า มีสายรายงานว่าตนเป็นกลุ่มผู้ค้ายาเสพติด ทั้งที่ตนไม่เคยไปข้องเกี่ยวแต่อย่างใด จากนั้นก็ตรวจค้นร่างกายลักษณะลวนลาม แล้วก็พาไปยังโกดังร้างบังคับข่มขืน ระหว่างนั้นมีชาวบ้านขับรถไถเข้ามาทำให้คนร้ายรีบนำตนไปส่งยังจุดเดิม
เบื่องต้นเจ้าหน้าที่ได้ลงบันทึกประจำวันไว้ก่อน จากนั้นได้ไปดูจุดเกิดเหตุเพื่อตรวจสอบเส้นทางที่คนร้ายขับรถหลบหนี โดยหลังจากนี้จะได้สอบสวนหาพยานหลักฐานโดยเฉพาะกล้องวงจรปิดตามที่ต่าง ๆ เพื่อติดตามคนร้ายรายนี้มาดำเนินคดีตามกฎหมายต่อไป อย่างไรก็ตาม มีรายงานข่าวแจ้งว่า ก่อนหน้านี้ก็เคยมีเหตุการณ์คนร้ายอ้างเป็นตำรวจ ทำการลวนลามหญิงสาวในพื้นที่ จ.บุรีรัมย์ หลายครั้งหลายหนแล้ว ซึ่งเจ้าหน้าที่เชื่อว่าอาจเป็นฝีมือของคนร้ายรายเดียวกันก็เป็นได้.
