
# MIT研發自駕車轉角前瞻功能

Published at: **2019-11-05T13:43:47+00:00**

Author: ****

Original: [NTD 新唐人電視台](https://www.ntdtv.com/b5/2019/11/05/a102700680.html)

【新唐人北京時間2019年11月05日訊】麻省理工學院的研究者們在開發一套前瞻轉角物體的功能，利用地面陰影的變化探測前方側面或者轉角是否有移動的物體。這在自動駕駛的汽車、機器人或自動輪椅等方面有很好的應用前景。
這個版本基於2017年和2018年展示的版本做了改進。研究者對一輛停在車庫里的汽車進行的實驗中，新系統探測到其它移動車輛並剎車的反應速度，比傳統的利用激光進行探測的系統快了半秒。
半秒的差別，對於高速行進中的自動駕駛汽車，意味著嚴重車禍和「有驚無險」的差別。
研究者表示，這種系統還可以用於機器人，為機器人在繁忙的走道中自由移動進行導航；或者用於自動移動的輪椅，幫助其在醫院的走廊里順利移動。
不過目前這個系統只在室內的環境測試過。室內環境下，光照和陰影的情形相對比較穩定。
這種技術只是MIT在隔物探測領域研究的多種技術之一，其它技術包括使用雷達、人工智能，甚至Wi-Fi來實現探測的功能。
