
# அந்தமான் அருகே புதிய காற்றழுத்த தாழ்வுப்பகுதி -சென்னை வானிலை மையம்

Published at: **2019-11-04T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/State/2019/11/04150700/Near-Andaman-The-new-airborne-inferiorChennai-Meteorological.vpf)

சென்னைஅரபிக்கடலில் உருவான மஹா புயல் அதிதீவிரம் அடைந்துள்ள நிலையில், விரைவில் குஜராத் அருகே கரையை கடக்‍கும் என எதிர்பார்க்கப்படுகிறது.இந்நிலையில், சென்னையில்  நிருபர்களை சந்தித்த  வானிலை ஆய்வு மைய இயக்குனர் புவியரசன் கூறியதாவது:-
அந்தமானில் உருவாகியுள்ள காற்றழுத்த தாழ்வு பகுதி, அடுத்த 3 தினங்களில் வடமேற்கு திசையில் நகர்ந்து, மத்திய கிழக்கு வங்க கடலில் காற்றழுத்த தாழ்வு மண்டலமாக உருவாகக்கூடும்.அடுத்த சில தினங்களுக்கு சூறைக்காற்று வீசக்கூடும் என்பதால், மீனவர்கள் கடலுக்கு செல்ல வேண்டாம் என்றும், தென் மாவட்டங்களில் மழைக்கு வாய்ப்புள்ளது. சென்னையை பொறுத்தவரை வானம் மேகமூட்டத்துடன் காணப்படும் என்றும் தெரிவித்துள்ளார்.
