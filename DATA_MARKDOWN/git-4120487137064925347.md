
# 福州赛何济霆/谭强狂救4赛点 韩呈恺/周昊东胜强敌

Published at: **2019-11-06T02:56:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/05/sports-1572965771)

【新浪体育5日讯】2019年世界羽联S750级赛事福州羽毛球公开赛展开正赛首日的较量，男双比赛中，7号种子韩呈恺/周昊东19-21、21-16、21-16逆转日本组合保木卓朗/小林优吾。保木卓朗/小林优吾在世锦赛获得亚军后连续五站比赛一轮游（常州1000、仁川500、欧登塞750、巴黎750、福州750）。另一场比赛，何济霆/谭强在决胜局挽救了4个赛点，最终以22-20、13-21、27-25险胜俄罗斯组合伊万诺夫/索佐诺夫。首局比赛，何济霆/谭强在5-5平后一直处于领先，一度以13-7领先。在19-15时，中国组合连续浪费了四个盘点机会，被对手追平比分。好在两人还是顶住压力，在20-20平后连得两分拿下第一局。第二局，何济霆/谭强全程处于被动，被对手牢牢压制，最终以13-21输掉了这一局。决胜局，双方争夺激烈，何济霆/谭强在11-9领先时连丢了5分。后半段，中国组合在18-20落后情况下连续挽救赛点将比分扳平。在21-22和22-23时，中国组合又是化解赛点。两人也浪费了3个赛点机会，最终在25-25平时连得了2分险胜。女双首轮比赛中，李汶妹/郑雨21-9、21-8横扫印度组合蓬纳帕/斯基，董文静/冯雪颖21-15、21-9横扫马来西亚组合许嘉雯/叶铮雯。混双较量中，鲁恺/陈露22-24、18-21不敌5号种子、韩国组合徐承宰/蔡侑玎。
