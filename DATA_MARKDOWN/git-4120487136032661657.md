
# 欧冠-维纳尔杜姆破僵张伯伦建功 利物浦2-1根克

Published at: **2019-11-07T03:00:00+00:00**

Author: ****

Original: [Shangbao Indonesia](http://shangbaoindonesia.com/read/2019/11/06/sports-1573052361)

【网易体育6日讯】2019-2020赛季欧冠小组赛E组第四轮展开角逐，卫冕冠军利物浦坐镇安菲尔德球场迎战比利时球会根克。上半场比赛，维纳尔杜姆在第13分钟率先破门，根克在第40分钟将比分扳平。下半场比赛，张伯伦在第52分钟将比分反超。最终全场比赛结束，利物浦在主场以2-1的比分战胜了根克。两家球会在欧战史上交锋一次，即本赛季欧冠小组赛第一回合，红军客场4-1大胜。欧战史上，利物浦曾14次战比利时球会，取得了11胜2平1负，唯一一败是1978年欧洲超级杯第一回合客场1-3负于安德莱赫特，此后6战比利时球会取得5胜1平不败成绩。在安菲尔德球场，利物浦6战比利时球会取得了全胜战绩，打进15球，仅丢4球。根克曾在欧战史上4战英格兰球队，仅取得1胜3平成绩。双方出场名单：利物浦（4-3-3）：1-阿利松；66-阿诺德、4-范戴克、12-乔-戈麦斯、7-米尔纳；3-法比尼奥、5-维纳尔杜姆、8-纳比-凯塔（73’26-罗伯逊）；11-萨拉赫、15-张伯伦（73’10-马内）、27-奥里吉（89’9-菲尔米诺）。根克（3-5-2）：28-库克；2-德诺雷、33-卢库米、46-奎斯塔；6-德韦斯特、、17-赫罗绍夫斯基（84’27-邦贡）、25-博格、8-海嫩、31-梅勒；7-伊东纯也（67’77-恩东加拉）、10-萨马塔（84’18-奥努阿楚）。
