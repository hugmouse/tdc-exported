
# ចិន​ស្វាគមន៍​បារាំងជា មិត្តល្អ តែប្រាប់​មិន​ឱ្យ​លូកលាន់​រឿង​ហុងកុង

        លោក​ប្រធានាធិបតី​បារាំង អេម៉ាណុយអែល ម៉ាក្រុង អញ្ជើញ​ទៅ​បំពេញ​ទស្សនកិច្ច​ផ្លូវរដ្ឋ​នៅ​ប្រទេស​ចិន រយៈ​ពេល​បី​ថ្ងៃ​។ កម្មវត្ថុ​ គឺ​រឹត​បន្តឹង​​ទំនាក់ទំនង​​រវាង​ប្រទេស​ទាំង​ពីរ ក្នុង​បរិបទ​នៃ​ភាព​តាន​តឹង​ពិភពលោក។ កិច្ចព្រមព្រៀង​ជា​ច្រើន​នឹង​ត្រូវ​ចុះ​ហត្ថលេខា​ មាន​ជា​អាទិ៍ កិច្ច​សន្យា​ក្នុង​វិស័យ​កសិឧស្សាហកម្ម និង​ថាមពល។ ដោយ​ហេតុ​តែ​កំពុង​មាន​សង្គ្រាមពាណិជ្ជកម្មជាមួយ​អាមេរិក ចិន​បាន​រៀប​ចំ​ទទួល​ប្រមុខ​រដ្ឋ​បារាំង​ ក្នុងនាម​ជាមិត្ត​ យ៉ាង​កក់​ក្តៅ​បំផុត តែ​បាន​ឆ្លៀត​ប្រាប់​មុន​កុំ​ឱ្យ​​បារាំង​​លូកលាន់​នឹង​បញ្ហា​នៅហុងកុង។
      
វិបត្តិ​អន្តរជាតិ និង​អាកាសធាតុ​ ជា​ប្រធានបទដ៏​​សំខាន់នៃ​ជំនួប​ពិភាក្សា​របស់​ប្រមុខ​រដ្ឋ​ទាំងពីរ​មែន​ហើយ ប៉ុន្តែ​សំណុំ​រឿង​ពាណិជ្ជកម្ម​ ជា​អាទិភាព​ចំបង​បំផុត​នៃ​ទស្សនកិច្ច​របស់​ប្រធានាធិបតី​បារាំង​នៅ​ប្រទេស​ចិន។
បារាំង​ចង់​ជំរុញ​ឱ្យ​ចិន​ផ្តល់​លំនឹង​ពាណិជ្ជកម្ម​ទ្វេភាគី ព្រោះ​ថ្វី​បើ​នា​ឱកាសនេះ បារាំង​និង​ចិន​នឹង​ចុះ​ហត្ថលេខា​លើ​កិច្ចសន្យា​ពាណិជ្ជកម្ម​ជាង​៤០​មែន តែ​ជញ្ជីង​ពាណិជ្ជកម្ម​​ផ្អៀងខ្លាំង ៖ បារាំង​មាន​ឱនភាព​ខ្លាំង​ដល់​ទៅ​២៩​ពាន់​លាន​អឺរ៉ូ។ ប្រធានបទ​នេះ​នឹង​ត្រូវលោក​ ម៉ាក្រុង ​លើក​យក​ជជែក​ជាមួយ​លោក ស៊ី ជីនពីង​ ភ្លាម​នៅល្ងាច​​ថ្ងៃ​ចន្ទនេះ ក្នុង​ពិធី​ទទួល​ទានអាហារ​ និង​នៅ​ល្ងាច​ថ្ងៃ​អង្គារ​ស្អែក​ ក្នុងទម្រង់​​ ​ជា​លក្ខណៈ​ឯកជន​ នា​ពិធីបើក​សម្ពោធ​ពិព័រណ៍​នាំ​ចូល​នៅ​សៀងហៃ ដែល​បារាំង​ជា​ភ្ញៀវកិត្តិយស។
ក្នុង​ទស្សនកិច្ច​ផ្លូវរដ្ឋ​លើក​នេះ លោក​ប្រធានាធិបតី​ បាន​នាំ​​ថៅកែក្រុមហ៊ុន​បារាំង​​គ្រាក់ៗ ដូច​ជា​ក្រុមហ៊ុន​អ៊ែរបូស Airbus ឡូរ៉េអាល់ Oréal Sanofi ...ព្រម​ទាំង​ក្រុមហ៊ុន startups ​ ជា​ច្រើន​ទៀត​ទៅ​ជាមួយ។ នៅក្នុង​ចំណោម​គម្រោង​សំខាន់ៗ បារាំង​សង្ឃឹមនៅ​លើក​នេះ​ថា​នឹង​អាច​សម្រេច​គម្រោង​ស្ថាបនា​រោង​ចក្រ​ចម្រាញ់​សំរាម​វិទ្យុ​សកម្ម​ ដែល​ប្រទេស​ទាំង​ពីរ​បាន​ចរចា​​ជិត​១០​ឆ្នាំ​មក​ហើយ​ ព្រមទាំ​ងសង្ឃឹម​ថា​នឹង​អាច​ជ្រៀត​ចូល​ទីផ្សារ​លក់​សាច់​ជ្រូក​ចិន​បាន​ទៀត។ ក្នុង​ទស្សនកិច្ច​កាលពី​ឆ្នាំ​ទៅ​ លោក​ប្រធានាធិបតី​បាន​សម្រេច​លើក​ទណ្ឌកម្ម ១៧ឆ្នាំ​​របស់​​ចិន​លើ​សាច់​គោ​បារាំង។
លើ​ផ្នែក​វប្បធម៌​វិញ បារាំង​និង​ចិន មានទំនាក់ទំនង​ល្អ​នឹង​គ្នា​ខ្លាំង ដោយ​នៅ​ថ្ងៃ​អង្គារ​ស្អែក បារាំង​នឹង​បើក​សម្ពោធ​មជ្ឈមណ្ឌល​សិល្បៈ​សម័យ​​ ជា​សាខា​មជ្ឈមណ្ឌល George Pompidou ​ ប្រចាំ​ក្រុង​សៀងហៃ​ដែល​មាន​ប្រជាជន​រស់​នៅ​ជាង​២៤​លាន​នាក់។
​វិមាន​ប្រធានាធិបតី​បានធានា​ថា​​លោក​ម៉ាក្រុង​​នឹងលើក​យក​បញ្ហា​សិទ្ធិមនុស្ស​ព្រមទាំង​ស្ថានភាព​នៅហុងកុង និង​នៅ​ស៊ីងជាំង​ ទៅ​ជជែក​ជាមួយសមភាគី​ចិន​ដោយ​គ្មាន​រារែក​ទេ។
​ប៉េកាំង ដែល​កំពុងចង់​ពង្រឹង​ទំនាក់ទំនង​ជាមួយ​អឺរ៉ុប​ ដើម្បី​ស្តារ​សេដ្ឋកិច្ច​​ពី​សង្គ្រាម​ពាណិជ្ជកម្ម​ជាមួ​យ​អាមេរិកវិញ បាន​ទទួល​ស្គាល់​ថាការ​ចរចាសេដ្ឋកិច្ច​​បារាំង-ចិន​ពេល​នេះ​ មាន​សារៈ​សំខាន់​ណាស់ ប៉ុន្តែ​វិបត្តិ​ហុងកុង​និង​ស៊ីងជាំង ជា​កិច្ចការ​ផ្ទៃក្នុង​ចិន​ ដែល​មិន​សក្តិសម​នឹង​លើក​មក​ធ្វើ​ជា​របៀបវារៈ​នៃ​ជំនួប​បែបការ​ទូត​ឡើយ៕
