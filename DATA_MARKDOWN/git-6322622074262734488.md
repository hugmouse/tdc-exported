
# 首要媒体考虑新一轮裁员

Published at: **2019-11-01T05:33:06+00:00**

Author: **Ahmad Naqib Idris**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E9%A6%96%E8%A6%81%E5%AA%92%E4%BD%93%E8%80%83%E8%99%91%E6%96%B0%E4%B8%80%E8%BD%AE%E8%A3%81%E5%91%98)

（吉隆坡1日讯）媒体行业营运环境严峻，首要媒体（Media Prima Bhd）董事部正在考虑新一轮的裁员。
一名知情的业内人士透露：“各部门负责人已接到通知，将在今午3时召开会议，讨论董事部会议的结果。”
消息说：“据推测，这将涉及裁员，但我们必须等待董事部会议的结果。”
如果属实，这将是首要媒体进行的第二轮员工合理化活动。
去年11月，首要媒体旗下经营免费电视台TV3的Sistem Televisyen Malaysia Bhd向190每名职员发出3个月的裁员通知，同时向另外43名职员提供互惠离职方案（MSS）。
随着与商业大亨丹斯里赛莫达相关的Aurora Mulia私人有限公司，于今年9月收购该媒体集团的8829万股或7.9%股权后，首要媒体进行业务合理化。
Aurora于10月8日增持750万股，使持股权达到31.898%。
该投资工具还收购了Dilof私人有限公司的70%股权，后者在马来前锋报（Utusan Melayu (M) Bhd）关闭后，拥有《Utusan Malaysia》及《Kosmo!》的出版准证和档案。Dilof之后改名为Media Mulia私人有限公司。
截至6月杪2019财政年第二季，首要媒体净亏883万令吉，连续第二个季度蒙亏，归因于传统业务如出版和电视网络的贡献走低。
该集团自2016财年第三季以来，除了2016财年末季、2018财年次季和2018财年末季之外，均出现季度亏损。
