
# 商业环境改善 大马制造业PMI创6个月新高

Published at: **2019-11-01T06:38:36+00:00**

Author: **Cameron Chong**

Original: [The Edge Markets](https://www.theedgemarkets.com/article/%E5%95%86%E4%B8%9A%E7%8E%AF%E5%A2%83%E6%94%B9%E5%96%84-%E5%A4%A7%E9%A9%AC%E5%88%B6%E9%80%A0%E4%B8%9Apmi%E5%88%9B6%E4%B8%AA%E6%9C%88%E6%96%B0%E9%AB%98)

（吉隆坡1日讯）根据IHS Markit，随着商业环境改善，10月大马制造业采购经理人指数（PMI）升至49.3，创下6个月新高水平。
报告指，该指数恢复到与历史平均水平大致上一致的位置，9月为47.9。
IHS Markit首席经济师Chris Williamson说：“PMI生产增长的主要指标已升至一年来的最高水平，大致表明经济年增长率超过5%。”
增长是由新订单和新产量所驱动，两个指标均显示出改善的迹象。
HIS表示，就目前水平而言，调查的产出指数与年产量增长率约5.5%一致。
不过，报导指外围环境在第四季度初变得更具挑战性，导致出口订单连续第二个月下降。
IHS Markit强调，中国和欧洲是造成需求疲软的两个主因。
尽管如此，随着新订单总体指数上升，调查数据表明，内需条件帮助提供了抵消外部需求下滑的主要动力。
