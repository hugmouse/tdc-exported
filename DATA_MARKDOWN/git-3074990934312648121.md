
# '인텔' 공인대리점, 정품 CPU 퀴즈 프로모션 진행

Published at: **2019-11-06T01:05:09+00:00**

Author: **인벤팀 기자**

Original: [INVEN](http://www.inven.co.kr/webzine/news/?news=229491)


        자료제공 - 인텔
      
인텔 공인대리점 3사(코잇, 피씨디렉트, 인텍앤컴퍼니)는 ‘9세대 인텔 코어 프로세서’의 정품 구매 고객들을 위한 퀴즈 프로모션을 진행한다.
11월 23일까지 진행되는 퀴즈 프로모션에서 정답을 맞힌 참여자를 대상으로 추첨을 통해 여러 가지 경품이 제공된다.
이번 프로모션으로 인텔 정품 CPU에 대한 정확한 정보와 정품 CPU 혜택을 확인할 수 있다. 특히 인텔 CPU 구입 시 정품 CPU를 구매했는지 확인하는 방법도 알아볼 수 있다.
인텔 정품 CPU 구매를 통해 여러 혜택을 누릴 수 있다. 3년 A/S 보증, 로젠 택배 이용 시 왕복 택배비 무료, 소프트웨어 한정 인텔 테크니컬 콜센터를 통한 원격 A/S 서비스 제공, A/S 기간 내 구매 제품 단종 시 차상위 제품으로의 교환 등이 정품 CPU 구매 혜택으로 제공된다.
인텔 프로세서 제품군의 정품 CPU 혜택은 CPU 박스 또는 완제품 조립 PC 표면에 부착된 인텔 정품 CPU 스티커를 통해 시각적으로 확인할 수 있다. 또한 Real CPU 홈페이지에 방문해 제품번호를 체크하고 확인하는 방법도 있다.
인텔 공인대리점 3사는 정품 고객들에게 제공되는 다양한 혜택들을 알리고 올바르지 않은 경로로 반입된 물품에 의한 불의의 피해를 방지하고자 퀴즈 프로모션을 진행한다며 공인대리점 로고 정품 스티커를 확인하고 다양한 혜택과 더불어 매월 열리는 이벤트에서 정품 유저의 혜택을 누리길 바란다고 밝혔다.
