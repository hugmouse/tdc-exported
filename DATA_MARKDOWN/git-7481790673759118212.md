
# ＡＳＥＡＮ、米との会議への首脳参加３か国のみ…対抗措置か

Published at: **2019-11-03T13:35:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/world/20191103-OYT1T50169/)

　【バンコク＝田原徳容】東南アジア諸国連合（ＡＳＥＡＮ）は、バンコクで４日に予定される米国との首脳会議について、首脳の出席は加盟１０か国のうち議長国タイなど３か国にとどめることを決めた。複数のタイ政府高官が明らかにした。米国が首脳級を欠席させたことに対抗したとみられる。
　関係者によると、首脳が出席するのはタイと来年の議長国ベトナム、対米交渉窓口のラオス。他の７か国の出席者は各国が判断する。２日のＡＳＥＡＮ首脳会議でこうした対応を決めた。
　米国は今回、東アジア首脳会議（ＥＡＳ）を含むＡＳＥＡＮ関連首脳会議に、オブライエン国家安全保障担当大統領補佐官を特使として派遣した。昨年はペンス副大統領が出席した。トランプ大統領は３年連続のＥＡＳ欠席となり、ＡＳＥＡＮ内で「東南アジアを軽視しているのでは」との見方が出ている。
