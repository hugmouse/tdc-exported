
# ഷാനവാസ് കൊലപാതകം; രണ്ട് പേര്‍ അറസ്റ്റില്‍

Published at: **2019-11-05T22:01:00+00:00**

Author: **Sreehari**

Original: [Anweshanam | Part of A.N.N News Private Limited](http://anweshanam.com/index.php/kerala/news/2-arrested-in-shanavas-murder)

കാസര്‍കോട്: കാസര്‍കോട്ടെ ഷാനവാസിന്റെ കൊലപാതകത്തില്‍ രണ്ടുപേര്‍ പിടിയില്‍. മുന്ന, ജയേന്ദ്രന്‍ എന്നിവരാണ് പിടിയിലായത്. കാസര്‍കോട് ടൗണ്‍ പോലീസാണ് ഇവരെ അറസ്റ്റ് ചെയ്തത്.
ജയേന്ദ്രന്‍ കൊട്ടാരക്കര സ്വദേശിയെ കെഎസ്ആര്‍ടിസി ബസ് സ്റ്റാന്റില്‍ വെച്ച് സോഡാ കുപ്പി ഉപയോഗിച്ച് കുത്തിക്കൊലപ്പെടുത്താന്‍ ശ്രമിച്ച കേസിലെ പ്രതിയാണ്. മുന്ന കാസര്‍കോട്ടെ കഞ്ചാവ് മൊത്ത വില്‍പ്പനക്കാരനാണെന്നും പോലീസ് അറിയിച്ചു.
കേസില്‍ ഒരാള്‍ കൂടി പിടിയിലാവാനുണ്ട്. കഞ്ചാവ് വില്‍പനയുമായി ബന്ധപ്പെട്ട പണമിടപട് തര്‍ക്കമാണ് ഷാനവാസിന്റെ കൊലപാതകത്തിലേക്ക് നയിച്ചതെന്ന് പോലീസ് അറിയിച്ചു.
