
# អាល្លឺម៉ង់​ជួយ​ឥណ្ឌា ដើម្បី​អភិវឌ្ឍ​មធ្យោបាយ​ធ្វើដំណើរ​ ដែល​មិន​ធ្វើ​ឱ្យ​ខូច​បរិស្ថាន​

          ប្រមុខ​រដ្ឋាភិបាល
          អាល្លឺម៉ង់ បាន​សន្យា​ផ្តល់​ជំនួយ​ទឹក​ចំនួន​១លាន​អឺរ៉ូឱ្យ​ឥណ្ឌា រៀង​រាល់​ឆ្នាំ ដើម្បី​ ​អភិវឌ្ឍ​មធ្យោបាយ​ធ្វើដំណើរ​ ដែល​មិន​ធ្វើ​ឱ្យ​ខូច​បរិស្ថាន​។ ដំណឹង​នេះ បា​នត្រូវ​បញ្ជាក់​ពី​ថ្ងៃសៅរ៍​ ទី​២វិច្ឆិកា​ នៅ
          ក្នុង​ដំណើរ​ទស្សនកិច្ច​ផ្លូវ​រដ្ឋ របស់​លោកស្រី​អធីការ​បតី​អាល្លឺម៉ង់ 
          Angela Merkel
           នៅក្រុង​ញូដេលី ប្រទេស​ឥណ្ឌា។​ 
        
      
ទឹក​ប្រាក់​១លាន​អឺរ៉ូ​របស់​អាល្លឺម៉ង់​សម្រាប់​ជួយ​ឥណ្ឌា គឺ​មាន​រយៈ​ពេល​​៥ឆ្នាំ ដែល​តម្រូវឱ្យ​​សហការ​ធ្វើ​គម្រោង​ផ្សេងៗនៅក្នុង​បណ្តាទីក្រុង​និង​ប្រជុំជន​ឥណ្ឌា ដើម្បីរក​​មធ្យោបាយ​ធ្វើ​ដំណើរ ដែល​ប៉ះពាល់​បរិស្ថានតិចតួច ឬ​ហៅ​តាម​បែប​បច្ចេកទេស គឺ​មធ្យោបាយ​ធ្វើ​ដំណើរ​បៃតង។​ លើសពីនេះ លោកស្រីអធីការបតីអាល្លឺម៉ង់ Angela Merkel ថែមទាំង​​ជូន​យោបល់​ទៅ​ឥណ្ឌា ថា​គួរ​តែ​ឈប់​ប្រើរ​ថយន្ត​ចាក់​ប្រេង​សាំងនិងប្រេង​ម៉ាស៊ូតទៅ​ ហើយ​ងាក​មកប្រើ​រថយន្ត​ដើរ​ដោយ​ថាមពល​អគ្គិសនី​វិញ ដើម្បី​ជួយ​កាត់​បន្ថយ​បញ្ហា​បំពុល​បរិយាកាស ​នៅក្នុង​ក្រុង​ញូដេលី។
រដ្ឋធានី​ញូដេលី របស់​ឥណ្ឌា ពេ​លនេះ​កំពុង​រង​គ្រោះខ្លាំង​ដោយ​សា​រ​បញ្ហាកម្តៅ​អាកាសធាតុ ​ចុះ​អព្ទ​ ផ្សែង​ពុល ខ្យល់​កខ្វក់ ដែល​ធ្វើ​ឱ្យ​មនុស្ស​ឈឺ​ ​ស្ងួត​បំពង់​ក លំបាក​ដកដង្ហើម ពិសេស​លំបាក​ខ្លាំង​ចំពោះ​ក្មេងៗ​និង​មនុស្ស​ចាស់​។ បញ្ហា​ស្ថានភាព​គ្រោះថ្នាក់​ផ្លូវ​ដង្ហើមនេះ គឺ​ ដល់​កម្រិតថ្នាក់«​ប្រកាស​អាសន្ន​»ទៅហើយ។​​ 
បញ្ហា​ខ្យល់​កខ្វក់​នៅ​ញូដេលី​នេះ គឺ​ត្រូវគេ​វាយ​តម្លៃ​ថា មកពីឧស្ម័ន​បញ្ចេញពី​រោងចក្រ យានយន្ត និង​មកពី​ការដុត​ព្រៃ​ធ្វើកសិកម្ម​ជាដើម​។
​ជាវិធានការ​ចំពោះ​មុខ,​ អាជ្ញា​ធរ​ឥណ្ឌា​បាន​ប្រកាស​បិទ​សាលា​រៀន​ រហូត​ដល់​ថ្ងៃ​អង្គារខាងមុខ ​ទើប​បើក​​សាលា​ឱ្យ​កូន​សិស្ស​ចូល​រៀ​នវិញ។ ទន្ទឹ​មនេះ អាជ្ញាធរឥណ្ឌា​មាន​គម្រោង​ចែក​ម៉ាស់​បិទមុខ ការពារ​នៅ​ពេល​ដកដង្ហើម ចែកចាយ​ដល់កូន​សិស្ស​ ចំនួន​៥លាន​ម៉ាស់​។​ លើសពីនេះ រដ្ឋបាល​ក្រុង​ញូដេលី បាន​ចេញបម្រាមឱ្យ​កាត់​បន្ថយ​ម៉ោង​ធ្វើការ​នៅ​ការរដ្ឋាន​មួយ​ចំនួន ដែល​ធ្វើការ​ក្នុង​សភាព​ក្តៅខ្លាំង​ ហើយ​និង​​ហាម​ប្រាម​មិន​ឱ្យ​ប្រើ​ប្រាស់​ម៉ាស៊ីន​​ភ្លើង​ចាក់​ប្រេងម៉ាស៊ូត​ មួយ​រយៈ​សិន គឺ​ហាម​ឃាត់​ការ​ប្រើ​ប្រាស់​រហូត​ដល់​ថ្ងៃទី​​១៥វិច្ឆិកា​ខាងមុខ។
បើ​តាម​ព្រឹត្តិបត្ត​វិទ្យាសាស្ត្រ​ The Lancet នៅក្នុង​ប្រទេស​ឥណ្ឌាឆ្នាំ​២០១៧, បញ្ហាខ្យល់​កខ្វក់ បាន​សម្លាប់មនុស្ស ប្រមាណ​១លាន​២សែន​នាក់៕​
