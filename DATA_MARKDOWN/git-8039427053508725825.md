
# విజయం మాదే..: జేఏసీ చైర్మన్‌ అశ్వత్థామరెడ్డి

హైదరాబాద్ : విజయం మాదే అని ఆర్టీసీ జేఏసీ చైర్మన్‌ అశ్వత్థామరెడ్డి అన్నారు. విద్యార్థి, నిరుద్యోగ ఫ్రంట్‌ చైర్మన్‌ చనగాని దయాకర్‌ ఆధ్వర్యంలో గురువారం ఆర్ట్స్‌ కళాశాల ఆవరణలో చనిపోయిన ఆర్టీసీ కార్మికులకు సంతాపసూచకంగా కొవ్వొత్తుల ర్యాలీ నిర్వహించారు. అనంతరం అశ్వత్థామరెడ్డి మాట్లాడుతూ తమ ప్రాణాలను, ఉద్యోగాలను ఫణంగా పెట్టి పోరాటాలు చేస్తున్నామని, సకలజనులరీతిలో చేస్తున్న సమ్మె తమ కోసం కాదని, తెలంగాణ ప్రజలకు సౌక్యమైన ప్రయాణం అందించడం కోసం అన్నారు. ఆర్టీసీ పరిధిలో ఉన్న కోట్లాది రూపాయల విలువైన భూములను పరిరక్షణ కోసం పోరాటం చేస్తున్నట్లు చెప్పారు.
 
ఆర్టీసీని ప్రైవేటుపరం చేస్తున్న కుట్రకు వ్యతిరేకంగా పోరాటం చేస్తున్నామన్నారు. ఓయూ విద్యార్థుల మద్దతుతో సమ్మె చివరిదశకు చేరుకుందని, తప్పకుండా విజయం సాధిస్తామన్నారు. తెలంగాణ ఇంటిపార్టీ రాష్ట్ర అధ్యక్షులు డా.చెరుకు సుధాకర్‌, బీసీ సంక్షేమ సంఘం రాష్ట్ర అధ్యక్షులు జాజుల శ్రీనివా్‌సగౌడ్‌ మాట్లాడుతూ ఎవ్వరూ అధైర్యపడవద్దని, పోరాటాల ద్వారానే ఆర్టీసీ కార్మికుల సమస్యలు పరిష్కరమవుతాయన్నారు. ఈ కార్యక్రమంలో ఓయూ విద్యార్థి నేతలు ఆర్‌.ఎన్‌.శంకర్‌, కాంపల్లి శ్రీనివాస్‌, రంజిత్‌, గడ్డం శ్యామ్‌, కృష్ణ మాదిగ, మౌనిక, గుణ, లెనిన్‌రాజు తదితరులు పాల్గొన్నారు.
