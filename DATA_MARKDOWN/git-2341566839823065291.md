
# 美商務部長：美中貿易協議樂觀 很快鬆綁華為

Published at: **2019-11-04T14:20:00+00:00**

Author: **https://www.facebook.com/cnanewstaiwan, 中央通訊社**

Original: [中央社 CNA](https://www.cna.com.tw/news/firstnews/201911040078.aspx)

（中央社曼谷4日綜合外電報導）美國商務部長羅斯（Wilbur Ross）對本月與中國達成「第一階段」貿易協議表示樂觀，還說放行美國企業向華為出售零件的許可「很快」就會發布。
美國總統川普昨天在白宮告訴媒體，一旦協議敲定，就會在美國簽署；他在結束一趟紐約市的行程後說：「首要之務是我想簽署這份協議，對我而言，簽署地點根本不是問題。」
羅斯表示，智利因國內動亂取消主辦亞太經合會（APEC），使美中無法在場邊舉行峰會簽署協議後，美國的愛阿華州、阿拉斯加、夏威夷、甚至中國各地，都可能成為川習簽署協議的地點。羅斯稱這項協議「尤其複雜」，美國也確信，雙方對彼此所達成的共識，有著非常正確、清晰與詳細的認知。
羅斯在曼谷接受美國財經媒體專訪時說：「我們目前態勢良好，也取得不錯進展，沒有理由簽不成。」但被問到協議能否如期在本月簽成，羅斯仍說「是否又會稍加拖延，誰知道呢，總有這樣的可能」。
中國在這項第一階段協議裡將加購美國農產品、保持人民幣匯率穩定，以及向美國公司開放金融服務市場，北京希望藉此換取美國打消12月15日再對包括智慧手機等中國貨品加徵關稅的措施。
羅斯正出席東南亞國家協會高峰會（ASEAN Summit），他重申川普政府尚未允諾是否暫緩12月這波加徵關稅之舉，還說兩國後續階段的貿易協議，取決於中國的相關立法與落實機制，「沒有這些，談出的協議不過是堆廢紙」。
川普今年6月與中國領導人習近平在G20峰會場邊舉行川習會時允諾，將准許美國企業向華為出售特定零組件，但至今仍未宣布。羅斯說，相關許可「很快便會放行」，稱美國政府已接獲260件申請。
羅斯還淡化由中國所支持、16國參與的「區域全面經濟夥伴協定」（RCEP）重要性。根據這項協定，經濟規模相當於全球經濟1/3的RCEP將互降關稅，亞洲各國領袖預計在這次東協峰會後宣布RCEP將取得突破。
羅斯說：「RCEP算不上是協議，它不是自由貿易協議，遠不像是跨太平洋夥伴協定（TPP），也比不上我們跟日、韓各別達成的自貿協議。我不認為你們想去吹捧RCEP，那不過是級別很低的協議。」（譯者：陳亦偉/核稿：劉文瑜）1081104
