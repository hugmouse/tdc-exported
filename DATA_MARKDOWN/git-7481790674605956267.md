
# 英語民間試験の議事録公開へ…文科相、経緯検証の意向

Published at: **2019-11-07T13:52:00+00:00**

Author: ****

Original: [読売新聞オンライン](https://www.yomiuri.co.jp/politics/20191107-OYT1T50242/)

　萩生田文部科学相は７日の衆院本会議で、２０２０年度からの大学入学共通テストで導入を見送った英語民間試験について、活用を決めた経緯の検証に取り組む意向を示した。文科省の会議の議事録を公開するほか、下村博文氏ら歴代の文科相からも聞き取る。
　議事録を公開するのは、同省が１６年４月に設置した有識者らによる「検討・準備グループ」の会議。１７年５月に英語民間試験の活用を明記した実施方針案を公表するまで９回開かれた。
　萩生田氏は衆院本会議で「議事概要は非公開を前提に作成したものだが、今後の検証に必要な情報であることから、当時の委員の同意や機微な情報の取り扱いも含め、確認を取りつつ、公開していきたい」と述べた。これに先立ち、参院文教科学委員会で「歴代文科大臣にも経緯はぜひお伺いしたい」と明言した。
　文科省は昨年１２月以降、大学や高校、試験団体の関係者と意見交換する会議も開いており、この会議内容の公開も検討している。
