
# காஷ்மீரில் கடுமையான பனிபொழிவு: போக்குவரத்து பாதிப்பு

Published at: **2019-11-07T00:00:00+00:00**

Author: ****

Original: [Dailythnathi.com](https://www.dailythanthi.com/News/India/2019/11/07000242/Heavy-snowfall-in-Kashmir-impact-on-traffic.vpf)

ஸ்ரீநகர்,  யூனியன் பிரதேசமான லடாக்கை, காஷ்மீருடன் இணைக்கும் ஸ்ரீநகர்-லே நெடுஞ்சாலையில் கடுமையான பனிப்பொழிவு காரணமாக போக்குவரத்து நிறுத்தப்பட்டது.  மேலும் அதிகப்பனிப்பொழிவால் பூஞ்ச் மற்றும் ரஜோரி சாலையை ஷோபியான் மாவட்டத்துடன் இணைக்கும் முக்கிய சாலையான முகலாய சாலையிலும் போக்குவரத்திற்கு தடைவிதிக்கப்பட்டது.
முன்னதாக ஜம்மு-காஷ்மீர் மற்றும் லடாக் யூனியன் பிரதேசங்களில் வரும் 3 நாட்களுக்கு மிதமானது முதல் கடுமையான பனிப்பொழிவுடன், மழைக்கும் வாய்ப்புள்ளதாக அம்மாநில வானிலை ஆய்வு மையம் தெரிவித்திருந்தது. ஜம்மு-காஷ்மீர் மற்றும் லடாக் யூனியன் பிரதேசங்களில் நிலவும் கடுமையான பனிப்பொழிவால் அப்பகுதி மக்களின் இயல்பு வாழ்க்கை கடுமையாக பாதிக்கப்பட்டுள்ளது.
